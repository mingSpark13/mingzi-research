# 📖 花火格式笔记

## 🎯 题目

ViVa: A Video-Generative Value Model for Robot Reinforcement Learning

## 📝 三句摘要

1. **问题背景**：VLA 在真实操作里容易被部分可观测和延迟反馈拖后腿，传统基于静态视觉语言模型的 value model 很难稳住长时程任务评估。
2. **核心方法**：论文提出 ViVa，把预训练视频生成器改造成 value model，联合预测未来本体状态与当前状态标量价值，用时空先验替代静态打分。
3. **关键结果**：ViVa 集成进 RECAP 后，在真实 box assembly 上明显提升策略学习效果，还能对新物体保持更可靠的任务进度估计。

## 💎 价值评估

- **🔬 研究价值**：它把“视频生成先验→价值估计”这条链真正打通了，比普通 reward/value head 更贴近时序具身任务，对 D07 和主人关心的世界模型+控制闭环都很有参考意义。
- **🚀 实践价值**：如果后续主人想做 UAV/机械臂长时程任务，ViVa 这类 foresight value module 很适合接在已有 policy 上做 critic 升级，不必整套重训。
- **📈 扩展潜力**：可以继续往 imagined rollout scoring、planner verification、failure recovery 方向延展，和 Dream2Fix、世界模型验证器路线天然能接上。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 或 UE 仿真中，把现有策略的 critic 从静态观测 value head 替换为 video-generative value model，验证长时程装配/跟随/导航任务的收益。
- 对比基线：普通 MLP critic、VLM-based value head、无 foresight 的 advantage estimation
- 度量指标：任务成功率、value calibration 误差、长时程回报、失败恢复率
- 预期结果：带视频生成先验的 critic 在延迟奖励和部分可观测任务上更稳，尤其能减少“前面看着对、后面突然崩”的估值漂移
