# 📖 花火格式笔记

## 🎯 题目

Data Analogies Enable Efficient Cross-Embodiment Transfer

## 📝 三句摘要

1. **问题背景**：通用机器人策略虽然能吃下大规模异构演示，但“什么样的数据组织方式”最有利于跨载体迁移仍不清楚，尤其 morphology shift 往往不能靠单纯堆多样性解决。
2. **核心方法**：论文系统比较视角、外观、末端形态三类 domain shift，提出用 **data analogies** 组织跨机器人配对演示，让不同 embodiment 在场景、任务与轨迹层面对齐，而不是仅做无结构扩容。
3. **关键结果**：在仿真与真实实验中，这种配对式数据组成比大规模无配对数据更有效，真实跨载体迁移平均成功率提升 **22.5%**。

## 💎 价值评估

- **🔬 研究价值**：这篇把 D04 从“模型怎么设计”往前推进到“数据该怎么采”，很适合作为 morphology-aware backbone 之外的数据侧主线证据。
- **🚀 实践价值**：对主人做空地/异构末端迁移很有参考意义——若能采到同场景同任务的类比轨迹，可能比盲目扩大杂数据更划算。
- **📈 扩展潜力**：可和 PMI、HEX、Morphology Embedding 组合成“接口层 + 结构先验 + 配对数据”三件套，支撑 D04 的完整方法叙事。

## 🎯 可落地实验点

**实验设计**：在主人现有异构平台上构建小规模“配对演示集”，比较 data analogies 与无配对混合数据对跨载体迁移的影响。
- 对比基线：随机混合异构演示 / 仅目标载体少样本微调 / morphology embedding
- 度量指标：目标任务成功率、样本效率、跨载体适配收敛速度
- 预期结果：同任务同场景配对演示在 morphology shift 下显著优于仅增加数据量
