# 📖 花火格式笔记

## 🎯 题目

ExoActor: Exocentric Video Generation as Generalizable Interactive Humanoid Control

## 📝 三句摘要

1. **问题背景**：人形机器人在开放场景中的交互控制需要同时建模环境、对象、动作和任务意图，传统监督式控制很难覆盖这种高维互动。
2. **核心方法**：ExoActor把第三人称视频生成当作统一交互建模接口，先依据任务指令和场景合成可行动作过程，再估计人体运动并交给通用控制器执行。
3. **关键结果**：该框架无需新增真实机器人数据也能泛化到新场景，说明生成式视频模型可直接充当跨载体行为先验与交互动力学建模器。

## 💎 价值评估

- **🔬 研究价值**：把视频生成模型从“看世界”推进到“生成人形交互行为”，对跨载体泛化和行为先验建模很有启发。
- **🚀 实践价值**：适合给主人后续做空地迁移/跨平台动作迁移时借鉴，用外部视频先验减少真机示教成本。
- **📈 扩展潜力**：若和VLA或世界模型结合，可演化成“生成未来交互视频→反推出控制序列”的统一规划器。

## 🎯 可落地实验点

**实验设计**：验证“视频生成先验”是否能提升跨平台行为迁移
- 对比基线：直接行为克隆 / 纯轨迹优化 / VLA微调
- 度量指标：新场景成功率、动作自然度、零样本泛化成功率
- 预期结果：引入第三人称生成先验后，在少真机数据条件下能提高新场景交互鲁棒性
