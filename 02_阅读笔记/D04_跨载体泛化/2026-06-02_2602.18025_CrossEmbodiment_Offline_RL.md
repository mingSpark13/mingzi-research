# 📖 花火格式笔记

## 🎯 题目

Cross-Embodiment Offline Reinforcement Learning for Heterogeneous Robot Datasets

## 📝 三句摘要

1. **问题背景**：跨载体预训练常依赖高质量示教做行为克隆，但真实机器人里更容易获得的是混杂大量次优轨迹的异构离线数据。
2. **核心方法**：本文系统研究“跨载体学习 + offline RL”的组合，在 16 种机器人步态数据上分析梯度冲突问题，并提出按形态相似性分组的更新策略缓解多载体互相干扰。
3. **关键结果**：结果表明当数据中次优轨迹占比较高时，cross-embodiment offline RL 明显优于纯 BC；而随着机器人类型和次优比例增加，分组更新能显著优于朴素 offline RL 与已有冲突消解方法。

## 💎 价值评估

- **🔬 研究价值**：它把“跨载体泛化”从 imitation learning 推到 offline RL 场景，直接回答了异构低质量数据是否能被吃下这个关键问题。
- **🚀 实践价值**：对主人后续想做多平台机器人共训、吃入次优 rollout、减少每个平台昂贵示教成本很有启发。
- **📈 扩展潜力**：形态分组更新可迁移到 VLA / flow-matching 训练，甚至能和 PMI/动作空间统一思路结合，做更强的跨机型预训练管线。

## 🎯 可落地实验点

**实验设计**：在现有多机器人或多仿真平台数据上，对比 BC、naive offline RL、形态分组 offline RL 的跨载体迁移表现。
- 对比基线：Behavior Cloning、IQL/CQL 等 naive offline RL、无分组的多载体联合训练
- 度量指标：目标机器人成功率、样本效率、梯度冲突统计、不同次优数据占比下的鲁棒性
- 预期结果：在次优数据较多或机器人异质性较强时，形态分组更新相对 naive 联训更稳定，目标平台迁移性能更高
