# 📖 花火格式笔记

## 🎯 题目

From Pixels to Tokens: A Systematic Study of Latent Action Supervision for Vision-Language-Action Models

## 📝 三句摘要

1. **问题背景**：异构机器人数据混训时，VLA 往往被动作空间不一致和监督形式碎片化拖后腿，导致跨任务、跨 embodiment 的泛化收益吃不满。
2. **核心方法**：论文把 latent action supervision 系统化拆成 image-based trajectory regularization 与 action-based target-space unification 两大路线，并在统一 VLA baseline 下比较四种代表性集成策略。
3. **关键结果**：实验发现“图像型 latent action 更利于长时程推理与场景级泛化，动作型 latent action 更利于复杂电机协调”，而直接用离散 latent action token 监督 VLM 效果最好，说明 latent action 是混合数据训练里很实用的统一接口。

## 💎 价值评估

- **🔬 研究价值**：这篇不是再造一个花哨 VLA，而是把 latent action 监督路线做了少见的系统对照，对 D04 的“异构源先验如何真正转成下游 uptake”很有解释力。
- **🚀 实践价值**：主人做跨载体/跨平台迁移时，可以把不同机器人动作先投到 token/latent 空间，再决定是偏场景理解还是偏控制协调，少走很多接口试错弯路。
- **📈 扩展潜力**：很适合和 JoyAI-RA、Unified Latent Space、LeWorldModel 一起串成“统一表征/统一接口/统一规划”路线，后续可直接催生 AirSpark 的跨数据源训练配方。

## 🎯 可落地实验点

**实验设计**：在 AirSpark 或无人机+机械臂混合数据集上，对比原始动作监督、图像型 latent action、动作型 latent token 三种训练方式，验证哪种更适合异构数据混训。
- 对比基线：原始连续动作回归、仅图像 latent regularization、离散 latent action token 监督
- 度量指标：zero-shot 新任务成功率、长时程任务完成率、动作协调稳定性、跨平台迁移后首轮可用率
- 预期结果：离散 latent token 监督应最稳地提升 mixed-data 泛化；图像型 latent 更可能改善长时程场景推理，无人机任务可能在复杂机动协调上更依赖动作型 latent
