# 📖 花火格式笔记

## 🎯 题目

CEI: A Unified Interface for Cross-Embodiment Visuomotor Policy Learning in 3D Space

## 📝 三句摘要

1. **问题背景**：跨 embodiment 策略迁移常被机器人形态、末端执行器和动作接口不一致卡住，导致示教数据和策略都很难在异构平台间复用。
2. **核心方法**：CEI 提出统一的 3D cross-embodiment interface，把不同机械臂与夹爪/灵巧手的示教和控制都映射到共享空间，并支持双向 demonstration transfer 与 policy transfer。
3. **关键结果**：论文在仿真中覆盖 16 种 embodiment、3 个任务，在真实世界支持平行夹爪与五指灵巧手双向迁移，整体 transfer ratio 达到 82.4%，证明它确实是跨载体数据复用的强接口基线。

## 💎 价值评估

- **🔬 研究价值**：这篇把“跨 embodiment 统一接口”做得非常具体，是 D04 很硬的一块地基，尤其适合拿来和 PMI、Unified Latent Space、latent action 监督路线做机制对照。
- **🚀 实践价值**：主人以后做无人机机械臂/地空异构平台共享操作数据时，完全可以借鉴 CEI 的 shared 3D interface 思路，先统一示教语义和动作承载层，再谈策略迁移。
- **📈 扩展潜力**：它最值得继续追的是把 3D interface 往动态平台、空中操作、触觉反馈和在线纠偏扩展，能直接接到主人关心的跨平台具身控制链路上。

## 🎯 可落地实验点

**实验设计**：在“地面机械臂 + 空中抓取平台”混合数据上实现一个轻量版 CEI 接口，对比直接动作映射与共享 3D interface 的跨平台迁移效果。
- 对比基线：原始动作回归、简单末端位姿对齐、CEI 风格 shared 3D interface
- 度量指标：cross-platform transfer success rate、首轮可用率、示教复用率、异构末端执行器任务成功率
- 预期结果：shared 3D interface 应显著提升异构平台间示教可复用性，尤其在末端执行器差异较大时更稳
