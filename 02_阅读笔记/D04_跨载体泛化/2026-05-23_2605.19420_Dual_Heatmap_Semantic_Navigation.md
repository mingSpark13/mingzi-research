# 📖 花火格式笔记

## 🎯 题目

Beyond Waypoints: Dual-Heatmap Grounding for Cross-Embodiment Semantic Navigation

## 📝 三句摘要

1. **问题背景**：开放语义指令在落地成机器人可执行局部目标时，经常因为点式 waypoint 回归忽略空间不确定性，把目标打到不可通行区域，跨载体执行时更容易脆断。
2. **核心方法**：论文提出统一视觉语言框架，用 Dual-Heatmap 同时预测可达区域热力图和朝向热力图，把语义 grounding 从单点回归改成可微的连续 affordance field，并支持 Jetbot、H1、Aliengo 等多载体迁移。
3. **关键结果**：方法在同量级 8B baseline 中达到 SOTA，并显著提升跨 embodiment 的 Affordance Rate，说明显式可达区域建模比直接回归 waypoint 更稳。

## 💎 价值评估

- **🔬 研究价值**：这篇很准地打中了“语义 grounding 不该只输出一个点”这个关键缺口，把跨载体导航里的可执行性问题显式建模出来。
- **🚀 实践价值**：对主人做跨平台机器人导航、VLM 高层决策接低层局部规划特别有参考价值，能减少落点不可达和朝向错误导致的执行失败。
- **📈 扩展潜力**：后续可直接接入 UAV/UGV/humanoid 统一接口，把热力图语义场当成中间表示，兼容不同动力学和下游局部规划器。

## 🎯 可落地实验点

**实验设计**：在现有语义导航管线中，把单点目标头替换为“可达区域+朝向”双热力图头，测试不同 embodiment 下的可执行性与迁移鲁棒性。
- 对比基线：point waypoint regression、单热力图 affordance 头、规则后处理落点筛选
- 度量指标：Affordance Rate、Success Rate、目标可达率、朝向误差、跨 embodiment 泛化性能
- 预期结果：双热力图表示会明显降低不可执行目标落点，尤其在形态差异较大的平台上更稳
