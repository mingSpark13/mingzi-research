# 📖 花火格式笔记

## 🎯 题目

AdaTracker: Learning Adaptive In-Context Policy for Cross-Embodiment Active Visual Tracking

## 📝 三句摘要

1. **问题背景**：主动视觉跟踪想做成单模型跨多机器人通用，但不同载体的运动学约束和动态特性差异太大，现有方法常常一机一模，扩展性很差。
2. **核心方法**：AdaTracker 引入 Embodiment Context Encoder，从历史观测中推断载体特有约束，再用 Context-Aware Policy 做上下文调制，实现对未见载体的 in-context 自适应控制。
3. **关键结果**：论文在仿真和真实机上都证明了它在跨载体泛化、样本效率、zero-shot 适配上明显优于现有方法，说明“先识别载体上下文、再出动作”这条路线很有戏。

## 💎 价值评估

- **🔬 研究价值**：把跨载体问题从“统一参数拟合”推进到“上下文条件化策略”，很贴合主人关注的 embodied generalization 主线。
- **🚀 实践价值**：适合迁到无人机/UGV/机械臂等异构平台共享跟踪策略，能减少每种硬件单独训模型的成本。
- **📈 扩展潜力**：可继续往跨任务、跨传感器、跨动作接口扩展，也能与 PMI 或 action-space unification 体系结合。

## 🎯 可落地实验点

**实验设计**：在 UAV 与 UGV 共享的目标跟踪任务上复现“上下文编码 + 条件策略”范式。
- 对比基线：单一共享策略、每载体独立策略、显式 one-hot embodiment id 条件策略
- 度量指标：跟踪成功率、目标保持时长、zero-shot 新载体适配性能、样本效率
- 预期结果：上下文推断式策略应在未见载体与遮挡扰动场景中更稳，尤其适合主人后续空地迁移研究
