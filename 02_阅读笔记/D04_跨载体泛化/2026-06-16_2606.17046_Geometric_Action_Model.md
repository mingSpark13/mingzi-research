# 📖 花火格式笔记

## 🎯 题目

Geometric Action Model for Robot Policy Learning

## 📝 三句摘要

1. **问题背景**：现有 VLA / WAM 大多仍在 2D 图像或 2D latent 上学动作，3D 几何关系只能靠动作头隐式猜，导致接触操作、相机扰动和初始位姿变化下泛化不稳。
2. **核心方法**：论文提出 Geometric Action Model（GAM），把预训练 geometric foundation model 从中间层切开，前半做观测编码，中间插入 causal future predictor 结合语言、历史动作和本体状态预测未来几何 token，再回流后半 backbone 同时解码未来几何和 action chunk。
3. **关键结果**：GAM 在仿真与真实机械臂任务上同时拿到更高鲁棒性、更低延迟和更小模型体积；论文首页结果图显示其在 LIBERO-Plus camera 扰动和真实机成功率上分别达到约 65% 与 52.5%，优于 OpenVLA-OFT、Cosmos-Policy 等基线。

## 💎 价值评估

- **🔬 研究价值**：这篇很值得入 D04，因为它把“几何先验 + 未来预测 + 动作解码”塞进同一 backbone，像是在 VLA 和 world model 中间开了个更适合操作的折中点。
- **🚀 实践价值**：对主人关心的跨平台泛化很有启发——显式几何表征比单纯堆 2D foundation model 更稳，尤其适合相机位姿变化、抓取接触和多视角设置。
- **📈 扩展潜力**：后续可以追问这类几何 backbone 是否能外推到空地迁移、异构机械臂动作空间统一，甚至给主人自己的 PMI 接口提供更强几何状态中介。

## 🎯 可落地实验点

**实验设计**：在现有 manipulation/VLA 设定里，对比“纯 2D VLA”与“显式几何 backbone + future predictor”的差异，重点测视角扰动和初始位姿变化下的鲁棒性。
- 对比基线：OpenVLA-OFT、Cosmos-Policy、点云策略、传统视觉编码 VLA
- 度量指标：任务成功率、camera perturbation success、真实机成功率、推理延迟、模型参数量
- 预期结果：显式几何先验会显著提升跨视角和接触任务稳定性，并在小模型规模下保持更好泛化
