# 📖 花火格式笔记

## 🎯 题目

ROVE: Unlocking Human Interventions for Humanoid Manipulation via Reinforcement Learning

## 📝 三句摘要

1. **问题背景**：类人 VLA 在真实接触任务里常因全身运动学复杂、灵巧手控制困难而部署失稳，但人类接管示教本身也会犹豫、失误、动作低效，不能直接当最优专家轨迹吃进去。
2. **核心方法**：ROVE 提出一个 humanoid VLA post-training 框架，把部署 rollout、人工 intervention 和 cross-embodiment 人类经验视频放进统一经验池，用 Optimistic Value Estimation 学 critic，只把高价值恢复行为的 advantage 反哺给 actor，而不是盲目模仿全部干预动作。
3. **关键结果**：在真实接触型与细粒度类人操作任务上，ROVE 相比经验学习基线与 IIL/RL 后训练方法都更强，而且会随着多轮 rollout-intervention 持续提升，说明“不完美人类干预 + 价值筛选”比直接 imitation 更适合 humanoid 场景。

## 💎 价值评估

- **🔬 研究价值**：这篇很香，在 D04 里补上了“cross-embodiment human experience + humanoid intervention RL”这条线，把类人 VLA 后训练从纯示教拉向更真实的部署修正闭环。
- **🚀 实践价值**：对主人未来做异构平台/类人/空中操作都很有启发——关键不只是采 intervention，而是要识别哪些修正动作真的值学。
- **📈 扩展潜力**：后续可以把 ROVE 这种 critic-based 筛选机制接到跨载体数据飞轮、遥操作采集和空中操作恢复策略里，研究“失败恢复经验”如何跨平台复用。

## 🎯 可落地实验点

**实验设计**：在现有机器人后训练设定中构造 near-failure intervention 数据，对比“全量模仿干预”与“critic 选择高价值干预”的收益差异。
- 对比基线：纯 SFT、HG-DAgger / RECAP 类 intervention imitation、标准 RL post-training
- 度量指标：多轮迭代成功率、恢复成功率、无效干预比例、接触任务完成率
- 预期结果：价值筛选能减少低质量接管示教带来的噪声吸收，并在复杂 whole-body 任务上更稳定提升
