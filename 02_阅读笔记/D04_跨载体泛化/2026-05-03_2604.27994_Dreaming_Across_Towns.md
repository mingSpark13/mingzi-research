# 📖 花火格式笔记

## 🎯 题目

Dreaming Across Towns: Semantic Rollout and Town-Adversarial Regularization for Zero-Shot Held-Out-Town Fixed-Route Driving in CARLA

## 📝 三句摘要

1. **问题背景**：学习型驾驶/导航 agent 一换到没见过的新 town 就容易掉性能，说明 latent dynamics 学到的常常是场景记忆，不是真正可迁移的结构先验。
2. **核心方法**：论文在 Dreamer 风格世界模型上加入 semantic rollout 预测和 town-adversarial regularization，两者都只在训练时施加约束，让 latent state 更偏向可迁移语义而不是 town-specific 外观特征。
3. **关键结果**：在 CARLA 的 held-out town fixed-route 设定里，方法拿到更高的平均零样本成功率，说明“想象未来语义 + 压制场景域特征”确实能提升跨环境泛化。

## 💎 价值评估

- **🔬 研究价值**：虽然载体是自动驾驶，但问题本质和 D04 很像——如何让 world model 学到跨环境、跨分布可迁移的结构，而不是记住训练域纹理。
- **🚀 实践价值**：对明子的 UAV 跟随/导航任务很有借鉴意义，后续完全可以把 town 换成不同园区/楼宇/林地，把 town-adversarial 换成 scene-domain adversarial。
- **📈 扩展潜力**：能与 VLA/world model 路线拼起来，做“语义 rollout + 域对抗”的 cross-embodiment / cross-environment 泛化增强模块。

## 🎯 可落地实验点

**实验设计**：在无人机仿真导航中设置多个训练场景域与未见测试域，比较普通 latent world model 与带语义 rollout/域对抗正则的版本。
- 对比基线：Dreamer 类世界模型、无对抗正则版本、无语义 rollout 版本
- 度量指标：未见场景成功率、路线偏差、碰撞率、latent 域分类准确率、重规划频次
- 预期结果：加入语义 rollout 与域对抗后，latent 表征更不依赖特定场景纹理，跨场景零样本性能更稳
