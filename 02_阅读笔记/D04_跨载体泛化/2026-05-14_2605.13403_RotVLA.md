# 📖 花火格式笔记

## 🎯 题目

RotVLA: Rotational Latent Action for Vision-Language-Action Model

## 📝 三句摘要

1. **问题背景**：现有 latent action VLA 虽能处理异构数据，但离散量化动作空间常出现表征容量不足、重建退化和物理结构缺失的问题。
2. **核心方法**：RotVLA 用连续旋转群 SO(n) 建模 latent action，并结合 triplet frame learning 与 flow-matching action head，让 latent action 既保留时序动力学又能作为跨载体高层规划先验。
3. **关键结果**：在 1700+ 小时预训练下，RotVLA 以 1.7B 参数拿到 LIBERO 98.2%、RoboTwin2.0 89.6%/88.5%，并在真实操作任务中持续优于现有 VLA。

## 💎 价值评估

- **🔬 研究价值**：这篇直接打在“动作空间统一”核心点上，比单纯做微调策略更接近主人关注的跨 embodiment 真问题。
- **🚀 实践价值**：SO(n) latent action + flow matching 很适合给主人做统一接口/统一 planner 设计提供理论与工程参考，值得列为 D04 重点新作。
- **📈 扩展潜力**：后续可重点跟它与 LAP、RIO、PriorVLA 这条线的关系，看看 latent planner 是否能扩到无人机/机械臂跨平台数据。

## 🎯 可落地实验点

**实验设计**：在现有多平台操作数据上，做一个“离散 latent action vs 连续旋转 latent action”小规模预训练对比实验。
- 对比基线：VQ-style latent action、直接 robot action prediction
- 度量指标：迁移成功率、OOD 成功率、动作平滑性、latent 可插值性
- 预期结果：连续旋转 latent 在跨平台迁移和长时序动作平滑性上更优
