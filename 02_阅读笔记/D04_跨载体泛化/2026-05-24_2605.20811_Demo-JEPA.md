# 📖 花火格式笔记

## 🎯 题目

Demo-JEPA: Joint-Embedding Predictive Architecture for One-shot Cross-Embodiment Imitation

## 📝 三句摘要

1. **问题背景**：跨载体模仿学习常依赖共享动作空间或动作重定向，但人类与异构机器人之间很难直接对齐动作表示。
2. **核心方法**：论文把示范视为“未来目标”的隐式规格，在 JEPA 世界模型的共享预测表征空间中，把源示范翻译成目标载体可消费的潜在未来轨迹，再由目标机器人结合自身动力学完成规划执行。
3. **关键结果**：在 RLBench 与真实操控任务中，Demo-JEPA 只用视觉示范和目标载体自身经验，就能实现 one-shot cross-embodiment imitation，并在未见任务与未见载体配置上保持强泛化。

## 💎 价值评估

- **🔬 研究价值**：它把 D04 当前“共享动作接口”叙事向前推进了一步，证明跨载体迁移可以绕过显式动作对齐，转而依赖共享预测表征与未来子目标翻译。
- **🚀 实践价值**：对“人示范→机器人执行”或“地面/空中异构平台迁移”都很有启发，尤其适合作为少样本新载体冷启动方案。
- **📈 扩展潜力**：可继续和 D04 里的功能接口、paired demos、世界模型路线拼接，检验 latent subgoal translation 与 morphology-aware residual 是否互补。

## 🎯 可落地实验点

**实验设计**：把 Demo-JEPA 式潜在未来轨迹翻译，作为 D04 跨载体策略的上游子目标接口，与 shared-action / morphology-token / paired-demo 路线做并行对照。
- 对比基线：CEI、LAP、Data Analogies、DreamZero/WAM
- 度量指标：跨载体成功率、未见载体 zero/one-shot 迁移成功率、子目标预测误差、执行规划稳定性
- 预期结果：若 latent future translation 真能替代动作对齐，则在未见形态与动作空间不兼容场景下应显著优于 shared-action baselines
