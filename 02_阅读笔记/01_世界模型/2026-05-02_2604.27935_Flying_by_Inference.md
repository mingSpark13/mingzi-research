# 📖 花火格式笔记

## 🎯 题目

Flying by Inference: Active Inference World Models for Adaptive UAV Swarms

## 📝 三句摘要

1. **问题背景**：多机无人机轨迹规划若每次都重新做组合优化，在线代价高、可解释性差，也不利于复杂动态环境下的快速重规划。
2. **核心方法**：论文把多机任务分配改写为分层概率推理问题，用 Mission-Route-Motion 三层字典学习专家规划先验，再通过最小化 KL 异常度做在线任务分配、路径插入与碰撞规避。
3. **关键结果**：该方法在仿真与真实飞行轨迹验证中都能保持接近专家规划的结构，同时比改造版 Q-learning 更平滑、更稳定，说明 world model + active inference 很适合做可解释 UAV swarm autonomy。

## 💎 价值评估

- **🔬 研究价值**：它把 world model 从单机感知预测，推进到多机任务-路线-运动的层级先验建模，正中“可解释规划型世界模型”这条线。
- **🚀 实践价值**：对 AirSpark 这类低空数据闭环平台很有启发，可以把任务分配、航线生成、局部修正拆成可记录可学习的层次缓存，而不是全靠端到端黑盒策略。
- **📈 扩展潜力**：后续可把其符号层 world model 接到 VLA/语言任务接口，形成“语言目标→任务分解→多机执行”的无人机群体智能管线。

## 🎯 可落地实验点

**实验设计**：在多无人机仿真中复现 Mission-Route-Motion 三层字典，并比较“重跑优化器”与“world model 推理重规划”两类方法。
- 对比基线：传统 GA/MPC 规划器、Q-learning 多机分配器
- 度量指标：任务完成率、平均重规划时间、碰撞率、轨迹平滑度、专家行为相似度
- 预期结果：推理式 world model 在在线时延和稳定性上更优，尤其适合动态插单和局部环境扰动场景
