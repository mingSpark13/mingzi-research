## 🎯 题目

世界模型火了，但大多数人都搞错了！——从控制理论视角厘清 World Model 本质

## 📝 三句摘要

1. **问题背景**：World Model、Visual Planner、WAM/VAM 等名词在机器人操作领域混用，开发者常混淆它们的核心目标、训练差异和应用逻辑。
2. **核心论点**：World Model 的本质是**建模系统状态的演化过程**（即前向动力学），而非生成任务解决路径；Visual Planner 生成的是"解的轨迹"，WAM/VAM 则是两者的中间形态。
3. **关键洞察**：World Model 可用随机探索数据训练（数据来源广），而 Visual Planner/WAM/VAM 必须依赖稀缺的任务相关专家轨迹，这是它们本质的区别。

## 🔬 核心概念辨析

### World Model（世界模型）
- **定义**：显式或隐式地建模系统"状态-动作"演化关系的模型
- **显式方式**：`o_{t+k} = f(o_t, a_t)` 直接建模传感器观测演化
- **隐式方式**：通过编码器将高维观测压缩到低维隐空间再建模演化
- **训练数据**：任意状态/观测转移对 `o_t, a_t → o_{t+1}`，不挑数据，随机探索即可
- **核心能力**：可微，可反向传播；可用于在线规划（搜索+优化）

### Visual Planner（视觉规划模型）
- **建模目标**：不是演化模型，而是任务可达性问题的**解路径**
- **输入**：任务指令 + 当前观测图像
- **输出**：未来观测图像序列（即完成任务的动作路径）
- **训练数据**：必须是有任务语义且能通向成功集合的专家轨迹，数据稀缺
- **本质**：观测空间的 Policy，而非世界演化模型

### WAM/VAM（World Action Model / Video Action Model）
- 代表工作：Nvidia DreamZero、Cosmos Policy、ETH Mimic Video
- **联合生成**：将 Visual Planner + 逆动力学模型压缩为一个模型，联合生成路径和动作
- **与 World Model 的区别**：仍在专家数据上训练，不以建模开放式系统演化为目标
- **与 Visual Planner 的区别**：学到的是紧凑且任务相关的时空表征，sample efficiency 高

## 🛠️ World Model 辅助决策的两种方式

### 1. 紧凑的时空表征（Temporal-Spatial Representation）
- World Model 先用无监督方式学习环境的紧凑时空表征
- 策略网络直接利用这些表征（而非原始高维观测）来学习动作
- 表征 = 编码器压缩的观测特征（空间信息）+ RNN 记忆层（时间信息）
- **核心假设**：World Model 学到了系统演化模型 → 必然捕捉到了关键时空表征
- 代表论文：Recurrent World Models Facilitate Policy Evolution（NIPS 2018）

### 2. 在线规划（Online Planning）
**基于搜索**：
- 采样大量候选动作 → World Model 预测结果 → 价值函数评估 → 选最优
- 更高效的方式：在生成式策略的概率分布内采样（如多次采样 Diffusion Policy）

**基于优化（反向传播）**：
- World Model 可微 → 可计算动作对价值的梯度 → 梯度上升优化动作
- 初始动作 → World Model 预测结果 → 估计价值 → 反向传播求梯度 → 更新动作

**当前缺陷**：
- 计算开销大（需在线实时运行 World Model）
- Value Estimation / Reward Modeling 本身困难
- World Model 训练难度高，预测误差会传导到决策

## 💡 论文核心洞察与思考

### WAM/VAM vs World Model 的本质差距：数据
- WAM/VAM 依赖专家数据 → 分布局限，学不到通用动态演化
- DreamZero 强调：即使收集专家数据，也要有**多样性**而非重复性
- World Model 用随机探索数据 → 覆盖性更广、数据规模更大
- **未来方向**：随机探索数据预训练 + 任务专家数据微调

## 🎯 可落地实验点

**实验设计**：对比 World Model 与 Visual Planner 在空中机器人操作任务上的差异
- 构建任务：四旋翼穿越动态障碍场景
- World Model 分支：用随机飞行数据训练隐空间演化模型，在线规划辅助控制
- Visual Planner 分支：用专家穿越轨迹训练条件视频生成模型
- **对比维度**：样本效率、泛化能力、对未见过障碍的适应性
- **核心预期**：随机数据训练的 World Model 泛化性更强，专家数据训练的 Visual Planner 在分布内更精准
