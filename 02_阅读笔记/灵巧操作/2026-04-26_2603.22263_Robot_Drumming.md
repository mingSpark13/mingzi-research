# 📖 花火格式笔记

## 🎯 题目

In-Hand, Contact-Rich, and Long-Horizon Dexterous Robot Drumming

## 📝 三句摘要

1. **问题背景**：灵巧手在接触丰富、长时程且高速动态任务里很难稳定控制，尤其像击鼓这类需要连续碰撞与手内调整的任务。
2. **核心方法**：论文把机器人击鼓拆成手内持棒、接触建模和长时程节奏执行联合优化，用奖励显式约束 stick-finger 与 stick-drum 接触，并学习可持续多步演奏策略。
3. **关键结果**：方法在复杂接触和长序列执行上展示出稳定鼓点控制能力，说明高动态灵巧操作可以从“短时抓取”推进到“长时接触技能”。

## 💎 价值评估

- **🔬 研究价值**：把灵巧操作难点从静态抓取推进到长时程接触技能，对高动态 manipulation benchmark 很有启发。
- **🚀 实践价值**：其接触显式建模与节奏执行思路，可迁移到敲击、装配、插接等需要连续碰撞反馈的任务。
- **📈 扩展潜力**：后续可接触觉、视觉和策略分层，做更泛化的 contact-rich skill library。

## 🎯 可落地实验点

**实验设计**：在 Isaac/仿真灵巧手环境复现“连续敲击 + 姿态保持”任务，验证接触显式奖励对长时程稳定性的作用。
- 对比基线：纯模仿学习策略、无接触分解奖励的强化学习策略
- 度量指标：连续成功击打数、节奏误差、持棒稳定率、失手率
- 预期结果：显式接触建模在长时程稳定性和节奏精度上明显更优
