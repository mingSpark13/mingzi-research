# 📖 花火格式笔记

## 🎯 题目

CUBic: Coordinated Unified Bimanual Perception and Control Framework

## 📝 三句摘要

1. **问题背景**：双臂操作比单臂更依赖跨臂协调，但现有方法常在“各自独立”与“强行耦合”之间二选一，难兼顾感知解耦与协同控制。
2. **核心方法**：论文提出 CUBic，用共享 token 表征统一双臂感知与控制，通过单向感知聚合、双向感知协调和统一 diffusion policy，让独立性与协同性从结构中自然涌现。
3. **关键结果**：在 RoboTwin 基准上，CUBic 持续超过现有 visuomotor baseline，显著提升双臂协同精度与任务成功率。

## 💎 价值评估

- **🔬 研究价值**：这篇把双臂 coordination 问题从“控制耦合”改写成“统一感知建模”，视角很新，适合补强主人对双臂操作表征设计的理解。
- **🚀 实践价值**：如果后续主人做双机械臂协同装配或空中-地面协作操作，这种共享 token + diffusion policy 的设计值得当作强 baseline。
- **📈 扩展潜力**：可继续追它是否能扩到多机械臂/人形双手场景，也适合和 latent action / ACT 分块做结合。

## 🎯 可落地实验点

**实验设计**：在现有双臂或模拟双末端任务里，对比“独立单臂策略拼接”与“共享 token 协调策略”两种建模方式的成功率差异。
- 对比基线：单臂独立策略、显式 cross-attention 耦合策略
- 度量指标：双臂碰撞率、协同成功率、接触时序误差、任务完成时间
- 预期结果：统一表征策略在需要对时与协同接触的任务上更稳
