# 📖 花火格式笔记

## 🎯 题目

$χ_{0}$: Resource-Aware Robust Manipulation via Taming Distributional Inconsistencies

## 📝 三句摘要

1. **问题背景**：长程操作常受训练分布与真实执行分布不一致影响，尤其在资源受限平台上，策略容易在关键接触与协作步骤崩掉。  
2. **核心方法**：论文围绕 distributional inconsistencies 做资源感知建模，目标是在有限算力/数据条件下提升复杂操作任务的稳健性与泛化性。  
3. **关键结果**：从摘要与检索片段看，方法已在双臂协作衣物操作等长程任务中验证，说明它对高难 manipulation 的稳定部署有实际价值。  

## 💎 价值评估

- **🔬 研究价值**：它不是单纯提精度，而是直击“分布失配导致真实部署不稳”这个痛点，和主人后续做具身策略落地非常贴。  
- **🚀 实践价值**：适合拿来参考机械臂长程任务中的稳健训练、低资源部署和异常分布修正。  
- **📈 扩展潜力**：若把这套 inconsistency 处理思路迁到 UAV 感知-控制闭环，可能有助于提升 sim2real 与异常工况鲁棒性。  

## 🎯 可落地实验点

**实验设计**：在机械臂长程装配/布料任务中人为注入观测偏移，比较有无 distribution correction 的稳健性
- 对比基线：原始行为克隆策略、无资源约束的高算力策略
- 度量指标：成功率、恢复率、分布外样本性能、推理时延
- 预期结果：资源感知的分布修正方法在中低算力条件下更稳，且 OOD 退化更小
