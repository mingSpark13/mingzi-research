# 📖 花火格式笔记

## 🎯 题目

IMPACT: An Implicit Active-Set Augmented Lagrangian for Fast Contact-Implicit Trajectory Optimization

## 📝 三句摘要

1. **问题背景**：接触隐式轨迹优化 CITO 很适合统一处理 manipulation 和 locomotion，但 MPCC 数值病态严重，导致接触丰富任务里的规划与 MPC 很难又快又稳。
2. **核心方法**：论文提出 IMPACT，把 augmented Lagrangian 与 implicit active-set 识别结合，在优化迭代中动态识别隐式接触模态分支，并给出面向 trajectory optimization workload 的高效 C++ 实现。
3. **关键结果**：在开源 CITO benchmark 上相对强基线取得 2.9x–70x 加速（几何平均 13.8x），并在 dexterous manipulation 的 CI-MPC 仿真与真实 T 形物体 pushing 上验证了更好的控制质量。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯求解器调参，而是把 contact-implicit optimization 的核心瓶颈“接触模态分支识别”显式融进算法设计，挺硬。
- **🚀 实践价值**：对主人后面做接触丰富 manipulation、推物、插接、甚至 aerial manipulator 接触任务时，IMPACT 都能作为高质量 planning / MPC teacher。
- **📈 扩展潜力**：可继续探索把 IMPACT 产生的接触计划蒸馏进策略学习，或与 residual RL / VLA policy 做 planner-policy hybrid。

## 🎯 可落地实验点

**实验设计**：在一个接触丰富操作任务上比较 IMPACT 生成的轨迹/MPC 与常规接触优化器，测试其是否能提升 planner 质量和 teacher data 价值。
- 对比基线：传统 augmented Lagrangian CITO、SQP 类接触优化器、学习式接触策略
- 度量指标：求解时间、收敛率、接触稳定性、真实机任务成功率
- 预期结果：IMPACT 在不牺牲轨迹质量的前提下大幅降低求解时间，并为 imitation / RL 提供更干净的 contact-aware teacher
