# 📖 花火格式笔记

## 🎯 题目

OA-WAM: Object-Addressable World Action Model for Robust Robot Manipulation

## 📝 三句摘要

1. **问题背景**：世界动作模型（WAM）通过联合预测场景演化和机器人动作增强 VLA 策略，但现有方法将预测世界表示为整体图像/视频 token 流或全局隐变量，导致动作解码器难以定位指令引用的特定目标物体。
2. **核心方法**：提出 OA-WAM，将每帧分解为 N+1 个 per-slot 状态 token（1 个机器人 slot + N 个物体 slot），每个 slot 包含持久地址向量 addr_k 和时变内容向量 content_k^t，并与文本、图像、本体感觉和历史动作 token 融合为块因果序列；世界头预测下一帧 slot 状态，流匹配动作头在同一前向传播中解码 16 步连续动作 chunk。
3. **关键结果**：在 LIBERO（97.8%）和 SimplerEnv（79.3%）上匹配 SOTA VLA 和 WAM 基线；在 LIBERO-Plus 上于几何轴上达到新 SOTA，与 π₀.5 在七轴聚合上持平（83.9% vs 85.7%）。

## 💎 价值评估

- **🔬 研究价值**：提出对象可寻址的世界建模方式，解决了 WAM 中物体身份与周围上下文纠缠的问题；流匹配动作头 + 世界预测联合训练有方法论创新。
- **🚀 实践价值**：对主人的 UAV 人体跟随 + 数据闭环中感知-动作建模有直接参考价值；slot-based 对象表示可帮助无人机在复杂场景中稳定跟踪特定人体目标。
- **📈 扩展潜力**：可与 π0.5 层次化架构结合；地址向量机制可迁移到多目标跟踪场景；流匹配动作 chunk 对 UAV VLA 动作生成有参考价值。

## 🎯 可落地实验点

**实验设计**：借鉴 OA-WAM 的 slot 分解机制，为 UAV 人体跟随设计对象级注意力跟踪模块
- 对比基线：整体特征图 → bbox 回归 vs slot-based 目标分解 + 状态估计
- 度量指标：多目标遮挡场景下的跟踪成功率、人体姿态变化适应性
- 预期结果：对象可寻址表示提升 UAV 在复杂场景的人体跟踪鲁棒性
