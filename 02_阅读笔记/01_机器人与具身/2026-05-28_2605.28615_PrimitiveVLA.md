# 📖 花火格式笔记

## 🎯 题目

PrimitiveVLA: Disassemble and Assemble Primitives for Data-Efficient and Generalizable VLA

## 📝 三句摘要

1. **问题背景**：现有 VLA 常把指令直接映射为整段控制轨迹，容易过拟合任务级示范，导致数据效率低、长程泛化差。
2. **核心方法**：论文提出 PrimitiveVLA，用共享的 Multimodal Canonical Representation 先把示范自动拆成可复用动作原语，再在推理时结合 VLM planner 与 LLM switch 模块进行闭环组装执行。
3. **关键结果**：这种 primitive-centric 范式显著提升数据效率，并在未见任务与长程任务上优于传统 direct instruction-to-control VLA，说明“先拆后组”比整轨迹记忆更适合通用具身策略。

## 💎 价值评估

- **🔬 研究价值**：它不只是换 backbone，而是重构 VLA 的学习对象，从“整轨迹”改成“可复用原语”，很贴近主人关心的泛化本质问题。
- **🚀 实践价值**：对做机器人通用策略很有参考意义，尤其适合数据有限、任务组合多、希望支持长程执行与闭环纠错的场景。
- **📈 扩展潜力**：后续能与世界模型、高层任务规划、无人机 skill library 结合，形成统一的 primitive planner + executor 架构。

## 🎯 可落地实验点

**实验设计**：把现有操作/导航示范切成动作原语库，比较“直接模仿整轨迹”与“planner + primitive switch”两类策略在未见组合任务上的泛化。
- 对比基线：标准 VLA fine-tuning、ACT/trajectory chunking 策略、层次化 planner-policy 基线
- 度量指标：任务成功率、未见任务泛化、长程步骤完成率、重规划次数、样本效率
- 预期结果：primitive 组装式策略在长程与组合泛化上显著优于直接轨迹映射
