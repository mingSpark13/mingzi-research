## 🎯 题目

"Beyond Attention Magnitude: Leveraging Inter-layer Rank Consistency for Efficient Vision-Language-Action Models"

## 📝 三句摘要
提出 TIES，一种基于层间注意力排名一致性的动态 token 选择框架，在 CogACT+SIMPLER 基准上以 78% token 削减换来成功率提升 6%。

## 📝 三句摘要
1. VLA 模型因处理大量视觉 token 面临高推理延迟问题，现有 token 剪枝方法主要依赖注意力大小作为静态选择依据；
2. 论文发现高注意力 token 是任务依赖的，甚至可能导致策略性能下降，据此提出 TIES（Inter-layer Efficient Selection），通过层间 token 排名一致性动态平衡注意力量与排名稳定性；
3. TIES 无需额外训练，在 CogACT + SIMPLER 基准上平均成功率提升 6%，同时 token 使用量降低 78%。

## 💎 价值评估
⭐⭐⭐（中等偏高）
- 直接解决 VLA 推理效率问题，方向契合主人具身智能+空中manipulation 研究
- 实验只跑了两个基准，覆盖面有限，claim 强度中等
- 可考虑作为推理加速的 baseline 对比方法

## 🎯 可落地实验点
结合主人空中manipulation 场景，在 RT-2/OpenVLA 等 VLA 模型上验证 TIES 对无人机操控任务的推理加速效果，测量精度损失与延迟下降的 tradeoff。
