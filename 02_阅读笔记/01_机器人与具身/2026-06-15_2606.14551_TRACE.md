# 📖 花火格式笔记

## 🎯 题目

TRACE: Trajectory-Routed Causal Memory for Delayed-Evidence Visuomotor Imitation

## 📝 三句摘要

1. **问题背景**：长时序操作里，关键线索常在早期短暂出现，等到真正决策时已经看不见了，单帧或短历史策略容易选错分支。
2. **核心方法**：TRACE 用轨迹 path signature 给记忆寻址，把早期可见的任务证据写入固定大小 latent memory，并在后续歧义时刻按轨迹条件读取出来辅助决策。
3. **关键结果**：在真实长时程 manipulation 任务上，它比短历史和 recurrent memory 基线都更稳，显著提升 branch selection 与任务成功率。

## 💎 价值评估

- **🔬 研究价值**：这篇抓住了 delayed evidence 这个很实际但常被忽略的问题，对长程操作策略很有启发。
- **🚀 实践价值**：如果主人后面做多阶段操作或导航-操作串联，TRACE 这种轻量记忆头比重做整套 backbone 更容易接进现有策略。
- **📈 扩展潜力**：可和 VLA / 扩散策略结合，把记忆模块做成可插拔 adapter，专门补长时序上下文缺失。

## 🎯 可落地实验点

**实验设计**：在多阶段操作任务中构造“早看后选”的分支场景，比较无记忆、RNN 记忆、TRACE 记忆头的效果。
- 对比基线：短历史堆帧、recurrent memory、TRACE adapter
- 度量指标：branch selection accuracy、任务成功率、长期任务累计错误率
- 预期结果：TRACE 在出现视觉歧义的长任务中能明显降低后期选错分支的概率
