# 📖 花火格式笔记

## 🎯 题目

Switch: Learning Agile Skills Switching for Humanoid Robots

## 📝 三句摘要

1. **问题背景**：现有人形/腿足机器人强化学习通常能学会单一 locomotion skill，但在技能间实时切换时容易失稳，限制真实任务可用性。
2. **核心方法**：论文提出层次化多技能系统 Switch，用 Skill Graph 建模跨技能可行转移，再训练全身跟踪策略并配合在线调度器做图搜索执行。
3. **关键结果**：实验表明该方法能在保持运动模仿性能的同时，实现高成功率的敏捷技能切换，适合需要多动作连续切换的人形平台。

## 💎 价值评估

- **🔬 研究价值**：把“多技能动作库 + 在线切换可行性搜索”系统化，补上单技能 locomotion 到任务级敏捷行为之间的关键缺口。
- **🚀 实践价值**：对主人关注的人形/腿足平台很有参考意义，尤其适合需要跑、跳、转向、恢复等多技能切换的真实部署场景。
- **📈 扩展潜力**：后续可把 Skill Graph 与风险估计、地形感知、接触恢复策略结合，做更稳的长程具身任务执行。

## 🎯 可落地实验点

**实验设计**：在 Isaac/仿真腿足平台上复现“技能图 + 在线切换调度”，验证多技能切换是否优于单策略混训。
- 对比基线：单一统一策略、无图搜索的 skill selector、手工状态机切换
- 度量指标：切换成功率、跌倒率、恢复时间、能耗、轨迹跟踪误差
- 预期结果：Skill Graph + 在线调度在复杂技能序列中更稳，跌倒更少，长序列成功率更高
