# 📖 花火格式笔记

## 🎯 题目

Efficient Long-Horizon Vision-Language-Action Models via Static-Dynamic Disentanglement

## 📝 三句摘要

1. **问题背景**：VLA 在长时序任务里常被上下文长度和二次 attention 开销卡住，导致多帧整合不足、推理太慢。
2. **核心方法**：论文提出 SD-VLA，把视觉 token 拆成静态/动态两部分，只保留一份静态 token 并复用其 KV cache，必要时再用轻量 recache gate 更新。
3. **关键结果**：该方法在长程依赖 benchmark 上比基线成功率高 39.8%，在 SimplerEnv 上提升 3.9%，同时带来 2.26× 推理加速。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯堆大模型，而是直接戳中 VLA 真瓶颈——长时序信息冗余与推理成本，属于很实用的架构级优化。
- **🚀 实践价值**：主人后面做实时机器人控制或 UAV 端侧部署时，这种“静态背景缓存 + 动态局部更新”的思路很适合移植到低延迟策略系统。
- **📈 扩展潜力**：可继续与流匹配/扩散策略、事件触发更新、主动感知结合，做更长 horizon 且更省算力的具身策略。

## 🎯 可落地实验点

**实验设计**：在现有 VLA baseline 上复现“静态-动态 token 拆分 + KV cache 复用”，验证长时序操控与实时性收益。
- 对比基线：原始 VLA、只做帧采样压缩、SD-VLA
- 度量指标：任务成功率、平均推理延迟、GPU 显存占用、有效上下文长度
- 预期结果：SD-VLA 在长程任务中显著提速，并保住甚至提升成功率
