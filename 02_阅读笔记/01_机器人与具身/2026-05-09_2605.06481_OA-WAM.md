# 📖 花火格式笔记

## 🎯 题目

OA-WAM: Object-Addressable World Action Model for Robust Robot Manipulation

## 📝 三句摘要

1. **问题背景**：现有世界动作模型常用整图或全局 latent 表示场景，遇到场景扰动或指令涉及特定物体时，动作解码容易把“对象是谁”和“对象当前状态”混在一起。
2. **核心方法**：论文提出 OA-WAM，把每帧拆成机器人槽位与多个物体槽位，并用持久 address 向量和时变 content 向量分离身份与状态，再联合文本、图像、本体感知和历史动作做 block-causal 建模，同时一趟前向完成未来槽位预测与 16 步动作块解码。
3. **关键结果**：在 LIBERO 达到 97.8%、SimplerEnv 达到 79.3%，并在 slot intervention 测试中把 swap-binding cosine 提升到 0.87，说明对象可寻址表征显著增强了鲁棒操作能力。

## 💎 价值评估

- **🔬 研究价值**：把“对象身份可寻址”显式纳入 WAM/VLA 设计，比单纯堆模型容量更接近可解释且可干预的结构化世界模型路线。
- **🚀 实践价值**：对具身操作里常见的换背景、换摆放、换目标物体这类 OOD 扰动很有参考意义，尤其适合主人关注的泛化型操作策略。
- **📈 扩展潜力**：这套 slot-address 机制可以继续接到更长时域规划、交互式纠错、甚至空中操作里的多物体目标选择任务上。

## 🎯 可落地实验点

**实验设计**：把对象可寻址槽位机制移植到现有 manipulation world model 或 VLA 骨干里，测试跨场景/跨物体泛化。
- 对比基线：holistic WAM、普通 VLA、无 address reset 的 slot 版本
- 度量指标：任务成功率、跨场景泛化、对象交换干预后的 binding 一致性
- 预期结果：显式 addressable object state 在 OOD 场景和目标切换任务上明显优于整图 latent 表征
