# 📖 花火格式笔记

## 🎯 题目

Aerial Manipulation with Contact-Aware Onboard Perception and Hybrid Control

## 📝 三句摘要

1. **问题背景**：空中接触操作过去常依赖 MoCap 和位置控制，离开实验室后在接触瞬间容易漂、估计崩、力控不稳。
2. **核心方法**：论文提出接触一致性因子增强的机载 VIO、IBVS 视觉伺服，以及 hybrid force-motion controller，把“看见接触—估计接触—稳定施力”闭成纯机载回路。
3. **关键结果**：在无 MoCap 条件下，接触阶段速度估计提升 66.01%，并能稳定完成接触逼近与力保持，说明它把 deployable aerial manipulation 真往野外推了一步。

## 💎 价值评估

- **🔬 研究价值**：它补的是空中操作最稀缺的一环——不是单纯抓取或轨迹跟踪，而是接触瞬间的感知-控制耦合处理，这比只做 planner 更贴近真实系统落地。
- **🚀 实践价值**：对明子的无人机接触作业、贴靠检测、表面维护这类任务很有借鉴性，尤其是“只靠机载传感器闭环”这一点非常关键。
- **📈 扩展潜力**：后续可接入 AeroGrab 的抓取前端、AirVLA 的高层策略和物理 guidance，形成“高层决策 + 接触感知 + 力控执行”的完整空中操作栈。

## 🎯 可落地实验点

**实验设计**：在空中贴壁/按压任务中复现 contact-aware state estimation 对接触稳定性的收益。
- 对比基线：普通 VIO + 位置控制、普通 VIO + 混合力位控制、接触一致性 VIO + hybrid force-motion control
- 度量指标：接触建立时间、法向力误差、接触阶段速度估计误差、任务成功率
- 预期结果：加入接触一致性因子后，接触瞬间状态估计更稳，法向力振荡和脱离次数明显下降
