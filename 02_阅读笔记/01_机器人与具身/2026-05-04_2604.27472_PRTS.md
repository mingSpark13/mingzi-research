# 📖 花火格式笔记

## 🎯 题目

PRTS: A Primitive Reasoning and Tasking System via Contrastive Representations

## 📝 三句摘要

1. **问题背景**：现有 VLA 预训练大多还是行为克隆，只学“像不像专家”，却缺少对任务进度与目标可达性的内在判断，所以一到长程、接触密集、零样本指令就容易掉链子。
2. **核心方法**：PRTS 把语言指令当成 goal，用 goal-conditioned reinforcement learning + contrastive RL 从离线轨迹里学“当前状态动作离目标还有多远/是否可达”，再通过 role-aware causal mask 融入 VLM/VLA 主干。
3. **关键结果**：在 LIBERO、LIBERO-Pro、LIBERO-Plus、SimplerEnv 和 14 个真实任务上拿到 SOTA，尤其对长程任务、接触丰富任务和新指令泛化提升明显。

## 💎 价值评估

- **🔬 研究价值**：这篇不是简单扩数据或堆参数，而是把“目标可达性”这个强化学习视角塞进 VLA 预训练，正中主人关注的高层语义到低层执行之间那层缺失的时序理解。
- **🚀 实践价值**：很适合给主人后续做双系统机器人策略时参考——高层 planner 不一定只输出文字计划，也可以输出 goal reachability 信号辅助低层动作筛选。
- **📈 扩展潜力**：可继续往世界模型、阶段式策略、空地迁移方向嫁接，尤其适合和因果/隐空间 world model 结合，形成“能想、能判、能做”的统一框架。

## 🎯 可落地实验点

**实验设计**：在现有 manipulation/VLA 训练流里增加一个 contrastive goal-reachability 头，对比纯行为克隆与“行为克隆+可达性估计”的效果。
- 对比基线：π0 / OpenVLA 风格纯模仿学习策略、普通 goal-conditioned BC
- 度量指标：长程任务成功率、novel instruction 成功率、接触密集任务成功率、训练收敛步数
- 预期结果：加入 reachability supervision 后，策略在多阶段任务和分布外指令上更稳，且训练更快收敛
