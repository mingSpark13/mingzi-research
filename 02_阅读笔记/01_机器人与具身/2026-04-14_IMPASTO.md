# IMPASTO 机器人油画临摹系统

## 三句摘要

1. IMPASTO 是首个能仅凭目标油画图像，自动推断并执行笔画轨迹、力度、颜色的机器人油画临摹系统
2. 核心方法：学习像素级动力学模型（预测笔画动作对画布的影响）+ MPC 模型预测规划 + 7-DoF 力控机械臂执行
3. 仅通过机器人**自博弈（self-play）** 学习，无需人类 step-by-step 示范，也不需要精确的物理仿真器

## 核心价值评估

| 维度 | 评分 | 说明 |
|------|------|------|
| **创新性** | ⭐⭐⭐⭐ | 像素级动力学 + MPC + 力控的整合框架，无人类演示学习 |
| **工程完整度** | ⭐⭐⭐⭐⭐ | 完整系统：规划→控制→执行，画作质量令人惊叹 |
| **可复现性** | ⭐⭐⭐⭐ | 自监督学习，硬件需求明确（7-DoF + 力控） |
| **对主人的价值** | ⭐⭐⭐ | 非主研究方向，但像素动力学模型思想可参考；ICRA Arts in Robotics 风向标 |
| **启发实验点** | ⭐⭐⭐ | 像素dynamics model 可迁移到其他连续控制任务 |

## 可落地实验点

- **无人机画画**：用无人机挂载画笔，实现空中作画的端到端规划控制
- **自博弈数据采集**：用无人机 + 笔构建自博弈数据集，学习"笔触-效果"动力学
- **MPC + 像素预测**：用 learned pixel dynamics 做预测控制，可迁移到无人机轨迹生成

## 📝 详细笔记

### 背景与问题

油画临摹难点：
- **软刷 + 颜料**：刷子是变形的（deformable），力度不同效果差异巨大
- **颜料物理复杂**：无法用精确物理仿真器建模
- **无人类示范**：传统模仿学习需要 step-by-step 演示，收集成本高
- **多笔画规划**：需要规划数百笔画的前后叠加效果

核心问题：**给定一张目标油画图像，机器人能否自动推断并执行临摹所需的笔画轨迹、力度和颜色？**

### 方法架构

```
┌─────────────────────────────────────────────────────┐
│               IMPASTO 系统架构                        │
│                                                      │
│  目标图像（Target Oil Painting）                     │
│         ↓                                            │
│  ┌─────────────────────────────────────────────┐    │
│  │  MPC 规划器（Receding-Horizon MPC）          │    │
│  │  - 学习得到的像素动力学模型                  │    │
│  │  - 规划笔画参数（轨迹 + 力 + 颜色）          │    │
│  └─────────────────────────────────────────────┘    │
│         ↓                                            │
│  ┌─────────────────────────────────────────────┐    │
│  │  力控控制器（Force-Sensitive Controller）    │    │
│  │  - 7-DoF 机械臂                            │    │
│  │  - 感知实际接触力                            │    │
│  └─────────────────────────────────────────────┘    │
│         ↓                                            │
│  实际画布（Canvas）                                  │
│         ↓                                            │
│  图像观测（Pixel Observations）                      │
│         ↓                                            │
│  → 反馈给像素动力学模型，更新预测                    │
└─────────────────────────────────────────────────────┘
```

### 核心模块

#### 1. 像素动力学模型（Learned Pixel Dynamics Model）

**输入**：
- 当前画布图像观测 $o_t$
- 参数化笔画动作 $a_t$（笔刷位置、方向、力度、颜色）

**输出**：
- 预测下一时刻画布图像 $\hat{o}_{t+1}$

**关键洞察**：
- 不建模颜料物理，直接学习"动作→像素变化"的映射
- 类似"世界模型"，但输出是图像空间
- 通过机器人自博弈收集训练数据

#### 2. MPC 规划器（Receding-Horizon MPC）

**目标**：最小化预测画布与目标画布的差异

$$\min_{\tau} \sum_{t=0}^{H} \| \hat{o}_t - o^* \|^2 + \lambda \| a_t \|^2$$

**规划内容**：每笔的（位置 + 方向 + 力度 + 颜色）

**特点**：
- 滚动时域优化（receding horizon）
- 每执行完一笔，用实际观测更新模型

#### 3. 力控执行器（Force-Sensitive Controller）

- 7-DoF 机械臂
- 末端力传感器
- 控制刷子与画布接触力
- 补偿刷子变形

### 关键创新点

1. **无需人类演示**：完全通过自博弈学习
2. **无需精确仿真器**：学习数据驱动的像素动力学
3. **闭环规划**：MPC + 实际观测反馈，在线修正误差
4. **力控 + 视觉结合**：底层力控 + 上层规划

### 实验结果

- 在真实画布上复现多幅油画
- 复现精度显著优于基线方法
- 能够泛化到未见过的艺术风格
