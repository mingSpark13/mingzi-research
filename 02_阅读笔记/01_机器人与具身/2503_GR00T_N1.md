## 🎯 题目

GR00T N1: Dual-System Architecture for Humanoid Robot Foundation Model

## 📝 三句摘要

1. **问题背景**：humanoid/generalist robot 需要同时具备高层语义理解和实时动作生成能力，单一模型难以兼顾语义推理速度和动作控制频率。
2. **核心方法**：GR00T N1 采用 dual-system architecture：System 2（VLM）负责视觉-语言理解，System 1（diffusion transformer）负责实时动作生成；训练数据混合真实机器人轨迹、人类视频和合成数据。
3. **关键结果**：在多种 embodiment 仿真基准上超越 imitation learning baselines，在 Fourier GR-1 双臂平台上完成实际部署。

## 💎 价值评估

- **🔬 研究价值**：明确了 dual-system（语义脑+实时动作生成）是 humanoid 机器人的主流架构路线，diffusion transformer 作为 System 1 是关键技术选择。
- **🚀 实践价值**：对主人无人机+机械臂联合操控方向有参考价值，dual-system 思路可迁移。
- **📈 扩展潜力**：可与主人的 world model 方向结合，构建"world model + dual-system"具身 agent。

## 🎯 可落地实验点

**实验设计**：将 dual-system 思路用于无人机-机械臂联合控制
- System 2: world model 做语义规划
- System 1: diffusion/mpc 做实时动作生成
- 对比基线：单模型 end-to-end policy
- 度量指标：任务成功率、实时性（Hz）、语义理解准确率
