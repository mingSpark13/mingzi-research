# 📖 花火格式笔记

## 🎯 题目

PRIOR: Perceptive Learning for Humanoid Locomotion with Reference Gait Priors

## 📝 三句摘要

1. **问题背景**：感知型人形越障 locomotion 往往训练链条又长又脆，常常依赖多阶段训练、对抗目标或大量真机校准，复现成本很高。
2. **核心方法**：PRIOR 基于 Isaac Lab，把 mocap 导出的参数化 gait prior、GRU 地形状态估计器和 terrain-adaptive footstep reward 组合起来，单阶段学出具有人形步态的感知 locomotion 策略。
3. **关键结果**：在 stairs、boxes、gaps 等多类复杂地形上，完整框架达到 100% 穿越成功率，还把感知分辨率与实时性权衡讲清楚，复现实用性很强。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯刷分，而是把“参考步态先验 + 感知估计 + 奖励设计”收成一条可复现路线，对 D07 研究很像一篇能拿来拆模块学习的工程模板。
- **🚀 实践价值**：主人后面若做 Isaac Lab 上的复杂 locomotion/全身控制，PRIOR 的 reward 和 estimator 拆法非常适合直接借到自己的训练底座里。
- **📈 扩展潜力**：它现在主要是腿足 humanoid，但方法能外推到四足、空地混合平台，甚至空中机械臂的“参考轨迹 + 感知补偿”训练框架。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 上复现“参考轨迹先验 + 感知估计器 + 自适应足步奖励”的三件套，再迁到主人关注的复杂 locomotion 或浮基控制任务上。
- 对比基线：无 gait prior、无 GRU estimator、无 adaptive footstep reward、传统多阶段训练
- 度量指标：地形穿越成功率、步态稳定性、能耗、推理时延、感知分辨率-性能曲线
- 预期结果：带 gait prior 的单阶段训练更稳，感知估计器能显著提升复杂地形成功率，而 reward 设计决定最后是否跑得像“人”而不只是“能过”
