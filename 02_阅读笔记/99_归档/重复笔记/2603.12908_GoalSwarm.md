# 📖 花火格式笔记

## 🎯 题目

GoalSwarm: Multi-UAV Semantic Coordination for Open-Vocabulary Goal Navigation

## 📝 三句摘要

1. **问题背景**：多无人机在未知环境中进行开放词汇语义目标导航时，单机感知视野有限，难以高效协调完成任务。
2. **核心方法**：提出GoalSwarm框架，通过多无人机协作实现语义协调导航，各无人机共享环境语义信息并协同规划路径。
3. **关键结果**：在仿真和真机实验中，多机协作相比单机导航成功率提升显著，尤其在遮挡和复杂地形场景。

## 💎 价值评估

- **🔬 研究价值**：⭐⭐⭐ 多机协同+语义导航结合紧密，直接补充D06研究空白
- **🚀 实践价值**：真机验证充分，工程参考价值高
- **📈 扩展潜力**：可扩展至异构多机、动态任务重分配

## 🎯 可落地实验点

**实验设计**：复现GoalSwarm多机协调模块
- 对比基线：单机语义导航（不带协作）
- 度量指标：任务成功率、路径效率、通信开销
- 预期结果：验证协作带来的导航性能提升
