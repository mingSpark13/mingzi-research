# 📖 花火格式笔记

## 🎯 题目

Weather-Robust Cross-View Geo-Localization via Prototype-Based Semantic Part Discovery

## 📝 三句摘要

1. **问题背景**：GNSS 受干扰时，无人机依赖 cross-view geo-localization 做定位，但现有方法容易把纹理和布局揉成单个全局向量，遇到天气扰动和高度变化就掉性能。
2. **核心方法**：论文提出 SkyPart，在 patch-based ViT 上加入 prototype semantic part grouping、训练期 altitude-conditioned modulation、graph-attention readout 与 uncertainty-weighted multi-objective loss，显式把布局、语义部件与天气鲁棒性拆开建模。
3. **关键结果**：在 SUES-200、University-1652 和 DenseUAV 上，SkyPart 以 26.95M 参数、22.14 GFLOPs 刷新单次检索协议下的 SOTA，且在 WeatherPrompt 十种天气扰动下优势进一步扩大。

## 💎 价值评估

- **🔬 研究价值**：这篇对 D03 很关键，因为它不是单纯追求更大的 backbone，而是针对空地视角差、天气变化、高度尺度偏移做结构化修正。
- **🚀 实践价值**：主人若考虑 GPS-denied 场景的 UAV 定位或空地协同，SkyPart 这种“语义部件 + 高度去偏置”思路很适合接到跨视角定位链路里。
- **📈 扩展潜力**：后续可把 semantic part prototype 与语言目标、区域约束甚至导航先验结合，往空地联合规划扩展。

## 🎯 可落地实验点

**实验设计**：在现有 cross-view geo-localization baseline 上加入 prototype part discovery 与 altitude-conditioned modulation，验证恶劣天气和高度变化下的鲁棒性收益。
- 对比基线：全局描述子 CVGL baseline、去掉 altitude modulation 的 SkyPart、去掉 prototype grouping 的 SkyPart
- 度量指标：Recall@1、Recall@5、天气扰动下性能下降幅度、推理时延
- 预期结果：SkyPart 在高度变化和天气腐蚀场景下显著降低定位性能塌缩
