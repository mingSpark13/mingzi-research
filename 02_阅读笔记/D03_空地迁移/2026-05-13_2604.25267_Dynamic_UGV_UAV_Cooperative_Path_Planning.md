# 📖 花火格式笔记

## 🎯 题目

Dynamic UGV-UAV Cooperative Path Planning in Uncertain Environments

## 📝 三句摘要

1. **问题背景**：灾后道路存在未知阻断时，单一地面车很难在信息不全条件下安全快速到达目标，急需空地协同的动态路径规划框架。
2. **核心方法**：论文提出 DUCPP 框架，让一个或多个 UAV 先行巡检不确定边，实时剔除不可通行道路，并比较多种 UGV-UAV 协同策略，重点包含双向搜索式协同方案。
3. **关键结果**：作者在 100 个城市路网中验证发现双向策略整体最优，多 UAV 能进一步缩短 UGV 通行时间，但会带来更高计算开销。

## 💎 价值评估

- **🔬 研究价值**：这篇不是纯感知堆料，而是很清楚地切中 D03 的空地协同规划问题，把“先验未知道路状态 + 在线巡检 + 动态重规划”串成了完整闭环。
- **🚀 实践价值**：对灾后补给、救援运输、复杂园区巡检很直接，尤其适合主人后续思考 UAV 为 UGV/机器人提供先验探路信息的协同系统。
- **📈 扩展潜力**：后续可以把路网级协同扩展到连续空间、视觉感知输入、学习式边代价预测，甚至嫁接到空地跨视角定位和多机器人任务分配里。

## 🎯 可落地实验点

**实验设计**：复现“UAV 先验探路辅助 UGV 重规划”原型，并加入学习式边通行概率预测。
- 对比基线：纯 UGV 最短路、单 UAV 辅助、论文双向协同策略
- 度量指标：到达成功率、总通行时间、巡检成本、重规划次数
- 预期结果：在不确定边比例升高时，双向协同 + 概率预测会比纯规则探测更稳，能更早排除高风险路径
