# 📖 花火格式笔记

## 🎯 题目

An End-to-end Flight Control Network for High-speed UAV Obstacle Avoidance based on Event-Depth Fusion

## 📝 三句摘要

1. **问题背景**：高速 UAV 在复杂障碍环境中仅靠单一传感器很难兼顾静态障碍感知和动态目标响应。
2. **核心方法**：论文把深度图与事件相机做双向 cross-attention 特征融合，并用 SPS 专家规划器生成高质量监督，训练端到端模仿学习飞控网络。
3. **关键结果**：方法在 17m/s 高速下取得 70–80% 成功率，较单模态或单向融合模型提升 10–20%，同时专家规划复杂度降到 O(n)。

## 💎 价值评估

- **🔬 研究价值**：这篇把 event-depth fusion 和高速飞控真正绑在一起，补上 D03 里“高速避障感知不足”这块缺口。
- **🚀 实践价值**：对主人做高速无人机感知控制特别有参考价值，尤其适合借鉴事件相机与深度模态互补设计。
- **📈 扩展潜力**：后续可叠加 RL 微调、安全屏障或语义目标导航，发展成兼顾速度、鲁棒和任务性的统一飞行策略。

## 🎯 可落地实验点

**实验设计**：实现 depth-only / event-only / event+depth 三组飞控策略，对比高速混合障碍环境下的成功率和延迟表现。
- 对比基线：depth-only policy、event-only policy、传统局部规划器
- 度量指标：成功率、平均速度、碰撞率、控制延迟、跨场景稳定性
- 预期结果：双模态融合在高速动态障碍场景下显著优于单模态，并减少 motion blur 导致的失控
