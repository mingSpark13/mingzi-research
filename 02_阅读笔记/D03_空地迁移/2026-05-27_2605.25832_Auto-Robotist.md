# 📖 花火格式笔记

## 🎯 题目

When Search Becomes Memory: Turning Robot Design Trials into Transferable Skills

## 📝 三句摘要

1. **问题背景**：多数进化式机器人形态搜索把仿真试错只当作一次性优化信号，缺少可复用、可审计的结构知识沉淀。
2. **核心方法**：论文提出 Auto-Robotist，让 LLM agent 把 morphology search 轨迹整理成自然语言 skill library，存储结构原型、正反规则和支持案例，并在后续搜索中检索调用。
3. **关键结果**：在 7 个 EvoGym 任务上，方法显著提升冷启动 5x5 搜索效果，并成功把学到的设计规则迁移到 10x10 设计空间，全面超过纯 GA 基线。

## 💎 价值评估

- **🔬 研究价值**：这篇把“搜索过程本身就是记忆资产”讲得很清楚，和主人关心的 agent memory、跨载体泛化、可审计研究循环天然同频。
- **🚀 实践价值**：很适合迁到无人机/机器人结构设计、策略搜索、甚至数据飞轮里，把失败实验显式沉淀成可检索规则，而不是埋在日志里发霉。
- **📈 扩展潜力**：后续可继续接结构搜索 + 策略学习联合优化，也能与 PMl/跨载体接口、自动实验记录系统形成一套闭环。

## 🎯 可落地实验点

**实验设计**：给主人现有实验管理流程加一个“失败实验→规则摘要→下轮检索提示”的 memory layer，验证是否能减少盲目 ablation。
- 对比基线：无记忆的试错搜索、只保留 top-k 结果、加入自然语言 skill memory 的搜索
- 度量指标：达到目标性能所需轮数、无效实验比例、跨任务迁移成功率、规则复用率
- 预期结果：显式 skill memory 能降低重复踩坑次数，并加快新形态/新任务冷启动效率
