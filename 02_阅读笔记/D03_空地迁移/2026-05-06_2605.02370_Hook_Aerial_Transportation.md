# 📖 花火格式笔记

## 🎯 题目

Robust Adaptive Predictive Control for Hook-Based Aerial Transportation Between Moving Platforms

## 📝 三句摘要

1. **问题背景**：带挂钩抓具的无人机在移动平台之间做自主取放时，既有复杂耦合动力学，又会受到气动与载荷不确定性影响，常规控制很容易失稳或违约束。
2. **核心方法**：论文构建了基于 MuJoCo 数字孪生的预测模型，并把 zoRO 不确定性传播与 EKF 参数估计并入 robust adaptive MPC，兼顾实时性、鲁棒性与约束满足。
3. **关键结果**：方法在复杂仿真与真实飞行实验中都完成了移动平台间的 hook-based aerial transportation，说明这套“数字孪生 + 自适应鲁棒 MPC”对真实空中操作链路有效。

## 💎 价值评估

- **🔬 研究价值**：把空中操作从“定点抓放”推进到“移动平台对移动平台”的动态交互，适合补强 D03 里空中平台动态作业这条支线。
- **🚀 实践价值**：对无人机补给、海上/车载起降协同、灾害场景跨平台递送都很实用，尤其适合高风险低容错任务。
- **📈 扩展潜力**：后续可把 VLA/语义决策负责高层目标分解，让这类鲁棒 MPC 作为低层 skill primitive，形成语言到控制的闭环。

## 🎯 可落地实验点

**实验设计**：复现“移动平台间挂钩运输”任务，并比较是否加入在线参数估计与鲁棒传播。
- 对比基线：nominal MPC、仅EKF自适应MPC、本文 robust adaptive MPC
- 度量指标：任务成功率、最小安全距离、轨迹误差、约束违反率、单次规划时延
- 预期结果：鲁棒传播 + 在线估计在存在载荷扰动和平台速度变化时能明显降低失败率与违约束次数
