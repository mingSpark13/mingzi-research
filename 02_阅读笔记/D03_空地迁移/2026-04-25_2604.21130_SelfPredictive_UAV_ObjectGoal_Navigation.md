# 📖 花火格式笔记

## 🎯 题目

Self-Predictive Representation for Autonomous UAV Object-Goal Navigation

## 📝 三句摘要

1. **问题背景**：UAV 的 object-goal navigation 不仅要解决 3D 搜索目标未知位置的问题，还长期受 RL 样本效率低拖累。
2. **核心方法**：论文提出自预测表征模型 AmelPred/AmelPredSto，把感知编码成更利于 actor-critic 学习的状态表示，再用于 3D OGN 策略训练。
3. **关键结果**：实验显示随机版 AmelPredSto 与 actor-critic 结合时效果最好，显著提升了 OGN 任务中的训练效率与性能。

## 💎 价值评估

- **🔬 研究价值**：它不只是做一个 UAV RL 小技巧，而是在“如何给空中平台学到更迁移友好的状态表示”上提供了可复用思路。
- **🚀 实践价值**：若主人后续做空中迁移 / 空地共表征，这类 self-predictive latent 可作为轻量感知前端。
- **📈 扩展潜力**：可继续验证它能否迁到 language-conditioned navigation，或与 VLA latent / world model latent 对齐。

## 🎯 可落地实验点

**实验设计**：在同一 3D UAV 导航任务上，对比原始像素输入、常规自编码器表征、AmelPredSto 表征，再测试跨场景泛化与少样本迁移。
- 对比基线：PPO/SAC + raw RGB，常规 SRL encoder，goal-conditioned latent policy
- 度量指标：成功率、收敛步数、sample efficiency、跨场景泛化差值
- 预期结果：AmelPredSto 在样本效率和新场景目标搜索稳定性上优于普通表征
