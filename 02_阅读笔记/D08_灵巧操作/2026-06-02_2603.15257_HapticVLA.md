# 📖 花火格式笔记

## 🎯 题目

HapticVLA: Contact-Rich Manipulation via Vision-Language-Action Model without Inference-Time Tactile Sensing

## 📝 三句摘要

1. **问题背景**：接触丰富操作里，VLA 如果缺少推理时触觉输入，往往既不安全也不稳定，而额外触觉硬件又提高了复现与部署门槛。
2. **核心方法**：论文先用带安全触觉奖励的 flow-matching 教师策略离线学出触觉感知动作分布，再通过 Tactile Distillation 把紧凑触觉 token 蒸馏给只看视觉和状态的学生 VLA，实现“训练时学触觉、部署时不用触觉”。
3. **关键结果**：方法在接触丰富操作中显著改善动作安全性与任务成功率，说明触觉知识可以作为离线监督信号注入 VLA，而不必依赖推理时实时触觉硬件。

## 💎 价值评估

- **🔬 研究价值**：这篇把“触觉是否必须在线输入”拆成了更细的问题，提出离线触觉蒸馏路线，对低成本 VLA 落地很有启发。
- **🚀 实践价值**：如果主人后面做真实机器人或 UAV 末端接触任务，这种“训练时多模态、部署时轻量化”的思路很适合受限平台。
- **📈 扩展潜力**：可以继续接到 world model、latent action、接触预测器里，研究视觉是否能内隐恢复接触状态并支持更长时程操作。

## 🎯 可落地实验点

**实验设计**：在现有 VLA / diffusion policy 流程中加入“离线触觉教师 + 视觉学生”蒸馏支路，验证无触觉推理是否仍保留接触安全性。
- 对比基线：vision-only VLA、online tactile VLA、reward-weighted diffusion / flow-matching baseline
- 度量指标：任务成功率、过力/滑移次数、接触恢复能力、推理延迟
- 预期结果：蒸馏版视觉学生在不加硬件的前提下逼近触觉教师，并显著优于纯视觉基线
