# 📖 花火格式笔记

## 🎯 题目

Zero-Shot Long-Horizon Dexterous Manipulation via Multi-View 3D-Grounded VLM Reasoning

## 📝 三句摘要

1. **问题背景**：长程灵巧操作在未见物体/工具/场景下的零样本执行仍是开放问题，单视角 RGB-D grounding 在遮挡、参考帧漂移和工具轨迹对齐上鲁棒性不足；现有端到端 VLA 又需要大量示教，难以真正 zero-shot。
2. **核心方法**：论文提出一个 zero-shot 框架——VLM 在多视角标定 RGB 上分别产出 reference-frame 任务 grounding 与 primitive 级 2D keypoint，再通过 triangulation + reference-view ray voting 提升到 3D；对工具使用，检索 object-centric atomic action 并把存储的 6D 工具轨迹对齐到当前场景；对灵巧执行，把 lift 出的 grasp keypoint 扩展成 task-conditioned grasp affordance region，并由 arm-hand motion generator 生成 feasible grasp-motion pair；执行阶段用闭环 status verification + replan 支持长程任务。
3. **关键结果**：真实机器人实验显示，在 3D grounding 精度与执行可靠度上均优于单视角 RGB-D grounding 与 fine-tuned VLA baseline；并在未见物体与新场景中实现 long-horizon 任务（含 pick-and-place 与工具使用）的 zero-shot 执行。

## 💎 价值评估

- **🔬 研究价值**：把"多视角 VLM 3D grounding + 6D 工具轨迹检索 + grasp affordance region + 闭环 replan"拼成一条可解释、可拆解的 zero-shot 灵巧操作 pipeline，绕开了端到端 VLA 对示教的依赖，为 D08 提供了一种"系统化 zero-shot"新范式。
- **🚀 实践价值**：reference-view ray voting 与 multi-view triangulation 的几何一致性检查对遮挡场景特别有用，对主人关心的"未见过物体 + 真实家庭/工坊杂乱环境"操作有直接迁移价值；闭环 status verification + replan 也能直接接到 D01 的"世界模型作为部署前/部署中监督器"主叙事上。
- **📈 扩展潜力**：可进一步引入 affordance-conditioned contact graph、世界模型预测 rollout 来替代 rule-based status verification，把"3D grounding → motion"扩展为"3D grounding → contact-aware plan → motion"。

## 🎯 可落地实验点

**实验设计**：在 D08 现有灵巧操作仿真/真机平台（如 Adroit / DexArt / 真机灵巧手）上对比"单视角 VLM grounding" vs "多视角 VLM grounding + ray voting" vs "多视角 VLM grounding + ray voting + 闭环 replan"三档零样本长程任务完成度。
- 对比基线：单视角 RGB-D grounding、fine-tuned VLA (OpenVLA / π0 / HapticVLA 等)、rule-based grasp planner
- 度量指标：未见物体零样本成功率、长程任务阶段完成率 (sub-goal SR)、单步 3D grounding 误差 (cm / degree)、闭环 replan 触发次数与恢复成功率
- 预期结果：多视角 + ray voting 应在遮挡/新物体上显著拉低 3D grounding 误差；闭环 replan 应在长程任务（≥5 step）上把整体成功率从 ~30% 拉升至 ~60%+
