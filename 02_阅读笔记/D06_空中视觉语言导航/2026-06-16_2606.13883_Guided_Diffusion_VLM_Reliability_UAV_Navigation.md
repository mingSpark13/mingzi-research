# 📖 花火格式笔记

## 🎯 题目

Guided Diffusion with Distilled Vision-Language Reliability for Aerial Navigation

## 📝 三句摘要

1. **问题背景**：端到端 UAV 扩散规划虽然能省掉感知-建图-规划接口误差，但会把玻璃、镜面、过曝区域当成可信输入，导致真实飞行里撞障风险很高。
2. **核心方法**：论文把视觉语言模型对“哪里不可信”的开放词汇判断蒸馏成轻量 reliability student，再把 reliability heatmap 与 RGB/几何上下文一起送进 diffusion planner，并用两阶段 ESDF cost 在去噪时同时规避实体障碍和虚拟不可信区域。
3. **关键结果**：在仿真和真实四旋翼实验中，方法把 obstacle-violation rate 从 40.3% 压到 9.6%，路径平均可靠度从 0.588 提到 0.925，且蒸馏后推理速度相比直接跑 VLM 最多快 2 倍。

## 💎 价值评估

- **🔬 研究价值**：这篇很对 D06 主线，不只是做 VLN 语言理解，而是把“感知可靠性”显式塞进扩散式轨迹生成，补上 UAV 真实部署里最容易被忽略的失败源。
- **🚀 实践价值**：reliability map + ESDF guidance 这套接口很适合迁到主人的单目/弱深度导航链路，尤其适合玻璃门、反光地面、强曝光走廊这类真实坑点。
- **📈 扩展潜力**：后续可以继续追问是否能把 reliability 信号并入 world model latent、语言子目标分解、或多视角机载感知，从而做更稳的长程空中 VLN。

## 🎯 可落地实验点

**实验设计**：在现有 UAV 导航/空中 VLN 设定里加入“可靠性热图引导的扩散规划”，验证它是否能减少感知异常区域导致的撞障。
- 对比基线：普通 diffusion planner、AerialVLA、FLIGHT VLA、AgenticDiffusion
- 度量指标：Success Rate、obstacle-violation rate、trajectory reliability、重规划次数、真实机部署成功率
- 预期结果：在镜面/玻璃/强反光场景中，reliability-guided planner 的安全性和真实可迁移性显著更稳
