# 📖 花火格式笔记

## 🎯 题目

Think Like a Pilot: Fine-Grained Long-Horizon UAV Navigation

## 📝 三句摘要

1. **问题背景**：现有 UAV VLN 多停留在离散动作或短时原子动作层面，缺少能执行长时程语义指令且输出连续可飞控制量的统一任务设定。
2. **核心方法**：论文提出 FLIGHT 基准，覆盖细粒度 VLN 与长时程 flow 两类任务，并设计异步 FLIGHT VLA，把低频 Streaming Pilot VLM 负责任务状态推理、高频 diffusion action model 负责连续控制。
3. **关键结果**：在闭环评测中，FLIGHT VLA 相比代表性 VLN/VLA 基线取得更强的多阶段任务完成率、子目标遵循和终点控制表现，同时其 Pilot Reasoning VLM 还能提升 UAV 视频推理能力。

## 💎 价值评估

- **🔬 研究价值**：这篇直接补上 D06 最缺的“长时程语义导航 + 连续控制 + 任务推理”交叉空白，不只是又一篇导航模型，而是连 benchmark 和监督信号都一起补齐了。
- **🚀 实践价值**：异步高层推理/低层控制架构很适合主人后续空中具身系统，把慢速语义决策和高速飞控接口拆开，工程上更容易落地。
- **📈 扩展潜力**：后续可继续接入真实机载感知、空中 tracking 或 manipulation 任务，也适合作为空中 VLA 系统的公共评测锚点。

## 🎯 可落地实验点

**实验设计**：复现 FLIGHT VLA 的异步架构，并对比同步式 VLA 或纯 VLN 方案在长时程导航中的表现。
- 对比基线：传统 VLN policy、同步式 VLA、去掉 Pilot Reasoning 的 diffusion control 版本
- 度量指标：任务完成率、子目标命中率、终点误差、控制平滑度、单步推理延迟
- 预期结果：异步设计在长程指令执行与实时控制稳定性上更优，尤其在多阶段任务下优势更明显
