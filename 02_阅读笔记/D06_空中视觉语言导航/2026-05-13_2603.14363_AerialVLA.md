# 📖 花火格式笔记

## 🎯 题目

AerialVLA: A Vision-Language-Action Model for UAV Navigation via Minimalist End-to-End Control

## 📝 三句摘要

1. **问题背景**：UAV 的视觉语言导航需要同时做语言理解、3D场景感知和连续控制，现有分层方案依赖 oracle guidance 或额外检测器，真实自主性不足。
2. **核心方法**：论文提出 AerialVLA，用双视角精简感知 + 机载传感器生成的模糊方向提示，把原始视觉和语言指令端到端映射到 3-DoF 控制与内置降落信号。
3. **关键结果**：在 TravelUAV 基准上达到 seen 场景 SOTA，并在 unseen 场景上把成功率拉到主流基线的近 3 倍，说明“少模块、强自主”的 VLA 路线更稳。

## 💎 价值评估

- **🔬 研究价值**：直接击中 D06 空中视觉语言导航主线，把 oracle-free、自主提示、连续控制三件事合到一个极简 VLA 范式里。
- **🚀 实践价值**：双视角输入和统一控制空间很适合后续往 sim2real、机载算力约束、着陆任务一体化延展。
- **📈 扩展潜力**：可继续追问 world model / action chunking / 历史记忆是否能进一步提升 unseen 泛化与长程指令稳定性。

## 🎯 可落地实验点

**实验设计**：在现有 UAV-VLN 管线里复现“模糊方向提示 + 端到端连续控制”这两个关键改动，验证是否真比分层模块更稳。
- 对比基线：OpenFly、VLN-Pilot、SpatialFly、带 oracle guidance 的分层方法
- 度量指标：Success Rate、SPL、unseen 场景成功率、精确着陆成功率
- 预期结果：去掉 oracle guidance 后 seen 可能略波动，但 unseen 泛化和真实部署一致性更强
