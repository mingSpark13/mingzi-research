# 📖 花火格式笔记

## 🎯 题目

What Limits Vision-and-Language Navigation ?

## 📝 三句摘要

1. **问题背景**：现有 VLN 智能体从仿真走向真实世界时，常被光照变化、运动模糊和指令欠明确拖垮，真正瓶颈不是模型越大越好，而是缺少稳定的空间 grounding。
2. **核心方法**：论文提出 StereoNav，用 stereo vision 把语义与几何统一起来，并引入跨域保持稳定的 Target-Location Priors 作为 sim-to-real bridge，让动作预测持续锚定目标位置。
3. **关键结果**：在 R2R-CE 与 RxR-CE 上，StereoNav 以更少参数和更少训练数据拿到 SOTA 的 egocentric RGB 指标（81.1/68.3 与 67.5/52.0 的 SR/SPL），且真实机器人部署也显著提升导航稳定性。

## 💎 价值评估

- **🔬 研究价值**：这篇直接点中 D06 最核心的痛点——语言导航不是只靠大模型规模，而是要有稳的几何 grounding 与跨域先验，对后续空中 VLN 设计很有指导性。
- **🚀 实践价值**：stereo + target prior 的组合很适合给主人当前无人机导航路线做“显式目标约束 + 感知稳健性”增强，不必一上来就依赖重型 VLA。
- **📈 扩展潜力**：可继续把 target prior 与语言指令、场景图谱、动态障碍建模结合，往真正开放环境 UAV-VLN 迁移。

## 🎯 可落地实验点

**实验设计**：在主人现有 UAV 导航栈中增加 stereo 几何分支与目标位置先验，对比单目 RGB-only 导航在光照扰动和模糊场景下的鲁棒性。
- 对比基线：AutoFly、单目深度导航基线、去掉 target prior 的 StereoNav 变体
- 度量指标：成功率、SPL、碰撞率、重规划次数、真实飞行稳定性
- 预期结果：带 target prior 的 stereo 方案在真实场景里明显减少漂移与语义歧义导致的失败
