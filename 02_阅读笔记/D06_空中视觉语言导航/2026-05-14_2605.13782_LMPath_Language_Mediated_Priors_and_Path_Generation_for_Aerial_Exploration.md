# 📖 花火格式笔记

## 🎯 题目

LMPath: Language-Mediated Priors and Path Generation for Aerial Exploration

## 📝 三句摘要

1. **问题背景**：传统无人机搜索多靠几何覆盖，几乎不利用“目标通常会出现在哪”的语义先验，所以大场景里会白白飞很多冤枉路。
2. **核心方法**：LMPath先用生成式语言模型把“目标-环境”关系转成语义探索先验，再结合卫星图像分割出高概率子区域，最后按不同目标函数生成搜索航迹。
3. **关键结果**：论文在真实无人机与仿真中都验证了这套语义先验路径优于传统规划，能更快找到目标或在有限航程下提升发现概率。

## 💎 价值评估

- **🔬 研究价值**：把语言先验真正接到 UAV 搜索规划里，不只是做 VLN 跟随，而是把高层语义变成可执行搜索策略。
- **🚀 实践价值**：很贴合主人关心的无人机自主搜索/巡检链路，尤其适合“给目标描述→快速缩小搜索区域”的任务。
- **📈 扩展潜力**：后续可以把卫星图像替换为机载在线感知，做动态重规划与闭环目标搜救。

## 🎯 可落地实验点

**实验设计**：给 AirSim/UE5 无人机搜索任务加入“语言→语义热区→航迹规划”前端
- 对比基线：纯 lawn-mower coverage / frontier exploration / 无语义先验的最短路搜索
- 度量指标：首次发现目标时间、单位里程发现率、有限电量下成功率
- 预期结果：语言语义先验在大场景稀疏目标搜索时显著减少无效覆盖
