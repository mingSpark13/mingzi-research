# 📖 花火格式笔记

## 🎯 题目

Aerial Vision-Language Navigation with a Unified Framework for Spatial, Temporal and Embodied Reasoning

## 📝 三句摘要

1. **问题背景**：现有 UAV-VLN 往往依赖全景、深度或里程计，系统重、成本高，不适合轻量级无人机真部署。
2. **核心方法**：论文把 aerial VLN 统一成 next-token prediction，用单目 RGB + 自然语言做多任务联合学习，同时加入关键帧筛选、动作合并和标签重加权来稳住训练。
3. **关键结果**：在 AerialVLN 和 OpenFly 上，该方法在仅 monocular RGB 设定下显著超越现有 RGB-only baseline，并缩小与全景 RGB-D SOTA 的差距，对 D06 很对口。

## 💎 价值评估

- **🔬 研究价值**：它把“轻感知输入 + 强推理输出”的 UAV-VLN 路线讲清楚了，尤其适合主人关注的单目、低载荷方向。
- **🚀 实践价值**：很适合迁到真实无人机人体跟随/语义到点任务，减少对深度和昂贵传感器的依赖。
- **📈 扩展潜力**：后面可以把世界模型、VLA 或在线避障模块接进来，做真正可飞的 long-horizon aerial agent。

## 🎯 可落地实验点

**实验设计**：复现其 monocular aerial VLN 设定，并接一个轻量避障头验证真实飞行可部署性。
- 对比基线：传统 RGB-only VLN baseline / 使用深度或全景输入的方法 / 加不加关键帧筛选
- 度量指标：导航成功率、SPL、未见环境泛化、推理延迟、避障成功率
- 预期结果：单目方案在成本更低前提下保住较强导航性能，并为真实 UAV 机载部署留下余量
