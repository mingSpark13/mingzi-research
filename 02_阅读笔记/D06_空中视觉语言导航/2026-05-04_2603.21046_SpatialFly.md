# 📖 花火格式笔记

## 🎯 题目

SpatialFly: Geometry-Guided Representation Alignment for UAV Vision-and-Language Navigation in Urban Environments

## 📝 三句摘要

1. **问题背景**：UAV VLN 在城市场景里常被 2D 视觉表征和 3D 轨迹决策空间错位拖慢，导致模型看得见语义却对空间关系理解发虚。
2. **核心方法**：SpatialFly 不显式重建 3D，而是把几何先验注入 2D 语义 token，再通过 geometry-aware reparameterization 用跨模态注意力对齐 2D 语义与 3D 几何 token。
3. **关键结果**：它在 seen/unseen 环境都稳定超过现有 UAV VLN 基线，在 unseen Full split 上把导航误差降 4.03m、成功率提 1.27%，轨迹也更平滑更贴路径。

## 💎 价值评估

- **🔬 研究价值**：这篇正中 D06 的核心痛点——不是再堆一个更大的 VLM，而是认真处理“语义理解如何落到几何可执行轨迹”这个结构错配。
- **🚀 实践价值**：它只依赖 RGB 观测、不要求显式 3D 重建，对机载部署更友好，适合给主人现有 UAV 导航链路加一个轻量几何对齐中间层。
- **📈 扩展潜力**：后续可继续和 Rule-VLN/FineCog-Nav 这种规则或认知模块路线拼接，形成“语义理解 + 几何校正 + 长程决策”三段式 D06 架构。

## 🎯 可落地实验点

**实验设计**：在主人现有 UAV 导航栈里插入 geometry-guided token alignment 模块，对比纯语义 token 导航器在 urban canyon / 室内走廊等强结构场景的表现。
- 对比基线：纯 RGB 语义导航器、显式深度/重建辅助导航器、Rule-VLN 式规则增强方法
- 度量指标：NE、SR、路径偏离率、轨迹平滑度、未见环境成功率
- 预期结果：几何对齐中间层能在不显著增大算力成本的情况下，提升空间推理稳定性与路径跟随质量
