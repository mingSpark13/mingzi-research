# 📖 花火格式笔记

## 🎯 题目

World-Gymnast: Training Robots with Reinforcement Learning in a World Model

## 📝 三句摘要

1. **问题背景**：真实机器人强化学习受限于物理交互成本，而传统软件仿真又存在操作场景的 sim-to-real gap，导致策略很难高效提升到真机可用水平。
2. **核心方法**：论文提出 World-Gymnast，把动作条件视频世界模型当作可 rollout 的训练环境，在 imagined trajectory 上用 VLM 做奖励评估，再对 VLA policy 做 RL finetune。
3. **关键结果**：在 Bridge 真机操作上，该方法相对监督微调最高提升约 **18×**、相对软件模拟器约 **2×**，还展示了 novel scene、test-time training 与在线回灌 world model 的闭环能力。

## 💎 价值评估

- **🔬 研究价值**：这篇把 D01 的世界模型角色从“预测器/规划器”推进成“训练器”，是 world model for control 叙事里非常关键的一跳。
- **🚀 实践价值**：对主人后续做无人机/具身策略优化很有启发——即使真机数据贵，也可以先把世界模型当低成本 RL 场，尤其适合语义任务或长尾场景微调。
- **📈 扩展潜力**：它天然能和 D05 数据飞轮串起来，形成“真实数据训练 world model → imagined RL 提升策略 → 真机回灌再训练”的自增强闭环。

## 🎯 可落地实验点

**实验设计**：在主人现有任务上做一个 world-model RL 原型，比较“纯模仿学习 / 软件仿真 RL / 世界模型 RL”三条线。
- 对比基线：SFT policy、软件模拟器 RL、无 VLM reward 的 imagined rollout
- 度量指标：真机成功率、样本效率、场景迁移成功率、奖励与真实成败的一致性
- 预期结果：世界模型 RL 在少量真机数据条件下优于纯 SFT，并在新场景微调速度上明显占优
