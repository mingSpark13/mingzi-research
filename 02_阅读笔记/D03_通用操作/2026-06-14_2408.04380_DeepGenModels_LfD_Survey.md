# 📖 花火格式笔记

## 🎯 题目

**Deep Generative Models in Robotics: A Survey on Learning from Multimodal Demonstrations**
（深度生成模型在机器人中的应用：从多模态示范学习的综述）

## 📝 三句摘要

1. **问题背景**：LfD/模仿学习/BC/IRL 等"从示范学习"范式长期受困于**示范数据的多模态性**（同一任务多种合理轨迹/抓法/策略），传统方法（GMM/HMM/GP）既无法表达高维条件分布，也难以扩展到大规模数据。机器人社区急需一个系统化的"深度生成模型 + LfD"地图。

2. **核心方法**：综述按**模型家族**对近年生图/视频/决策领域的生成范式进行系统盘点——**EBM（能量模型）/ Diffusion（扩散）/ Action Value Maps / GAN（对抗）/ Normalizing Flows / VAE**，并按**应用层**（抓取生成、轨迹生成、cost learning、运动规划）分类；同时深入剖析**泛化能力**这一核心瓶颈，给出不同的设计选择。

3. **关键结果**：本综述本身是 TRO 2026 期刊综述（20页、11张图），**不提出新方法**，但提供了一份"生成模型 × LfD 任务"的完整分类法 + 决策框架，并指出**多模态泛化、组合泛化、长horizon、跨具身迁移**四大开放方向。TRO 是 IEEE 旗下机器人顶刊综述，本文几乎可作为"机器人+生成模型"领域入门的"必读地图"。

## 💎 价值评估

- **🔬 研究价值（⭐⭐⭐⭐⭐）**：是当前**最完整、最新**的"深度生成模型在机器人 LfD 领域"系统综述（TRO 2026 顶刊，20页），提供了按**模型族**（EBM/Diffusion/Action Value Maps/GAN/Flows/VAE） × **应用层**（抓取/轨迹/cost/规划）双重维度的分类法。论文中提到的"多模态示范导致平均化错误"是主人研究 VLA 时的核心痛点之一。
- **🚀 实践价值（⭐⭐⭐⭐）**：对主人的具体价值在于**为下一篇 D02_VLA 方向论文选型提供决策树**——面对具体下游任务（抓取生成 vs 轨迹预测 vs 灵巧操作），可以快速对照本综述的"分类表"选定模型族，省去自己再读几十篇 baseline 论文的时间。
- **📈 扩展潜力（⭐⭐⭐⭐⭐）**：综述明确指出**生成模型 + VLA / 世界模型 / Sim2Real** 的交叉是未来方向。主人正在做 AirSpark/AerialVLA 方向，本综述可作为"为什么用 Diffusion Policy 不用 BC/IRL"等基础问题的**权威引用源**。

## 🎯 可落地实验点

**实验 1：Diffusion Policy vs EBM vs Flow Matching 在同任务对比**
- **目的**：用本综述的分类法做一次"模型族横评"——在 XLeRobot 上的同一抓取任务，对比 Diffusion Policy / EBM 策略 / Flow Matching 策略
- **基线**：BC (LSTM/GMM)、ACT（已有笔记 2023-01_ACT.md）
- **指标**：任务成功率 / 多模态分布覆盖率 / 训练耗时 / 推理延迟
- **预期**：Diffusion Policy 在多模态分布任务上显著优于 BC；Flow Matching 推理延迟最低
- **直接落点**：对接 `01_机器人与具身` 已有 ACT 笔记，复用同一 benchmark

**实验 2：多模态示范下的"平均化错误"复现 + 生成式消解**
- **目的**：按综述第 1 章描述的"多模态平均化问题"，构造 3 种**等价正确轨迹**的训练数据，对比 BC / Diffusion / Flow Matching 的分布恢复能力
- **指标**：mode coverage（KL 散度）、trajectory diversity（轨迹方差）
- **价值**：直接产出"D01/D02 方向"的开题实验素材，可作为下一篇 AirSpark 论文的 motivation 实验

**实验 3：综述作为 AerialVLA 论文 related work 的引用骨架**
- **目的**：把本综述的"模型族分类表"作为 AerialVLA 论文 Related Work 章节的**唯一图源**，把 Diffusion Policy / Flow Matching 的方法论挂到本综述
- **价值**：省去 1-2 周的 related work 整理时间

## 📌 待探索问题

1. **生成模型族横向对比的"任务适配性"**：综述按模型族分类，但**没说在哪些任务下哪个家族最优**。是否可做一次"模型族 × 任务类型"的系统 benchmark，得出"抓取→Diffusion / 轨迹→Flow / cost→EBM"的经验法则？
2. **多模态分布的"模式崩溃"诊断**：本综述提到生成模型会"平均化多模态"为错误行为，但缺乏对**哪些数据特性导致模式崩溃**的形式化分析。可结合 toy 数据集做"双峰分布"诊断实验。
3. **生成式策略 + VLA 的融合范式**：当前 VLA（如 π₀、OpenVLA）多为确定性或单峰动作输出，**如何将综述里的"多模态生成"思想融入 VLA 架构**？主人正在做的 AirSpark 方向是否能借鉴？
4. **Sim2Real 中的生成式数据增强**：综述提到 cost learning 与 RL 结合，但**生成式策略在 sim2real gap 下的鲁棒性**几乎没讨论。这是一个值得深挖的实战方向。
