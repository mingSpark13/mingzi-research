# 📖 花火格式笔记

## 🎯 题目

AFUN: Towards an Affordance Foundation Model for Functionality Understanding

## 📝 三句摘要

1. **问题背景**：开放世界操作里，机器人不仅要知道“哪里能交互”，还要知道“如何执行动作”，而现有 affordance 方法往往只做局部区域定位或只预测动作，难同时兼顾开放泛化。  
2. **核心方法**：AFUN 从单帧 RGB-D 与语言任务描述出发，联合预测 task-conditional functional mask 与 3D post-contact motion curve，并构建统一大规模 affordance 数据流水线，把机器人、人类、仿真和真实扫描数据映射到同一 schema。  
3. **关键结果**：论文在 8 个测试集上显著超越基线，gIoU/cIoU 提升 +23.9/+26.3，接触点命中率也大幅领先，还能无需 embodiment finetuning 直接落到真实操作任务。

## 💎 价值评估

- **🔬 研究价值**：这是少见把“交互区域 + 后接触运动”一起统一建模的 foundation-style affordance 工作，对通用操作和 open-world manipulation 都很关键。  
- **🚀 实践价值**：对主人做机器人操作很值钱，因为它把任务描述、视觉理解、交互点和运动曲线直接接起来，天然适合做高层语义到低层执行之间的中间接口。  
- **📈 扩展潜力**：后面可以接 VLA、PMI 或空中操作场景，把 affordance 预测当成可验证前置模块，尤其适合主人偏好的“可解释接口层”。

## 🎯 可落地实验点

**实验设计**：验证“先做 affordance 功能理解，再交给策略执行”是否比纯 end-to-end policy 更稳  
- 对比基线：纯 VLA / 抓取检测+策略 / AFUN 式功能区域+3D motion curve + 策略执行  
- 度量指标：未见物体成功率、接触点误差、动作轨迹可执行率、跨任务迁移表现  
- 预期结果：显式功能理解接口会提升开放世界任务中的可解释性与泛化稳定性
