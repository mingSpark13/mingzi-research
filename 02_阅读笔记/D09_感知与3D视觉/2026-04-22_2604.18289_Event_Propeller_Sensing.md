# 📖 花火格式笔记

## 🎯 题目

Relative State Estimation using Event-Based Propeller Sensing

## 📝 三句摘要

1. **问题背景**：多 UAV 协同飞行很依赖快速、稳健的相对状态估计，但普通单目相机方案延迟高、尺度模糊，而且在强光/高速场景下容易炸。
2. **核心方法**：论文用事件相机追踪对方无人机桨叶区域，从时间块里估计每个桨叶频率，把频率当作推力输入送入运动学状态估计器，并结合几何拟合恢复姿态轴。
3. **关键结果**：在 5 段真实室外飞行数据上，桨叶频率估计误差低于 3%，给多机去中心化相对定位提供了一条低延迟、强动态范围的新路子。

## 💎 价值评估

- **🔬 研究价值**：很妙，它不是常规事件相机目标检测，而是直接利用桨叶频率这个“动力学可观测特征”，把感知和控制变量绑起来了。
- **🚀 实践价值**：对无人机编队、追踪、空中操作避碰很有实际价值，尤其适合高速机动和逆光户外场景。
- **📈 扩展潜力**：后面能接到多机协同控制、相对导航甚至空中抓取平台，对明子做 UAV 方向很有味道。

## 🎯 可落地实验点

**实验设计**：在现有多机仿真/真机系统里加事件相机相对观测支路，验证高速编队时的状态估计改进。
- 对比基线：普通单目视觉定位、AprilTag/特征点相对定位、纯 IMU/UWB 融合
- 度量指标：相对位姿误差、估计时延、逆光/高速场景丢失率、编队控制稳定性
- 预期结果：事件桨叶感知在高速与高动态范围下明显更稳，能降低编队控制的抖动与追踪丢失
