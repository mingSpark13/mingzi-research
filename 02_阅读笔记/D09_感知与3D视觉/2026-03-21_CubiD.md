## 🎯 题目

**CubiD: Discrete Visual Generation on High-Dimensional Representation Tokens**

## 📝 三句摘要

1. **问题背景**：现有离散 Token 生成方法受限于低维隐空间（8-32维），牺牲了理解任务所需的高层语义信息；而高维预训练表征（768-1024维）虽然语义丰富，但其离散生成存在根本性挑战。

2. **核心方法**：本文提出 **Cubic Discrete Diffusion (CubiD)**，首个针对高维表征的离散扩散模型。CubiD 在高维离散表征的每个维度、每个位置进行细粒度掩码与预测，使模型能够学习位置内和跨位置的丰富关联，且生成步数固定为 T（与特征维度 hwd 无关）。

3. **关键结果**：在 ImageNet-256 上达到 SOTA 离散生成性能，模型从 900M 到 3.7B 参数均表现出强 Scaling Law。验证了离散化 Token 保留了原始表征能力，同一 Token 可同时服务理解和生成任务，为统一多模态架构提供新思路。

## 💎 价值评估

- **🔬 研究价值**：首次提出高维表征离散扩散生成范式，解决了离散生成"维度诅咒"问题，为多模态统一架构提供新生成机制。
- **🚀 实践价值**：离散 Token 同时服务理解和生成，有望简化 VLA 等多模态系统的 Tokenize pipeline；Code 已开源。
- **📈 扩展潜力**：可扩展至视频生成、3D 表征；与 π0/VLA 架构结合探索；高维离散 Token 作为统一多模态表示的可行性。

## 🎯 可落地实验点

**实验设计**：在无人机/机械臂 VLA 任务中，用 CubiD 离散化高维视觉特征（替代现有 VQ-VAE/clip 方案），验证离散 Token 同时支持视觉理解和动作生成的可行性。

- **对比基线**：原生 VQ-VAE Token（8-32维）、CLIP 视觉特征
- **度量指标**：下游任务成功率、Token 压缩率、生成质量（FID）
- **预期结果**：高维离散 Token 在保持语义丰富性的同时，可实现与低维 Token 相近的生成质量，并显著提升理解类任务（检测/分割）的迁移效果
