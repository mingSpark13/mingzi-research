# 📖 花火格式笔记

## 🎯 题目

ESARBench: A Benchmark for Agentic UAV Embodied Search and Rescue

## 📝 三句摘要

1. **问题背景**：现有 UAV 搜救研究多停在传统视觉或路径规划层，缺少把空间推理、线索搜索、安全飞行放在一起评测的统一具身 benchmark。
2. **核心方法**：论文提出 ESAR 任务与 ESARBench 基准，在 Unreal Engine 5 + AirSim 中构建四类高保真开放场景，并引入天气、昼夜、线索随机性与 600 个真实搜救风格任务。
3. **关键结果**：实验表明无论传统启发式还是地面/空中 MLLM ObjectNav agent，都在空间记忆、空中适配和搜索效率-飞行安全权衡上暴露明显瓶颈，说明该基准对 agentic UAV 很有诊断价值。

## 💎 价值评估

- **🔬 研究价值**：它把“空中具身搜索”从单点算法问题抬成系统评测问题，尤其适合主人当前关心的扰动分层、记忆保真和安全约束分析。
- **🚀 实践价值**：可直接借用其 weather/time/clue stochasticity 设定，给 Isaac / UAV 导航实验补一套更贴真实任务的 stress test 模板。
- **📈 扩展潜力**：后续能接入语言指令、多机协同、世界模型 rollout 与 safety shield，扩成更完整的 aerial embodied agent 评测栈。

## 🎯 可落地实验点

**实验设计**：在主人现有 UAV/具身导航实验中复刻 ESARBench 的扰动分层评测口径，比较是否真的提升“安全下的有效搜索”。
- 对比基线：纯启发式搜索、普通 ObjectNav agent、带记忆/世界模型的 agent
- 度量指标：任务成功率、平均搜救时间、碰撞率、安全违规次数、线索利用率
- 预期结果：带显式记忆与安全约束的策略在复杂扰动场景中更稳，能更清楚地区分“会找”与“敢飞”的能力差异
