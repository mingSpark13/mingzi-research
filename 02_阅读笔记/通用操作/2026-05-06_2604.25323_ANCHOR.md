# 📖 花火格式笔记

## 🎯 题目

ANCHOR: A Physically Grounded Closed-Loop Framework for Robust Home-Service Mobile Manipulation

## 📝 三句摘要

1. **问题背景**：开放词汇家庭服务移动操作常在动态场景里失效，不是因为语言不懂，而是符号计划、导航落点和真实可操作几何状态逐步脱钩。
2. **核心方法**：ANCHOR 把任务规划锚定到可观测几何实体，加入可操作性感知的基座对齐，并用最小责任层级恢复机制分别处理感知、基座-机械臂协同和执行异常。
3. **关键结果**：在 60 次未见环境实机试验中，任务成功率从 53.3% 提升到 71.7%，扰动下恢复率达 71.4%，证明显式物理锚定对长程移动操作很关键。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯再堆一个 planner，而是直击“到了但做不了”“环境变了地图还当真”这类长程具身系统老毛病，系统创新很扎实。
- **🚀 实践价值**：对主人未来做语言驱动移动操作、家庭机器人、具身 agent 执行闭环特别有参考价值，尤其适合补强 plan→navigate→manipulate 的接口稳定性。
- **📈 扩展潜力**：可以继续往 VLA 高层计划、世界模型校验、操作可达性预测器、失败责任归因学习这些方向接，潜力很大。

## 🎯 可落地实验点

**实验设计**：给现有语言驱动移动操作系统补一个“物理锚定 + 可操作落点检查 + 分层恢复”的中间层，对比纯语义地图+全局重规划方案。
- 对比基线：语义地图 + LLM 任务规划、无可操作性约束的导航到点、统一全局重规划恢复
- 度量指标：任务成功率、到达但不可操作比例、平均恢复次数、扰动后恢复成功率
- 预期结果：物理锚定和责任分层能显著降低级联失败，提高长程任务稳定性
