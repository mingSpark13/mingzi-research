# 📖 花火格式笔记

## 🎯 题目

Plug-and-Play Label Map Diffusion for Universal Goal-Oriented Navigation

## 📝 三句摘要

1. **问题背景**：目标导向导航在未知环境里最难的是边建图边猜目标位置，传统语义地图法又容易受不完整地图和语义关联不稳拖累。
2. **核心方法**：论文提出 PLMD，用扩散式标签地图补全未知区域的障碍与语义标签，把 obstacle prior 注入语义去噪过程，再作为可插拔模块接到现有 map-based navigation 策略中。
3. **关键结果**：在三个 GON 任务上做到 SOTA，说明“先补未知区域语义地图、再做目标定位”这条路比死守局部可见区域更有效。

## 💎 价值评估

- **🔬 研究价值**：它把 diffusion 从图像生成搬到“未知地图语义补全”上，而且是可插拔地服务导航器，思路很干净。
- **🚀 实践价值**：对主人关心的空中视觉语言导航很有参考价值，无人机在低空搜索或室内探索时也常面临“只看见局部，却得提前猜目标在哪里”的问题。
- **📈 扩展潜力**：后续可以把 label map diffusion 与 UAV 俯视图、语义占据栅格、跨视角地图对齐结合，做更强的 long-range goal prediction。

## 🎯 可落地实验点

**实验设计**：在无人机语义导航任务中加入“未知区域标签补全器”，先预测目标相关语义与障碍分布，再让局部规划器消费该先验。
- 对比基线：原始语义地图导航、仅 obstacle completion、仅 VLM goal ranking
- 度量指标：目标发现成功率、探索效率、平均飞行距离、碰撞率
- 预期结果：补全后的语义先验会提升远程目标搜索效率，并减少盲目探索
