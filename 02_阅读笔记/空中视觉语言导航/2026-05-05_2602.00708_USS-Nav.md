# 📖 花火格式笔记

## 🎯 题目

USS-Nav: Unified Spatio-Semantic Scene Graph for Lightweight UAV Zero-Shot Object Navigation

## 📝 三句摘要

1. **问题背景**：未知环境中的 UAV 零样本目标导航同时要求高层语义推理和机载轻量计算，这两件事常常互相打架。  
2. **核心方法**：论文提出 USS-Nav，在线构建统一时空语义场景图，用粗到细策略让 LLM 先选全局目标区域，再由局部规划器做信息增益驱动的探索。  
3. **关键结果**：该方法把语义推理与轻量拓扑表示绑在一起，在未知场景下实现了更高效的 UAV zero-shot object navigation，很贴主人 D06 主线。  

## 💎 价值评估

- **🔬 研究价值**：这是少见把 scene graph、LLM 语义决策和 UAV 零样本导航真正耦合起来的工作，命中“轻量语义导航”关键缺口。  
- **🚀 实践价值**：对主人做无人机开放场景目标搜索、语义引导探索、机载受限部署都很有直接参考性。  
- **📈 扩展潜力**：后续可继续接多模态目标描述、动态障碍建模，甚至并入空地协同导航共享图。  

## 🎯 可落地实验点

**实验设计**：在 AirSim 无人机场景复现“场景图+粗细两级规划”框架，对比纯 frontier 和纯 VLM 导航
- 对比基线：frontier exploration、端到端 VLM 导航器
- 度量指标：目标发现时间、导航成功率、平均路径长度、推理耗时
- 预期结果：USS-Nav 式场景图方法在机载约束下取得更优语义导航效率与稳定性
