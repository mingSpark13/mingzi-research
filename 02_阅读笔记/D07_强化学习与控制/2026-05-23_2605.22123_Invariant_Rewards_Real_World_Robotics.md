# 📖 花火格式笔记

## 🎯 题目

Beyond Pixels: Learning Invariant Rewards for Real-World Robotics From a Few Demonstrations

## 📝 三句摘要

1. **问题背景**：真实机器人强化学习里，视觉奖励模型很容易记住像素分布，换物体、机位或位置就失效，导致 reward 不能跨场景复用。
2. **核心方法**：论文提出从少至 5 条示范中学习 invariant symbolic reward，把任务级行为不变量与物理约束结合，再用符号-数值混合过程离线蒸馏奖励函数，无需在线交互。
3. **关键结果**：方法在 8 个 Meta-World 和 3 个 Franka 任务上优于 baseline，并在真实 OOD 实验里对视角、位置、物体变化实现 zero-shot 泛化，能加速后续策略学习。

## 💎 价值评估

- **🔬 研究价值**：它把“奖励泛化”从像素拟合转向任务不变量抽取，挺适合主人关注的真实机器人低样本学习路线。
- **🚀 实践价值**：如果主人后面做操作策略训练，这类 reward 学习能明显降低手工 reward engineering 成本，也更利于跨场景迁移。
- **📈 扩展潜力**：后续可和 VLA/模仿学习结合，把语言任务分解成结构化不变量，再做 reward 或 critic 监督，形成更稳的 real-world finetune 管线。

## 🎯 可落地实验点

**实验设计**：针对一个真实桌面操作任务，采集少量成功示范，比较像素奖励模型与 invariant reward 在 OOD 场景下的策略训练效果。
- 对比基线：像素对比式 reward、手工稀疏 reward、成功分类器 reward
- 度量指标：任务成功率、样本效率、跨视角/物体泛化、reward-process alignment
- 预期结果：invariant reward 在 OOD 条件下更稳，能显著减少 reward hacking 和重标定成本
