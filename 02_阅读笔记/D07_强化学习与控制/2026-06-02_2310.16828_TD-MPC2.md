# 📖 花火格式笔记

## 🎯 题目

**TD-MPC2: Scalable, Robust World Models for Continuous Control**

副标题：UCSD Hao Su + Xiaolong Wang 团队 ICLR 2024 经典，**317M 参数单一智能体跨 80 任务/多 embodiment/多动作空间泛化**，**路径二（世界模型替代手工建模 + MPC 局部轨迹优化）** 的代表工作。

## 📝 三句摘要

1. **问题背景**：TD-MPC 是基于模型的 RL 算法，在学习到的 implicit（无 decoder）世界模型的隐空间做局部轨迹优化。本文研究**如何 scale TD-MPC 到大规模 + 多任务 + 跨 embodiment**——核心挑战：异构 embodiment 的状态/动作空间/任务语义差异巨大，传统 RL 一个算法只能训一个 embodiment。

2. **核心方法**：本文提出 **TD-MPC2**，对 TD-MPC 做 5 项关键改进：(1) 任务/embodiment 统一 tokenizer（用同一套编码器处理不同状态空间）；(2) 局部轨迹优化在 shared latent space 进行；(3) value function 跨任务学习；(4) 课程式数据采样（先简单后难）；(5) 模型与数据规模同时 scale up。**关键架构 = Implicit world model + Latent space + 局部 MPC 优化**。

3. **关键结果**：**104 个在线 RL 任务 × 4 个 task domain** 显著超越 baseline；**单一超参数**（不需要 per-task 调参）；**agent 能力随模型+数据规模 scale 增加**；**单一 317M 参数 agent 跨 80 任务 × 多 embodiment × 多动作空间** —— **这是单一模型跨形态泛化的标杆**。论文还提供了大模型 agent 的 lesson / opportunity / risk 全面分析。

## 💎 价值评估

### 🔬 研究价值

- **隐式世界模型 + MPC 局部轨迹优化** 的范式成为后续工作基础（PathDreamer / DreamerV3 / GR00T 等都受影响）
- **首次证明"单一模型 + 单一超参"可跨 80 任务 + 多 embodiment 泛化**——给 D04 跨载体泛化提供标杆
- **scale up 实验规范**：方法论贡献（如何系统化 scale up model-based RL）

### 🚀 实践价值

- **可作为主人 D07 RL 项目的 baseline**：主人空中/地面/灵巧手多形态 RL 训练可参考 317M 模型设计
- **统一 tokenizer 思想** = 主人跨载体项目的核心技术（异构观测→统一 token）
- **主人 AirSpark 多机演示**：可用 TD-MPC2 思想做单机/多机协同

### 📈 扩展潜力

- **可与 LLM/VLM 结合**做高层指令解析（与 LLMPC 路径一结合）
- **可与 DIAL-MPC（路径三）结合**：世界模型预测 + 扩散采样精确求解

## 🎯 可落地实验点

### 实验一：把 TD-MPC2 的统一 tokenizer 思想用到 D04 跨载体项目

- **目标**：让主人 AirSpark 跨无人机/UGV/机械臂共享一套策略
- **方案**：
  1. 设计统一 observation token（图像 + IMU + GPS + 关节）
  2. 训一个 317M 级别 cross-embodiment agent
  3. 评估在 5 个任务上的 zero-shot 迁移性能
- **预期**：5 个任务平均成功率 > 60%（TD-MPC2 在 80 任务上的水平）
