# 📖 花火格式笔记

## 🎯 题目

Deep Reinforcement Learning for Robotic Manipulation under Distribution Shift with Bounded Extremum Seeking

## 📝 三句摘要

1. **问题背景**：接触丰富的 manipulation 策略在训练分布内往往表现不错，但一旦目标变化、摩擦分布变化或动力学漂移，纯 RL policy 很容易在部署时掉性能。
2. **核心方法**：论文把 DDPG manipulation policy 和 bounded extremum seeking 组合成 hybrid controller，让 RL 负责快速进入任务、ES 负责在分布外和时变条件下做在线鲁棒修正。
3. **关键结果**：在 pushing 和 pick-and-place 的多种 OOD 设置下，这种 RL+ES 控制器比单独 RL 更稳，说明 inference-time structured feedback 是处理 manipulation distribution shift 的有效补丁。

## 💎 价值评估

- **🔬 研究价值**：它不是再做一个更大的策略模型，而是明确把“部署期纠偏”单独拿出来建模，对真实机器人很实在。
- **🚀 实践价值**：主人如果后面做空中抓取或接触任务，遇到摩擦、风扰、目标变化等 test-time shift，这种 RL 外挂自适应反馈的思路很值得先做轻量验证。
- **📈 扩展潜力**：可和 MPC、安全约束、RFS 一类 pretrained policy steering 方法拼起来，形成“先验策略 + 在线纠偏”的统一框架。

## 🎯 可落地实验点

**实验设计**：在 pushing / pick-place 类任务里比较纯 RL 与 RL+在线反馈纠偏在 OOD 条件下的稳健性。
- 对比基线：DDPG / residual RL / RL + bounded ES / RL + MPC
- 度量指标：OOD 成功率、摩擦变化下恢复时间、目标漂移跟踪误差、推理时延
- 预期结果：bounded ES 组合在训练外工况下更稳，尤其适合目标时变或接触参数漂移的任务
