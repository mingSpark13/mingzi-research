# 📖 花火格式笔记

## 🎯 题目

Beyond Specialization: Robust Reinforcement Learning Navigation via Procedural Map Generators

## 📝 三句摘要

1. **问题背景**：导航 RL 策略常过拟合到少量人工设计地图，换一种地形分布就会性能暴跌。
2. **核心方法**：论文系统比较四类 procedural map generator，并在 MuRoSim 中对五类导航策略做跨生成器泛化评测，分析什么训练地图分布最能带来鲁棒性。
3. **关键结果**：单一场景专家跨域会塌得很厉害，而组合式 generator 训练可把平均成功率拉到 91.5%，再加 A* 子目标输入可进一步到 98.9%，直接说明“训练分布设计”比盲目换网络更关键。

## 💎 价值评估

- **🔬 研究价值**：这篇把“RL 导航为什么不泛化”拆到了环境生成层，结论对主人做 Isaac/仿真训练尤其有用，不只是改算法，而是改训练世界。
- **🚀 实践价值**：主人后面不管做地面导航、空中避障还是具身策略学习，都能把 procedural curriculum 当作更省钱的泛化增强手段。
- **📈 扩展潜力**：很适合迁到 Isaac Sim / UAV corridor / cluttered indoor 飞行环境，研究地图分布、课程调度和 planner hint 的联合效果。

## 🎯 可落地实验点

**实验设计**：在 Isaac 或自建 UAV/UGV 导航仿真里做“单一地图训练 vs 多生成器混合训练 vs 混合训练+A*子目标提示”的三组对照。
- 对比基线：固定少量手工地图训练、纯反应式 RL、混合地图 RL + planner hint
- 度量指标：跨地图成功率、样本效率、碰撞率、对新拓扑的泛化性能
- 预期结果：环境分布多样化将比单纯加大模型更有效，planner hint 会显著提升长程鲁棒性
