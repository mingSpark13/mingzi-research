# 📖 花火格式笔记

## 🎯 题目

**DIAL-MPC: Full-Order Sampling-Based MPC for Torque-Level Locomotion Control via Diffusion-Style Annealing**

副标题：CMU + UC Berkeley 联合出品，**扩散退火 + 全阶 MPC 实时采样** —— **路径三（算法升级）** 的代表工作。**第一个 training-free 实现四足跳跃带载**的 MPC 框架。

## 📝 三句摘要

1. **问题背景**：全阶动力学 NMPC（Nonlinear MPC）求解慢、实时性差，**实际工程常用简化模型**（单刚体模型），**牺牲控制精度和灵活性**。采样型 MPC（如 MPPI）能处理非凸/不连续问题，但**采样效率低、跟踪误差大、方差高**。本文目标：**用扩散模型的退火思想改造采样型 MPC**，实现全阶 MPC 实时高精求解。

2. **核心方法**：**DIAL-MPC (Diffusion-Inspired Annealing for Legged MPC)** = 采样型 MPC + 扩散风格退火。理论分析：(a) MPPI 与单步扩散的数学联系；(b) 基于此的 landscape 分析。算法：迭代 refinement + 全覆盖 + 局部收敛，**无需任何预训练神经网络**。在 Unitree Go1 四足机器人上**直接做扭矩级控制**。

3. **关键结果**：(a) **跟踪误差比标准 MPPI 降低 13.4 倍**；(b) **困难爬坡任务比 RL 策略强 50%**（无任何训练）；(c) **首次实现精确 real-world 四足跳跃带载**（unprecedented）；(d) 训练-free 是核心优势 — 不需要 GPU + 大量数据。

## 💎 价值评估

### 🔬 研究价值

- **MPPI 与单步扩散的数学等价性证明**——重要的理论贡献
- **扩散退火改造采样型 MPC** 是路径三的开创性工作
- **全阶 MPC 实时采样**：打破"全阶动力学 = 慢"的固有认知
- **训练-free**：相比 RL 训练成本极低（无 GPU / 无数据）

### 🚀 实践价值

- **可直接套到主人四足/UAV 项目**：用 DIAL-MPC 做底层 MPC，比简化模型精度高、比 RL 训练成本低
- **13.4× 跟踪误差降低** = 主人 Open Day 演示"高精度控制"的硬指标
- **可与 TD-MPC2 结合**：TD-MPC2 做世界模型预测 + DIAL-MPC 做精确求解 = 路径二 + 三融合

### 📈 扩展潜力

- **理论可推广到其他采样优化**（粒子滤波 / MCMC / 变分推断）
- **硬件平台扩展**：从 Go1 扩展到人形 / UAV / 机械臂
- **与 VLA 顶层规划结合**：主人 AirSpark "VLM 顶层 + DIAL-MPC 底层" 全栈

- **目标**：在主人 AirSpark UAV 上跑 DIAL-MPC，对比 PX4 自带 MPC

  1. 顶层：Qwen-VL 解析"飞越障碍物"指令

  3. 现场展示"语义理解 + 精确控制"分工
- **预期**：比端到端 VLA 演示 + 30% 任务完成度
