# 📖 花火格式笔记

## 🎯 题目

How to Utilize Failure Demo Data: Effective Data Selection for Imitation Learning

## 📝 三句摘要

1. **问题背景**：模仿学习依赖专家示范数据，但实际采集的数据中包含大量失败样本；现有方法通常丢弃失败数据，浪费了成功-失败差异中蕴含的丰富信息。
2. **核心方法**：提出一种方法，学习成功-失败差异的隐表示，并将其融入注意力机制中，从而在模仿学习过程中有效筛选和利用失败示范数据。
3. **关键结果**：通过在多样化离线模仿学习任务上的实验验证，该方法相比仅使用成功数据或其他失败数据利用策略，显著提升了策略性能和样本效率。

## 💎 价值评估

- **🔬 研究价值**：系统性解决失败数据利用问题，对模仿学习数据效率有重要贡献；隐表示学习 + 注意力筛选的思路具有方法论创新。
- **🚀 实践价值**：直接适用于主人的 AirSpark 数据采集场景：采集过程中产生的失败样本可通过本文方法变废为宝，提升数据效率；降低数据采集成本。
- **📈 扩展潜力**：可与扩散策略结合用于失败样本生成；可迁移到 UAV 人体跟随的在线学习场景；对数据合成中的"坏数据"利用有指导意义。

## 🎯 可落地实验点

**实验设计**：在 AirSpark 数据采集流程中集成失败数据利用模块
- 对比基线：仅使用成功示范 vs 成功+失败示范（无筛选）vs 成功+失败示范（本文方法筛选）
- 度量指标：策略成功率、数据效率（样本数/性能增益）
- 预期结果：本文方法在保持数据量不变的情况下显著提升策略性能
