# 📖 花火格式笔记

## 🎯 题目

Accelerating Robotic Reinforcement Learning with Agent Guidance

## 📝 三句摘要

1. **问题背景**：真实机器人强化学习常被低样本效率和 Human-in-the-Loop 监督的 1:1 人力瓶颈卡住，复杂操作任务尤其难扩展。
2. **核心方法**：论文提出 Agent-guided Policy Search（AGPS），把多模态 agent 当作语义世界模型，仅在 FLOAT 检测到策略分布漂移时触发，用动作路标与 3D 空间约束替代人工纠错。
3. **关键结果**：在 USB 插入、中国结悬挂、毛巾折叠三类真机任务上，AGPS 以零人工介入取得比 HIL 方法更高的样本效率，说明“agent 监督 RL”已具备可扩展潜力。

## 💎 价值评估

- **🔬 研究价值**：它把“多模态 agent = 可调用工具的语义监督器”正式嵌进真实机器人 RL 闭环，不只是高层规划，而是直接干预探索过程，这点很新。
- **🚀 实践价值**：对主人后续做真实机器人/空中操作 RL 很有参考价值，尤其适合人力昂贵、失败恢复代价高的任务，把人工纠错替成低频 agent 纠偏很香。
- **📈 扩展潜力**：如果把 AGPS 的 corrective waypoint 和 spatial constraint 接到 Isaac Lab 或空中操作任务，能自然延展成“仿真预训 + 真机 agent 纠偏”的数据飞轮。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 或机械臂插接任务中复现“失败检测 + agent 低频纠偏”的训练范式，用视觉 grounding 或规则工具生成纠偏 waypoint。
- 对比基线：标准 SAC/PPO、带人工演示 warm-start 的 RL、仅失败回放不纠偏的 RL
- 度量指标：样本效率、成功率、单次失败恢复时间、人工干预次数
- 预期结果：AGPS 风格的低频语义纠偏能显著减少无效探索，并在接触密集任务里更快收敛
