# 📖 花火格式笔记

## 🎯 题目

Emergent Dexterity via Diverse Resets and Large-Scale Reinforcement Learning

## 📝 三句摘要

1. **问题背景**：长程、接触丰富的灵巧操作任务里，常规 RL 很容易因为探索窄、奖励工程重、课程复杂而早早卡死，难以随着算力继续扩展。
2. **核心方法**：论文提出 OmniReset，用仿真器 reset 主动把策略暴露到大量关键交互状态，再配合大规模 on-policy RL，在几乎不调超参、少课程和少人工示范的前提下学习长时程灵巧技能。
3. **关键结果**：方法在多类灵巧操作任务上稳定扩展到复杂恢复行为，并可蒸馏成视觉运动策略实现 zero-shot sim-to-real，说明“多样 reset + 大规模 RL”本身就是很强的 dexterity 训练范式。

## 💎 价值评估

- **🔬 研究价值**：它把“探索难”转成“状态分布设计”问题，给 D07 提供了比传统 curriculum 更干净、可扩展的训练思路。
- **🚀 实践价值**：主人若在 Isaac/Genesis 做灵巧操作或接触丰富控制，这套 reset 机制很适合直接迁进现有训练流水线，先提升收敛效率和恢复行为。
- **📈 扩展潜力**：后面可把 reset 策略和世界模型、失败检测、自动 curriculum 拼起来，做更自动化的长程 manipulation RL 系统。

## 🎯 可落地实验点

**实验设计**：给主人现有 Isaac 强化学习任务补一个 OmniReset 风格的多样状态重置器，验证其对长程接触任务的训练收益。
- 对比基线：默认初始状态训练、手工 curriculum、OmniReset 多样重置
- 度量指标：成功率、收敛速度、失败后恢复成功率、真实机 zero-shot 成功率
- 预期结果：OmniReset 在长时程任务上更快学出 retry/recovery 行为，并显著提高 sim2real 后的鲁棒性
