# 📖 花火格式笔记

## 🎯 题目

Learning Reactive Dexterous Grasping via Hierarchical Task-Space RL Planning and Joint-Space QP Control

## 📝 三句摘要

1. **问题背景**：灵巧抓取在真实硬件上常被高维动作、动态障碍和安全约束卡住，纯端到端 RL 很难同时拿到反应性、稳定性和可部署性。
2. **核心方法**：论文把高层任务空间意图与低层关节执行解耦，用 arm/hand 双智能体层级 RL 生成 task-space velocity，再由 GPU 并行 QP 控制器把命令投影为满足关节极限与避碰约束的安全关节速度。
3. **关键结果**：该框架在 sim2real 流水线下实现了对 7-DoF 机械臂 + 20-DoF 拟人手的零样本转移，面对未见物体与扰动仍能稳定恢复抓取，说明“RL 规划 + 解析安全层”这条路线对真实灵巧操作很有价值。

## 💎 价值评估

- **🔬 研究价值**：把层级 RL 和 QP 安全控制明确拆层，给高自由度灵巧抓取提供了比单体策略更可解释、更稳的结构范式。
- **🚀 实践价值**：零样本可调安全裕度、动态障碍避让、真实拟人手落地，这些都很适合主人后续做高维操作控制或空中机械臂末端控制借鉴。
- **📈 扩展潜力**：可继续接到 Isaac Lab 训练、触觉输入、动作先验或世界模型预测层，形成“高层生成意图—低层约束执行”的统一控制栈。

## 🎯 可落地实验点

**实验设计**：把主人现有 Isaac 强化学习机械臂控制链路改成“高层策略输出 task-space velocity，低层 QP 做约束投影”的两层结构。
- 对比基线：纯 joint-space RL policy、带 action clipping 的 end-to-end RL、传统抓取轨迹跟踪控制
- 度量指标：抓取成功率、碰撞率、恢复成功率、sim2real 性能掉点、推理延迟
- 预期结果：两层结构在真实部署稳定性与安全性上明显优于纯 RL，尤其在动态干扰和未见物体条件下更稳
