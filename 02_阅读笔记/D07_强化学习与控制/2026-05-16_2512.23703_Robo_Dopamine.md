# 📖 花火格式笔记

## 🎯 题目

Robo-Dopamine: General Process Reward Modeling for High-Precision Robotic Manipulation

## 📝 三句摘要

1. **问题背景**：真实机器人 RL 最大瓶颈仍是 reward 难设计，尤其长时程、高精度、接触丰富的 manipulation 任务里，稀疏奖励学不动，手工 dense reward 又很难扩展。
2. **核心方法**：Robo-Dopamine 先用 35M 样本、多视角、步骤感知的数据训练通用过程奖励模型 GRM，再用 policy-invariant reward shaping 把 dense reward 安全接入在线 Dopamine-RL。
3. **关键结果**：GRM 在进度评估上达到 92.8%+ 准确率，且新任务只需单条 expert trajectory 适配奖励模型，就能把策略在约 150 次真实机 rollout 内从接近 0 提到 95% 成功率。

## 💎 价值评估

- **🔬 研究价值**：这篇把 PRM 从“会打分”推进到“能给 manipulation RL 稳定供血”的层次，尤其 policy-invariant shaping 对避免 semantic trap 很关键。
- **🚀 实践价值**：主人做机械臂、空中操作或接触任务时，如果想减少手工 reward 设计，Robo-Dopamine 的 step-aware reward model + one-shot task adaptation 很值得抄主线。
- **📈 扩展潜力**：可直接并入 D05 数据飞轮，做“多视角过程标注 → 通用奖励模型 → 在线自提升”闭环，也适合给 VLA / world model 提供过程监督。

## 🎯 可落地实验点

**实验设计**：在 1-2 个接触任务上复现 step-aware reward model，并比较不同 dense reward shaping 对在线 RL 的影响。
- 对比基线：稀疏奖励 RL / 手工 dense reward / 单视角 PRM / Robo-Dopamine 式 GRM + policy-invariant shaping
- 度量指标：成功率提升速度、奖励排序准确率、OOD 布局泛化、真实机 rollout 数
- 预期结果：多视角步骤奖励能显著缩短探索期，并在遮挡和中间步骤判断上优于单视角 reward model
