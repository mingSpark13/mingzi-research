# 📖 花火格式笔记

## 🎯 题目

Knowledge-Guided Manipulation Using Multi-Task Reinforcement Learning

## 📝 三句摘要

1. **问题背景**：机械臂在部分可观测、遮挡和布局变化场景下做多任务操作时，纯视觉或纯本体感觉策略容易丢失关系信息，泛化也不稳。
2. **核心方法**：论文提出 KG-M3PO，把在线 3D scene graph、开放词汇检测、关系边更新和 RL 策略联合进同一计算图，用知识查询增强状态表示。
3. **关键结果**：在遮挡、干扰物、新物体与未见布局下，知识增强策略相对强基线取得更高成功率、更好样本效率和更强长程鲁棒性，并被 ICRA 2026 接收。

## 💎 价值评估

- **🔬 研究价值**：把知识图谱从“后处理提示”推进到 RL 计算图内部，证明关系表示能直接被控制目标塑形，这对具身多任务策略很有启发。
- **🚀 实践价值**：很适合给主人在 Isaac Lab 机械臂任务里做“语义状态层”原型，把视觉检测、容器关系、可供性边编码进策略输入。
- **📈 扩展潜力**：后续可继续接 VLA 高层指令、接触反馈或记忆模块，做语言-图谱-动作统一闭环。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 搭一组多任务桌面操作环境，把场景图关系作为额外 observation，与纯 RGB/状态基线比较。
- 对比基线：PPO/SAC + RGBD，本体感觉；不带图谱的多模态编码器
- 度量指标：success rate、sample efficiency、遮挡场景成功率、未见布局泛化
- 预期结果：带关系图输入的策略在 long-horizon 操作和遮挡恢复上更稳，收敛样本更少
