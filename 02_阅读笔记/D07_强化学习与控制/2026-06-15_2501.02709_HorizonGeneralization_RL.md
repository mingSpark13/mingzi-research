# 📖 花火格式笔记

## 🎯 题目

**Horizon Generalization in Reinforcement Learning**
（强化学习中的视野/时间跨度泛化）

## 📝 三句摘要

1. **问题背景**：RL 在真实世界（医疗/城市规划/机器人）的**长视野（long-horizon）**任务中表现糟糕，**目标越远可能解空间指数爆炸**。现有方法（domain randomization、data augmentation）只解决"环境泛化"问题，**没人解决"时间泛化"**——训练时只见过近目标，测试时却要给远目标。

2. **核心方法**：提出 **"视野泛化（Horizon Generalization）"** 的形式化定义——训练到近目标（容易学）的策略，能在测试时泛化到远目标。理论证明该性质等价于 **"规划不变性（Planning Invariance）"**——直接以最终目标为条件 vs 以中间 waypoint 为条件，**策略输出应相同**。提出满足三角不等式的 **"拟度量策略（Quasimetric Policy）"**，并实现 **CMD1 算法**。

3. **关键结果**：在高维 Ant / AntMaze / Humanoid 任务上，**CMD1 在导航距离翻倍至 30m 时仍保持 93% 成功率**，**显著优于 CRL 和 SAC**。理论上严格证明了"规划不变性 ⇒ 视野泛化"，为 RL 中借鉴 ML 其他领域（不变性/泛化）的技巧打开了大门。

## 💎 价值评估

- **🔬 研究价值（⭐⭐⭐⭐⭐）**：**首次形式化"时间跨度泛化"问题**，并给出"规划不变性"这一**优雅的等价条件**。这种"把 ML 概念（不变性）迁移到 RL 决策问题"的思路，是当前 RL 领域最被看重的方向之一。**Benjamin Eysenbach 组**（Princeton）的招牌工作质量保证。

- **🚀 实践价值（⭐⭐⭐⭐⭐）**：**直接对应主人的 D07 Isaac 强化学习方向**——机械臂/无人机/移动机器人都需要"训练时只到近目标、测试时到远目标"的能力。**CMD1 的算法逻辑可以无缝迁移到 Isaac Lab**。AntMaze 上距离翻倍 30m 保持 93% 成功率 = 工业级可靠性。

- **📈 扩展潜力（⭐⭐⭐⭐）**：论文明确指出未来方向——"把 ML 中其他不变性技巧（数据增强、对比学习、归一化）移植到 RL 实现规划不变性"。**这是个新研究范式的开篇**，未来 2-3 年会有大量 follow-up 论文。主人可考虑基于此**做一篇"语言条件 + 规划不变性"的扩展**（结合 VLA 方向）。

## 🎯 可落地实验点

**实验 1：CMD1 在 Isaac Lab 复现 + 机械臂任务**
- **目的**：把 CMD1 算法迁移到主人的 Isaac Lab 流水线，在**机械臂抓取/导航任务**上验证视野泛化能力
- **基线**：SAC / CRL / TD3（Hindsight Experience Replay）
- **指标**：任务成功率（近/中/远目标三档）/ 训练样本效率 / 推理延迟
- **预期**：CMD1 在远距离目标上显著优于 SAC（论文 30m 93% 成功率是公开数据，复现 1:1 即可）
- **直接落点**：主人 D07 方向 06-15 周一开工的可选实验之一

**实验 2：拟度量表示学习（Quasimetric Representation）的可视化**
- **目的**：在主人的任务上，**把 CMD1 内部的"距离表示"可视化**——验证它是否真的学到"状态间的三角不等式距离"
- **方法**：用 t-SNE / UMAP 把 representation 投影到 2D，看"远/近目标"是否呈现连续流形
- **价值**：可视化的 representation 解释性 → 可作为下一篇论文的核心图

**实验 3：与 Diffusion Policy 横评（和 06-14 TRO 综述呼应）**
- **目的**：在**同任务**（如长视野机械臂操作）对比"规划不变性（CMD1）" vs "生成式（Diffusion Policy）"两种不同范式
- **价值**：直接产出 D02_VLA × D07_RL 交叉方向的实验素材

## 📌 待探索问题

1. **规划不变性 vs 数据增强的边界**：本文说"规划不变性 ⇒ 视野泛化"，但**和传统数据增强的边界在哪**？数据增强能解决哪种泛化、规划不变性解决哪种？**两者能否结合**？
2. **多模态示范下的规划不变性**：06-14 TRO 综述提到"多模态示范导致平均化错误"，**当示范本身就是多模态时，规划不变性还成立吗**？主人 AirSpark 方向的"VLA + 长视野"问题可能需要同时考虑这两个性质。
3. **Sim2Real 中的视野泛化**：论文实验全在仿真。**真实机器人 + 视野泛化**还能保持 93% 成功率吗？sim2real gap 会不会破坏"三角不等式"的假设？
4. **CMD1 的样本复杂度**：30m 任务的训练样本量是多少？**和 Diffusion Policy 比样本效率**（后者通常更 sample-efficient）如何？这是落地到 Isaac Lab 之前必须回答的问题。
