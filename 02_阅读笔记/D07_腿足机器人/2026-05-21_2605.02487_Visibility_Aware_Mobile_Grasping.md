# 📖 花火格式笔记

## 🎯 题目

Visibility-Aware Mobile Grasping in Dynamic Environments

## 📝 三句摘要

1. **问题背景**：移动抓取在动态未知环境里会卡在“该先看清再动，还是先推进任务”的矛盾上，传统把感知和操作分开做的方法很难保证遮挡场景下的安全性。
2. **核心方法**：论文提出统一移动抓取系统，把 velocity-aware active perception 驱动的低层全身规划器，与基于 behavior tree 的高层子目标生成器结合起来，边探索边修正失败恢复路径。
3. **关键结果**：在 400 个随机仿真场景和 Fetch 实机上，未知静态/动态环境成功率分别达 68.8%/58.0%，较对比方法提升 22.8%/18.0%，同时碰撞安全性更好。

## 💎 价值评估

- **🔬 研究价值**：它把主动感知从“看哪里”真正接进移动操作闭环，兼顾可见性约束、动态障碍和执行失败恢复，属于规划-控制-感知联合优化的扎实工作。
- **🚀 实践价值**：对主人后续做无人机/移动平台上的目标接近、抓取前观测和在线 replanning 都很有借鉴意义，尤其适合“感知受限但任务必须连续推进”的场景。
- **📈 扩展潜力**：如果把 behavior tree 高层与语言/任务先验结合，再把 low-level planner 替换为空地协同平台，可继续往空中操作或主动语义搜索方向推。

## 🎯 可落地实验点

**实验设计**：在主人后续移动操作或无人机接近任务中加入“主动观察位姿 + 执行位姿”双层规划，比较 visibility-aware 策略与传统最短路径/被动观察策略。
- 对比基线：固定观察策略 + 经典局部规划；仅任务驱动的移动抓取规划
- 度量指标：任务成功率、碰撞率、目标首次识别时间、恢复次数、总耗时
- 预期结果：在遮挡和动态障碍存在时，visibility-aware 策略能明显降低盲动碰撞并提升恢复效率
