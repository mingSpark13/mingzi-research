# 📖 花火格式笔记

## 🎯 题目

MUJICA: Multi-skill Unified Joint Integration of Control Architecture for Wheeled-Legged Robots

## 📝 三句摘要

1. **问题背景**：轮足机器人需要同时兼顾轮式移动与腿式控制，但现实电机约束、感知噪声和多技能切换让统一鲁棒控制很难做好。
2. **核心方法**：论文提出 MUJICA，用单一全本体 proprioceptive policy 联合训练全向移动、越障、跌倒恢复等多技能，再配一个高层 skill selector 动态选择合适技能。
3. **关键结果**：方法在 Unitree Go2-W 上完成仿真到真实验证，展示了复杂非结构环境下更平滑的多技能切换与更强适应性。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯刷 locomotion 指标，而是把轮足平台的多技能统一控制问题系统化，正对 D07 里“混合形态平台如何统一策略”的核心矛盾。
- **🚀 实践价值**：对主人后续看 wheeled-legged / 全身控制 / RL 落地很有参考性，尤其适合和 AWARE 一起对照“反射避障”与“统一多技能”两条路线。
- **📈 扩展潜力**：若后续接入视觉或任务层指令，MUJICA 这类统一低层 skill policy 可以继续往 VLA / 层次控制方向接。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 或相近平台上复现“多低层技能共享骨干 + skill selector”方案，对比单技能独立策略与统一策略在复杂地形切换任务上的表现。
- 对比基线：单技能独立 RL policy、固定 rule-based mode switch
- 度量指标：任务成功率、跌倒恢复率、切换稳定性、sim2real 性能保持率
- 预期结果：统一策略在连续切换场景中更稳，且迁移到真实平台时鲁棒性更高
