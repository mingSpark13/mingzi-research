# 📖 花火格式笔记

## 🎯 题目

Terrain-aware Active Gaze Learning for Generalizable Agile Humanoid Locomotion

## 📝 三句摘要

1. **问题背景**：复杂地形上的人形运动既需要大范围环境覆盖，又需要在算力受限条件下保留局部几何细节，固定感知窗口很难兼顾两者。
2. **核心方法**：论文提出 TAGA，用视觉+本体感觉+运动指令联合学习 terrain-aware active gaze，让策略主动挑选 height scan 中最有信息量的区域供下游控制网络使用。
3. **关键结果**：方法无需额外监督即可涌现凝视行为，在仿真和实机上都提升训练效率与泛化能力，并报告了 1.2m 的真实世界跨沟距离。

## 💎 价值评估

- **🔬 研究价值**：把主动感知显式嵌进人形运动控制，说明“看哪里”本身就是可学习控制变量，不只是感知预处理。
- **🚀 实践价值**：对主人后续做资源受限平台上的地形运动、足点选择、感知-控制耦合设计很有参考价值。
- **📈 扩展潜力**：可继续迁移到四足/UAV 低时延避障，或与世界模型结合做可解释的感知预算分配。

## 🎯 可落地实验点

**实验设计**：在腿足平台上复现“主动关注区域”对地形通过率的影响
- 对比基线：固定视野 height scan 策略、随机关注区域策略
- 度量指标：训练收敛速度、跨地形成功率、足点误差、感知延迟
- 预期结果：主动关注机制在稀疏落脚点和遮挡场景中更稳，且算力开销更可控
