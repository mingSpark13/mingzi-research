# 📖 花火格式笔记

## 🎯 题目

Affordance-Based Hierarchical Reinforcement Learning for Quadruped Pedipulation

## 📝 三句摘要

1. **问题背景**：四足机器人做物体交互常常只能学低层动作，高层接触点和站位仍依赖人工设计轨迹，限制了自主 pedipulation 能力。
2. **核心方法**：论文提出三层层级强化学习框架，用位姿 affordance 引导导航策略、导航再驱动 locomotion，并用交互点 affordance 约束足端操作策略，实现自主选站位与交互点。
3. **关键结果**：方法在 IsaacSim 和真实四足平台上都能完成多种真实物体交互任务，证明 affordance 引导的层级 RL 能把移动与操作真正串起来。

## 💎 价值评估

- **🔬 研究价值**：这篇把 locomotion、导航和足端操作用统一层级 RL 串联起来，补的是四足从“能走”到“能干活”的关键缺口。
- **🚀 实践价值**：如果后面做四足开门、推物、触发器操作，这套 affordance→站位→操作链路很适合直接当系统模板。
- **📈 扩展潜力**：可继续接入视觉 affordance、语言目标描述，甚至和 VLA 或 world model 结合做更长程的 loco-manipulation。

## 🎯 可落地实验点

**实验设计**：在四足平台上复现“固定专家站位 vs affordance 自主选站位”的物体交互表现差异。
- 对比基线：人工指定 base pose + 足端策略、两层 RL、本文三层 affordance HRL
- 度量指标：任务成功率、接触点误差、平均规划时间、真实机迁移成功率
- 预期结果：三层 affordance HRL 在未知物体摆放下有更高成功率，且减少人工高层轨迹设计成本
