# 📖 花火格式笔记

## 🎯 题目

Learning Dynamic Pick-and-Place for a Legged Manipulator

## 📝 三句摘要

1. **问题背景**：腿足机械臂虽然兼具移动性和操作性，但在动态行走中同时保持底盘稳定与末端精确抓放一直很难，现有方法也多停留在轻物体、分阶段、慢动作执行。
2. **核心方法**：论文提出分层强化学习框架，用低层 whole-body controller 稳住 locomotion 与姿态，再用高层统一策略一次性覆盖抓取、搬运、放置和撤离，并加入在线质量估计模块适配不同载荷。
3. **关键结果**：仿真中对最高 2.3 kg 负载达到 86.05% 成功率，真机六类场景下对最高 1.3 kg 物体平均成功率 73.3%、平均执行仅 4.06 s，明显快于以往准静态腿足操作方法。

## 💎 价值评估

- **🔬 研究价值**：它把动态 locomotion 与 manipulation 真正统一到一个长时段 pick-and-place 策略里，是腿足具身操作从“会走会抓”走向“连续高效做事”的关键一步。
- **🚀 实践价值**：对需要跨高度搬运、边走边抓放的四足平台很有参考意义，尤其适合主人关注的腿足全身协调与移动操作方向。
- **📈 扩展潜力**：后续可继续接入视觉感知、接触/力觉反馈和更复杂物体交互，向真实场景中的动态 mobile manipulation 推进。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 或四足仿真平台复现“低层稳定 locomotion + 高层统一抓放”的分层策略，对比动态执行与分阶段静态抓放方案。
- 对比基线：分阶段 pick-place 控制器、无质量估计的分层 RL、准静态腿足操作策略
- 度量指标：任务成功率、执行时长、底盘姿态扰动、不同载荷下的放置稳定性
- 预期结果：统一分层策略在中高载荷和大高度差任务中更快、更稳，且对物体质量变化更鲁棒
