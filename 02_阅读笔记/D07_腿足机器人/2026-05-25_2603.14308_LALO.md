# 📖 花火格式笔记

## 🎯 题目

Load-Aware Locomotion Control for Humanoid Robots in Industrial Transportation Tasks

## 📝 三句摘要

1. **问题背景**：工业搬运中的 humanoid 不只要走稳，还得在搬箱子时处理负载变化、上肢动作扰动和部分可观测带来的动态耦合难题。  
2. **核心方法**：论文提出 decoupled-yet-coordinated 的 loco-manipulation 架构，用运动学参考 + height-conditioned joint offset 提供下肢名义姿态，再让 RL policy 只学 residual joint action，同时用 history-based state estimator 吸收负载和操作引入的隐性扰动。  
3. **关键结果**：方法完全在仿真中训练、无需真机微调就部署到全尺寸 humanoid；实验显示训练更快、基座高度跟踪更准、episode 更稳定，并在真实工业搬运场景完成稳定负载行走。  

## 💎 价值评估

- **🔬 研究价值**：它把“负载感知 locomotion”从纯模型法推进到更实用的 residual RL 路线，还清楚展示了 reference shaping 与状态估计器各自的增益来源。  
- **🚀 实践价值**：主人以后做空中/地面载荷控制时，可以借这套“名义参考 + residual policy + 历史估计器”结构，把复杂耦合扰动压缩成更容易学的修正任务。  
- **📈 扩展潜力**：这条路很适合继续接 payload-aware whole-body control、多模态力觉估计，甚至迁到空中吊载或双足+双臂协同任务。  

## 🎯 可落地实验点

**实验设计**：在现有载荷运动控制任务里加入 history-based latent estimator，并比较“直接学全动作”与“参考轨迹 + residual 修正”两种策略结构
- 对比基线：纯 RL locomotion、固定 nominal offset + RL、MPC/whole-body control 基线
- 度量指标：成功搬运率、姿态稳定性、基座高度 MSE、滑步/碰撞次数、训练收敛速度
- 预期结果：residual 结构在负载变化和上肢扰动下更稳，且样本效率更高
