# 📖 花火格式笔记

## 🎯 题目

Whole-Body Impedance Model Predictive Control for Safe Physical Human--Robot Interaction on Floating-Base Platforms

## 📝 三句摘要

1. **问题背景**：浮动基人形机器人在与人发生持续物理接触时，既要维持平衡又要保持柔顺安全，传统 WBC 或固定增益阻抗控制往往会在持续外力下累积稳态误差。
2. **核心方法**：论文把 fixed-base impedance MPC 扩展到 floating-base 平台，构建“质心 MPC + 优先级 WBC + 剩余零空间阻抗 MPC”三级架构，并用 Kalman 扰动估计与接触模式切换协方差膨胀来持续补偿 pHRI 外力。
3. **关键结果**：在 17-DOF 双足与 Unitree G1 仿真上，这套方法实现了接触一致的高频 whole-body impedance control，兼顾平衡、柔顺交互和零稳态误差，对安全人机协作人形控制很有参考价值。

## 💎 价值评估

- **🔬 研究价值**：这篇不是普通 locomotion 论文，而是把 whole-body balance、contact-consistent null-space 和 impedance MPC 真正耦合起来，补强了 D07 里“执行层如何在受扰状态下保持全身控制诚实性”的控制侧证据。
- **🚀 实践价值**：如果明子后续做人形/移动操作平台的高频执行器，这篇的三级架构和 disturbance carry-over 机制可直接借去做 support-side safety 与 recovery controller 设计。
- **📈 扩展潜力**：很适合和现有 D07 的 EQS/SEG/TPS ceiling 体系拼接，形成“高层调度/中层参考/低层扰动抑制”完整闭环，避免把收敛改善误判成 execution magic。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 或 MuJoCo 的浮动基 humanoid/mobile-manipulator 上复现三级控制架构，比较固定阻抗、普通 WBC 与 impedance MPC 在持续外力接触下的恢复与稳态性能。
- 对比基线：固定增益 task-space impedance、无扰动估计的 WBC、带扰动估计的三级 impedance MPC
- 度量指标：末端稳态误差、接触安全约束违例率、恢复时间、重心稳定性、控制频率
- 预期结果：impedance MPC 在持续外力与接触模式切换时能显著减少稳态误差，同时维持 whole-body balance 与交互柔顺性
