# 📖 花火格式笔记

## 🎯 题目

Learning Whole-Body Control for a Salamander Robot

## 📝 三句摘要

1. **问题背景**：蝾螈机器人这类高关节、跨介质平台通常依赖 CPG 或模型控制，学习式全身控制尤其是真机迁移仍研究不足。
2. **核心方法**：论文使用强化学习直接从本体感觉和速度指令映射到关节级动作，并结合系统级 real-to-sim matching 与 sim-to-real transfer，把统一全身控制策略迁到物理蝾螈机器人。
3. **关键结果**：学得的策略在真实世界平地与不平地面上实现稳定协调行走，并在仿真中进一步支持步行与游泳之间的跨模态切换。

## 💎 价值评估

- **🔬 研究价值**：把 RL 全身控制推进到非标准腿足平台，还碰到跨陆地/水域运动模式切换，挺适合启发具身跨模态 locomotion 研究。
- **🚀 实践价值**：对两栖巡检、复杂地形搜救和仿生平台控制很有参考意义，尤其是统一低层策略的工程路线值得借。
- **📈 扩展潜力**：后续可以加视觉感知、接触估计或世界模型，继续往 amphibious 长时任务和自主切换推进。

## 🎯 可落地实验点

**实验设计**：在仿真里构建多地形蝾螈/蛇形平台，对比 RL 全身控制与 CPG/模型控制在跨地形、跨介质切换下的鲁棒性。
- 对比基线：CPG 控制器、模型控制器、无 real-to-sim 校准的 RL
- 度量指标：速度跟踪误差、姿态稳定性、跌倒率、跨模式切换成功率
- 预期结果：统一 RL 控制在复杂地形与模式切换时展现更强适应性，但对 sim2real 校准质量较敏感
