# 📖 花火格式笔记

## 🎯 题目

Locomotion Beyond Feet

## 📝 三句摘要

1. **问题背景**：传统人形 locomotion 大多只依赖双脚步态，面对钻行、翻越、陡坡楼梯等复杂地形时，单靠腿部接触很难同时兼顾稳定性与通过性。
2. **核心方法**：论文提出 Locomotion Beyond Feet，把物理校验过的 keyframe 动作作为参考，用 terrain-specific motion-tracking RL policy 学出各类 crawling/climbing 技能，再由基于视觉与深度估计的 skill planner 进行层级切换。
3. **关键结果**：系统在真实人形平台上实现了跨障碍尺寸、跨场景实例、跨地形序列的稳健 whole-body locomotion，证明“keyframe 先验 + 分技能 RL + 高层切换”是极端地形人形移动的有效路线。

## 💎 价值评估

- **🔬 研究价值**：它把 whole-body contact locomotion 从“会走会跑”推进到“会爬会钻会翻”，对腿足机器人在复杂接触场景中的技能库建设很关键。
- **🚀 实践价值**：对主人关注的人形/腿足控制很值钱，尤其是在数据稀缺时，keyframe 参考比纯 reward shaping 更可控也更易调。
- **📈 扩展潜力**：这套范式能继续扩到跳跃、搬运、窄空间穿越，甚至能和任务级 planner 结合成 contact-aware skill library。

## 🎯 可落地实验点

**实验设计**：在 Isaac/MuJoCo 中搭一个“keyframe 参考动作 + terrain-specific RL skill + 高层视觉切换”的原型，验证其对 crawling/climbing 等接触动作的训练效率提升。
- 对比基线：纯 end-to-end locomotion policy、单一统一策略、无 keyframe 参考的 reward-shaped RL
- 度量指标：复杂地形通过率、跌倒率、技能切换成功率、训练收敛速度、接触稳定性
- 预期结果：带 keyframe 参考与层级技能库的方法，在复杂接触地形上更稳、更快收敛，也更便于人工注入先验
