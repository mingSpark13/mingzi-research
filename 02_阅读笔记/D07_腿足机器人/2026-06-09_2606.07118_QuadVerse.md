# 📖 花火格式笔记

## 🎯 题目

QuadVerse: An Integrated Framework Aligning Visual-Physical Reality for Quadruped Simulation

## 📝 三句摘要

1. **问题背景**：腿足机器人仿真到真实迁移常把视觉 gap 和动力学 gap 分开处理，导致多种误差在长时 rollout 中叠加放大。
2. **核心方法**：QuadVerse 用真实 RGB 视频重建几何约束的 3DGS 场景，同时抽取可碰撞语义网格做接触摩擦校准，再基于真实轨迹回放训练残差动力学补偿器，把视觉、接触、执行器三层误差一起对齐。
3. **关键结果**：方法提升了场景重建质量与 locomotion tracking，并支持无需真实任务微调的零样本视觉导航部署，说明 integrated sim alignment 比单点修补更有效。

## 💎 价值评估

- **🔬 研究价值**：这篇很值钱的点是把 3DGS 视觉重建、接触参数后验校准、执行器残差建模串成一条完整 Sim2Real 管线，而不是零散打补丁。
- **🚀 实践价值**：明子以后做无人机/腿足平台仿真迁移时，可以直接借它的“场景重建 + 物理校准 + 执行器补偿”三段式框架。
- **📈 扩展潜力**：这套方法可迁到空中机器人、轮足平台，或者继续叠加语言条件任务与世界模型做可交互数字孪生。

## 🎯 可落地实验点

**实验设计**：复现“仅视觉随机化仿真 vs QuadVerse integrated calibration”的四足视觉导航迁移对比。
- 对比基线：domain randomization、仅 3DGS 重建、QuadVerse 全管线
- 度量指标：真实机导航成功率、轨迹跟踪误差、接触滑移次数、场景重建质量指标
- 预期结果：QuadVerse 在复杂地形与摩擦变化场景下显著提高零样本迁移稳定性
