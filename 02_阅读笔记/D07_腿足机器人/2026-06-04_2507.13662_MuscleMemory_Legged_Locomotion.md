# 📖 花火格式笔记

## 🎯 题目

Iteratively Learning Muscle Memory for Legged Robots to Master Adaptive and High Precision Locomotion

## 📝 三句摘要

1. **问题背景**：腿足机器人在复杂地形和外扰下常因模型失配导致跟踪误差大，传统控制器也难快速迁移到新速度、坡度和重力条件。
2. **核心方法**：论文把迭代学习控制（ILC）和类“肌肉记忆”的扭矩库结合起来，用混合系统轨迹优化提供物理先验，再用可复用控制剖面做在线快速适配。
3. **关键结果**：方法在 Cassie 双足和 A1 四足上通过仿真与实机验证，将关节跟踪误差在数秒内最多压低约 85%，并支持周期/非周期步态与地形适应。

## 💎 价值评估

- **🔬 研究价值**：这篇把 ILC 从“单一重复轨迹修正”推进到“可迁移技能库”，对腿足控制里的长期适应和跨条件复用很有启发。
- **🚀 实践价值**：如果主人后面做无人机/机器人高精度轨迹跟踪，可借鉴“误差补偿库 + 物理先验 + 在线快速查表”这套结构，而不必每次全量重学。
- **📈 扩展潜力**：后续可把 torque library 替换为更高维动作原语库，和视觉地形感知或语言条件策略耦合，形成更通用的具身技能记忆模块。

## 🎯 可落地实验点

**实验设计**：给现有腿足或飞行控制策略加一个“历史误差补偿原语库”，验证不同地形/速度切换时的快速重适应能力。
- 对比基线：纯 MPC / RL 控制器、无记忆库的 ILC
- 度量指标：轨迹跟踪误差、收敛步数、地形切换恢复时间、在线计算开销
- 预期结果：引入可复用补偿库后，在新工况下能更快恢复稳态并显著降低高频误差
