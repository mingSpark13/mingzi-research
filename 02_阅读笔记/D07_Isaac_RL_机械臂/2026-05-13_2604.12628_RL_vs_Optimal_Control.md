# 📖 花火格式笔记

## 🎯 题目

A Comparison of Reinforcement Learning and Optimal Control Methods for Path Planning

## 📝 三句摘要

1. **问题背景**：威胁区域下的自主载具路径规划既要求安全可达，又要求在线决策足够快，传统最优控制虽能给出优路径，但实时性常常不够。
2. **核心方法**：论文用 DDPG 学习从位置与速度到连续控制动作的直接映射，并与伪谱法最优控制做对比，重点分析可行域、求解速度与路径最优性的权衡。
3. **关键结果**：DDPG 在仿真中能比传统伪谱法快至少一个数量级地产生可行路径，更适合实时应用，但其可行域仍不完整，且路径最优性不如优化法。

## 💎 价值评估

- **🔬 研究价值**：这篇不是花哨新模型，但把“学习控制快、优化控制稳”这个经典矛盾量化得很清楚，适合作为主人后续做无人机在线规划时的基线参照。
- **🚀 实践价值**：若主人后面要做空中机器人避威胁/避障实时规划，可以直接借它的 feasible set 视角，用来判断 RL 控制器到底是在“会飞”还是“能覆盖任务空间”。
- **📈 扩展潜力**：很适合往 MPC/RL 混合路线、奖励修正、可行域估计器和安全护栏方向继续推进，比纯端到端 RL 更贴近工程落地。

## 🎯 可落地实验点

**实验设计**：在主人现有无人机仿真环境中复现“RL 路径规划 vs 优化规划”对比，并加入实时预算约束评估。
- 对比基线：DDPG、MPC/伪谱法、RL+安全护栏混合控制
- 度量指标：求解时延、成功率、路径代价、可行起点覆盖率
- 预期结果：纯 RL 在时延上明显占优，但需要安全约束或混合控制才能逼近优化法的覆盖率与稳定性
