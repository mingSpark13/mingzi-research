# 📖 花火格式笔记

## 🎯 题目

FreqCache: Accelerating Embodied VLN Models with Adaptive Frequency-Guided Token Caching

## 📝 三句摘要

1. **问题背景**：Embodied VLN 模型推理准确但代价高，现有视觉域 token cache 在视角迁移和时序场景变化下容易失效。
2. **核心方法**：论文提出 FreqCache，用频域分析指导 token 的建立、刷新与预算自适应调整，在不重训模型的前提下做训练外加速。
3. **关键结果**：在 VLN 推理中实现约 1.59× 加速且精度损失可忽略，说明“频域缓存”是具身导航部署里很实用的一类系统优化。

## 💎 价值评估

- **🔬 研究价值**：它不改导航策略本身，而是命中主人后面一定会遇到的部署瓶颈——大模型导航太慢，适合作为系统层加速积木。
- **🚀 实践价值**：如果后续做空中 VLN 或机载 VLM 导航，这种 training-free cache 比重新蒸馏/轻量化更容易先落地验证。
- **📈 扩展潜力**：可继续和事件触发刷新、轨迹不确定性估计、飞控实时约束结合，做机载长程导航的延迟-性能折中。

## 🎯 可落地实验点

**实验设计**：把 FreqCache 接到现有 UAV/UGV 视觉语言导航推理链，测延迟下降是否换来同等任务成功率。
- 对比基线：无缓存原始 VLN 推理、固定预算 token cache
- 度量指标：端到端推理时延、Success Rate、SPL、单位任务能耗
- 预期结果：在长程导航中把平均推理时延压到原来的 60%~70%，同时保持接近原始成功率
