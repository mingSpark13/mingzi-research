# 📖 花火格式笔记

## 🎯 题目

RA-Nav: A Risk-Aware Navigation System Based on Semantic Segmentation for Aerial Robots in Unpredictable Environments

## 📝 三句摘要

1. **问题背景**：现有无人机导航通常只绕开当前看见的静态/动态障碍，却很难处理“看起来静止、下一秒突然动起来”的高风险障碍。
2. **核心方法**：论文提出基于语义分割的风险感知导航框架，把障碍物分成静态、临时静态、动态三类，构建局部风险图，再做兼顾效率与安全的 risk-informed path search 和轨迹优化。
3. **关键结果**：在突发状态切换障碍场景中，RA-Nav 相比基线有更高成功率，并在真实数据驱动仿真里验证了风险建模对安全导航的帮助。

## 💎 价值评估

- **🔬 研究价值**：它把“语义理解 → 风险预测 → 局部规划”串成了闭环，重点补的是无人机导航里对潜在动态风险的建模缺口。
- **🚀 实践价值**：对园区巡检、狭窄空间飞行、存在行人/遮挡物的低空场景很有参考意义，比纯几何避障更贴近真实部署。
- **📈 扩展潜力**：后续可以继续接上语言目标、场景图谱或主动感知，让风险导航进一步升级成语义驱动的空中 VLN 系统。

## 🎯 可落地实验点

**实验设计**：给明子的 UAV 跟踪/导航栈增加“潜在动态障碍风险层”，验证对突然起步行人或遮挡后冒出目标的鲁棒性提升。
- 对比基线：传统 A*/D* 局部规划、仅几何代价地图、语义风险地图规划
- 度量指标：成功率、最小安全距离、重规划次数、平均路径代价
- 预期结果：语义风险地图在突发移动障碍场景下成功率更高，重规划更及时
