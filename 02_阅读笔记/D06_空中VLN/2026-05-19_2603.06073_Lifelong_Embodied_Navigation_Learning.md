# 📖 花火格式笔记

## 🎯 题目

Lifelong Embodied Navigation Learning

## 📝 三句摘要

1. **问题背景**：现有具身导航代理往往只在单任务上表现强，一旦连续学习多场景、多指令风格任务，就容易灾难性遗忘。
2. **核心方法**：论文提出 Uni-Walker，把导航知识拆成 task-shared 与 task-specific 两部分，并用 DE-LoRA、knowledge inheritance 与 experts co-activation 做持续增量学习。
3. **关键结果**：实验显示该框架在多任务连续导航设定下能同时保留旧知识并吸收新技能，说明“结构化参数隔离 + 共享迁移”对长期导航能力建设很关键。

## 💎 价值评估

- **🔬 研究价值**：它把 embodied navigation 从单次 benchmark 提升到持续学习范式，正好补主人知识库里“导航会不会越学越忘”的空缺。
- **🚀 实践价值**：如果后面做 UAV-VLN 或室内语义导航长期部署，这套 shared/specific 解耦思路很适合做在线增量更新。
- **📈 扩展潜力**：可继续接记忆回放、场景图谱或 flight log 数据，把持续学习从地面导航扩展到空中视觉语言导航。

## 🎯 可落地实验点

**实验设计**：在 UAV-VLN 任务上构造多场景顺序学习基准，比较普通 LoRA 微调与 DE-LoRA 持续学习框架的遗忘率。
- 对比基线：普通全量微调、单 LoRA 顺序训练、经验回放版本
- 度量指标：旧任务 success rate 保持率、平均遗忘率、未见场景泛化得分
- 预期结果：DE-LoRA 风格的共享/专属解耦能显著降低跨任务遗忘
