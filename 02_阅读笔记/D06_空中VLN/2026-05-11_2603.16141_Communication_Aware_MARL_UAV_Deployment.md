# 📖 花火格式笔记

## 🎯 题目

Communication-Aware Multi-Agent Reinforcement Learning for Decentralized Cooperative UAV Deployment

## 📝 三句摘要

1. **问题背景**：无人机集群做中继部署与感知覆盖时，经常面临部分可观测和机间通信受限，传统集中式方法难以直接落地。
2. **核心方法**：论文提出 communication-aware 的图结构 MARL 框架，采用 CTDE、agent-entity attention 与邻域 self-attention，在局部观测和有限通信图上完成分布式协同。
3. **关键结果**：在 DroneConnect 中，方法在受限通信和部分观测下取得 74% 覆盖率，并能无微调泛化到未见机群规模，在 DroneCombat 中也优于无通信基线。

## 💎 价值评估

- **🔬 研究价值**：把“受限通信 + 多机协同部署”问题做成了较完整的 CTDE 基线，对无人机集群研究很贴主线。
- **🚀 实践价值**：适合作为多 UAV 中继部署、编队感知覆盖、灾害通信恢复等任务的策略参考。
- **📈 扩展潜力**：可往视觉感知输入、异构机群、能耗约束、任务级路径规划继续扩展。

## 🎯 可落地实验点

**实验设计**：在多无人机协同探索/中继部署仿真中复现 CTDE + 邻域注意力，并增加通信丢包与带宽限制 ablation。
- 对比基线：无通信共享 policy、规则式覆盖算法、MILP 离线上界
- 度量指标：覆盖率、连通率、任务完成时间、跨机群规模泛化性能
- 预期结果：通信感知 MARL 在通信受限场景下明显优于无通信策略，并保持较强规模泛化
