# 📖 花火格式笔记

## 🎯 题目

ASIP-Planner: Adaptive Planning for UAV Surface Inspection in Partially Known Indoor Environments

## 📝 三句摘要

1. **问题背景**：室内基础设施巡检常只有 floorplan 级先验，真实环境里却会临时出现设备与遮挡物，导致传统基于完整先验图的 coverage planner 在执行时掉覆盖率、撞障风险也更高。
2. **核心方法**：论文提出 ASIP-Planner，用 segment-based global coverage planner 先按表面分块组织 inspection viewpoints，再由 inspection-oriented local module 在线调整 viewing angle 与无碰轨迹，在保留全局计划结构的同时补偿局部遮挡。
3. **关键结果**：随机场景仿真和真实飞行实验表明，该方法能在部分已知环境里维持近完整覆盖，同时比代表性基线更短轨迹、更能产出可用巡检数据。

## 💎 价值评估

- **🔬 研究价值**：它把“语言/任务目标驱动的空中导航”往更工程化的 task-oriented planning 推了一步，不再只关注到达目标，而是关注到达后如何保住观测覆盖质量。
- **🚀 实践价值**：对主人后续做 tunnel / facility inspection、目标搜索后的近距巡检补拍、或 GoalSearch 到 inspection handoff 都很有参考价值，尤其适合和现有 D06 的局部避障/语义规划链拼接。
- **📈 扩展潜力**：可进一步接入语言指定的 inspection priority、语义目标区域选择，形成“高层语义目标 → 全局覆盖 → 局部视角自适应 → 安全执行”的完整空中任务闭环。

## 🎯 可落地实验点

**实验设计**：把 ASIP-Planner 的“全局表面覆盖 + 局部视角自适应”框架嫁接到现有 UAV 导航栈，验证在部分已知室内环境中的覆盖完整度与执行稳健性。
- 对比基线：固定视角 coverage planner、仅全局规划不做局部视角补偿的方法、常规 frontier exploration
- 度量指标：coverage rate、轨迹长度、遮挡后补拍成功率、碰撞率、任务完成时间
- 预期结果：ASIP 风格的局部 view-angle adaptation 能在环境先验不完整时显著降低漏检，并保持较低执行代价
