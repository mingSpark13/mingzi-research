# 📖 花火格式笔记

## 🎯 题目

SpaAct: Spatially-Activated Transition Learning with Curriculum Adaptation for Vision-Language Navigation

## 📝 三句摘要

1. **问题背景**：现有基于 VLM 的 VLN 适配往往缺少对动态空间变化的显式建模，导致长程导航和未见场景泛化不稳。
2. **核心方法**：论文提出 SpaAct，用动作回溯与未来帧选择两种空间激活任务补强动态空间意识，并用 TriPA 三因子渐进课程学习稳定从基础移动到长程推理的迁移。
3. **关键结果**：在标准 VLN-CE 基准上，SpaAct 持续提升 VLM-based 导航性能并达到 SOTA，说明“转移预测 + 课程适配”是高价值增益口。

## 💎 价值评估

- **🔬 研究价值**：把“为什么走/走了会发生什么”拆成 backward reasoning 与 forward transition prediction，和 D06 想做的可解释导航推理链非常贴。
- **🚀 实践价值**：课程式适配对无人机空中 VLN 很有借鉴性，可直接迁到稀疏奖励、长视距、单目视角变化大的 aerial VLN 训练中。
- **📈 扩展潜力**：可与 OpenFly / GoalSwarm 一类空中平台结合，把未来帧预测替换成航迹片段或风险热图预测，进一步接到安全护栏。

## 🎯 可落地实验点

**实验设计**：把 SpaAct 的双任务监督迁到 UAV-VLN
- 对比基线：OpenFly / 现有 VLM-Nav mapless UAV 框架
- 度量指标：success rate、SPL、碰撞率、长指令完成率
- 预期结果：未来转移预测能明显降低长程指令中的迷航率，TriPA 风格课程能提高未见场景鲁棒性
