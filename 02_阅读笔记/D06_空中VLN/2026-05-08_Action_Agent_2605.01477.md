# 📖 花火格式笔记

## 🎯 题目

Action Agent: Agentic Video Generation Meets Flow-Constrained Diffusion

## 📝 三句摘要

1. **问题背景**：语言导航在真实机器人上常卡在“高层规划能说、低层执行不稳”，尤其无人机与多载体场景里缺少既能想象轨迹又能实时输出控制的统一方案。
2. **核心方法**：论文提出两阶段 Action Agent，先用 LLM 编排视频扩散模型生成物理可行的第一视角目标视频，再用 FlowDiT 把目标视频与语言指令映射成连续速度命令，实现 imagined trajectory 到 executable control 的解耦。
3. **关键结果**：43M 参数单模型可在仿真中达到 73.2% 导航成功率、在真实 Unitree G1 上达到 64.7% 任务完成率，并宣称同一范式可迁移到无人机与轮式平台，控制频率达 40–47Hz。

## 💎 价值评估

- **🔬 研究价值**：它把“视频想象 + 扩散控制”真正串成闭环，对 D06 很有价值，因为比纯 VLN policy 多了一层可解释的中间目标轨迹。
- **🚀 实践价值**：对主人做空中语言导航特别有参考性，尤其是先生成 ego-video 再落地速度控制的拆分，非常适合机载算力受限场景。
- **📈 扩展潜力**：若把 Stage I 换成更贴近航迹约束的空中世界模型、再把 Stage II 接 PX4/MPC safety shield，有机会变成可实机验证的无人机语言导航栈。

## 🎯 可落地实验点

**实验设计**：在主人现有无人机导航任务上比较“直接端到端 policy”与“目标视频中介 + 控制器”的双阶段方案。
- 对比基线：E2E-Fly / 直接 VLA 控制 / 不带 flow-constrained diffusion 的行为克隆控制器
- 度量指标：导航成功率、碰撞率、40Hz 控制稳定性、指令切换时的响应延迟
- 预期结果：双阶段方案在长走廊、遮挡转角和多目标语义切换时更稳，且更容易定位失败原因
