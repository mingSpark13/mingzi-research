# 📖 花火格式笔记

## 🎯 题目

Cross-Modal Navigation with Multi-Agent Reinforcement Learning

## 📝 三句摘要

1. **问题背景**：多模态 embodied navigation 往往受限于高质量对齐数据稀缺，以及单体大模型难以同时处理复杂视觉与声学输入。
2. **核心方法**：论文提出 CRONA，用多智能体强化学习把不同模态拆成轻量专长 agent，并用 centralized multi-modal critic 与 control-relevant auxiliary beliefs 做协同训练。
3. **关键结果**：在 visual-acoustic navigation 任务上，多智能体协同在性能与效率上都优于单智能体基线，且复杂大场景需要更丰富模态与更高模型容量。

## 💎 价值评估

- **🔬 研究价值**：把跨模态协作显式写成 MARL 问题，说明“多模态不一定要塞进一个大模型”，对具身导航建模很有启发。
- **🚀 实践价值**：可借鉴到机载视觉 + 声学/事件/测距等异构传感协同导航，但当前实验不直接是 UAV VLN 主线。
- **📈 扩展潜力**：适合延展到无人机多传感器协同、异构无人机编队、受限算力分布式决策等场景。

## 🎯 可落地实验点

**实验设计**：把视觉主导航策略与附加声学/深度/测距辅助策略拆成双 agent 协同训练，验证比单策略融合是否更稳。
- 对比基线：单体多模态 policy、视觉-only policy
- 度量指标：成功率、碰撞率、平均路径长度、推理延迟
- 预期结果：在复杂遮挡或目标声源可定位场景下，多 agent 协同比单体融合更稳且更易扩展
