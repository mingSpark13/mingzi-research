# 📖 花火格式笔记

## 🎯 题目

GRaD-Nav++: Vision-Language Model Enabled Visual Drone Navigation with Gaussian Radiance Fields and Differentiable Dynamics

## 📝 三句摘要

1. **问题背景**：现有语言驱动无人机导航常依赖手工技能、重模型或离板算力，难以在真实机载平台上实时闭环。
2. **核心方法**：论文提出轻量级机载 VLA 框架 GRaD-Nav++，在 3DGS 光照真实感仿真器中用可微强化学习训练低层控制，并用 MoE action head 提升多任务泛化与抗遗忘。
3. **关键结果**：仿真中训练/未见任务成功率达 83%/75%，真实机载中达 67%/50%，多环境迁移平均成功率也优于既有机载视觉语言飞行方案。

## 💎 价值评估

- **🔬 研究价值**：把 3DGS 仿真、可微动力学和轻量 VLA 机载部署绑在一起，补上 D06 里“能飞但不够轻”和“够轻但泛化弱”之间的空档。
- **🚀 实践价值**：强调 fully onboard 与 real-time，和主人关心的无人机落地链路高度贴合，可直接参考其感知-控制一体化设计。
- **📈 扩展潜力**：MoE action head、3DGS 场景构建、DiffRL 训练流程都能迁移到更复杂户外导航或空地协同任务。

## 🎯 可落地实验点

**实验设计**：验证 3DGS 仿真训练 + 轻量动作头 对无人机语言导航部署效率的提升。
- 对比基线：AutoFly、AerialVLA、LiteVLA-H
- 度量指标：成功率、机载推理延迟、未见环境泛化率、碰撞率
- 预期结果：GRaD-Nav++ 路线在机载延迟和跨环境稳定性上优于常规 VLA 基线
