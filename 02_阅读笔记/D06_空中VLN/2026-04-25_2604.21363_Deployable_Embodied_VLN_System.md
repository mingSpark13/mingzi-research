# 📖 花火格式笔记

## 🎯 题目

A Deployable Embodied Vision-Language Navigation System with Hierarchical Cognition and Context-Aware Exploration

## 📝 三句摘要

1. **问题背景**：现有 VLN 系统常在高层推理能力与真实机器人端实时部署之间二选一，难以落到资源受限平台。
2. **核心方法**：论文把系统拆成实时感知、记忆整合、高层推理三条异步链路，并用 cognitive memory graph + 子图推理配合 context-aware WTRP 探索规划。
3. **关键结果**：在仿真和真实机器人上同时提升导航成功率与效率，还维持了资源受限硬件上的实时运行能力。

## 💎 价值评估

- **🔬 研究价值**：把 UAV/具身 VLN 从“能推理但跑不动”往“能部署、能闭环”推进了一步，尤其适合给主人 D06 方向补“系统落地性”证据。
- **🚀 实践价值**：异步模块化 + 图记忆设计很适合移植到机载算力受限平台，可直接借鉴到无人机长程语义导航系统。
- **📈 扩展潜力**：后续可结合世界模型或 VLA 做更强前瞻推理，也可把 WTRP 目标改成多无人机协同探索。

## 🎯 可落地实验点

**实验设计**：在 OpenFly / 自建 UAV-VLN 环境中对比“纯 VLM 决策”“加入图记忆”“加入 context-aware exploration”三种配置。
- 对比基线：OpenFly 类 VLN pipeline、AutoFly、纯 frontier exploration
- 度量指标：Success Rate、SPL、平均决策延迟、单位路程指令完成率
- 预期结果：图记忆 + 上下文探索应在长指令、多房间、遮挡场景下明显优于纯 reactive baseline
