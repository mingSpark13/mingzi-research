# 📖 花火格式笔记

## 🎯 题目

Think before Go: Hierarchical Reasoning for Image-goal Navigation

## 📝 三句摘要

1. **问题背景**：图像目标导航在远距离、跨房间场景里很容易因为视觉线索稀疏而乱逛，端到端策略尤其容易在长程任务上失效。
2. **核心方法**：这篇提出 HRNav，用 VLM 先生成短程高层计划，再让在线 RL 低层策略按计划执行，并额外加了 Wandering Suppression Penalty 抑制无效徘徊。
3. **关键结果**：方法在仿真和真实环境都优于现有 image-goal navigation 基线，说明“高层推理+低层执行”对长程语义导航更稳。

## 💎 价值评估

- **🔬 研究价值**：把 VLM 推理显式放进导航闭环，不再只做端到端动作预测，和明子关心的“可解释中间接口”路线很贴。
- **🚀 实践价值**：对空中视觉语言导航也有直接启发，可把无人机长程语义任务拆成可执行的短程子目标，减少纯反应式策略漂移。
- **📈 扩展潜力**：后面若把 image-goal 改成语言目标、场景图或 PMI 接口，这套层次规划框架还能继续长大。

## 🎯 可落地实验点

**实验设计**：在空中 VLN 或室内无人机导航里加入“VLM 短程计划器 + 低层避障执行器”的双层结构，对比端到端策略。
- 对比基线：纯 imitation/RL 导航器、场景图导航器、带 waypoint 的 classic planner
- 度量指标：成功率、SPL、平均徘徊步数、重规划频率
- 预期结果：层次式推理在长程任务和遮挡场景下更稳，徘徊显著减少
