- **日期**: 2026-04-15

- **标题**: AerialVLA: A Vision-Language-Action Model for UAV Navigation via Minimalist End-to-End Control
- **方向**: D06 空中视觉语言导航
- **阶段**: 深挖（R723）
- **深挖来源**: 深挖论文队列 🔴 立即可挖

## 核心贡献（三句话）
1. **极简端到端 VLA**：原始视觉 + 模糊语言指令 → 连续物理控制信号，无需密集 oracle 引导或外部目标检测器。
2. **双视角感知**：前视 + 下视相机融合，减少视觉冗余同时保留导航和精确着陆所需的关键线索，并利于 sim-to-real 迁移。
3. **模糊方向提示机制**：仅用机载传感器生成模糊方向提示，完全消除对 oracle 引导的依赖，实现真正自主。

## 技术细节

### 架构
- **输入**：原始视觉观测（双视角）+ 模糊语言指令
- **输出**：连续 3-DoF 运动学指令（pitch/roll/yaw rate + 速度）+ 内置着陆信号
- **关键设计**：统一控制空间同时建模导航动作和降落动作，不需要额外目标检测器做精确定位

### dual-view perception
- 前视相机：负责前方语义导航
- 下视相机：负责精确地面定位（与语言目标接地）+ 降落信号
- 两者协同避免单视角歧义（正交视角解决俯仰大角度下的目标丢失问题）

### fuzzy directional prompting
- 传统方法需要 oracle 提供密集语义引导（如"目标在左前方15度方向"）
- AerialVLA 仅用机载 IMU/视觉里程计生成粗略方向提示（"目标大概在某个方向"）
- 无需 GPS，无需外部感知，真正机载自主

### 实验结果
- **TravelUAV 基准**：seen 环境 SOTA
- **unseen 环境泛化**：成功率约为最强基线的 3 倍
- 关键结论：极简、自主的方法比复杂模块化系统能捕获更鲁棒的视觉-运动表征

## 与本方向的关系
**直接对标龙虾导航路线**：龙虾项目也是"语言指令 → 无人机自主找到目标"，AerialVLA 提供了极简端到端 baseline，其 dual-view + fuzzy prompting 设计对龙虾的"手持D435i + 语义搜索"链路有直接参考价值。

## 可借鉴的技术点
1. **模糊方向提示**：用粗略语义引导替代精确 oracle，降低对外部感知依赖 → 可迁移到龙虾的"语音命令 → 粗略目标区域"链路
2. **双视角协同**：前视导航 + 下视定位的组合在龙虾手持 D435i 场景下也适用（双目/前向+下视配置）
3. **统一控制空间**：导航+降落信号联合建模 → 龙虾可以借鉴为"导航+抓取/放置"联合建模

## 局限 / 不足
1. 3-DoF 控制（水平面运动），未涵盖完整 6-DoF 动作空间
2. 着陆信号是二分类，精细降落控制仍依赖下游模块
3. 测试仅在 TravelUAV 仿真基准，未见真实室外飞行验证
4. 模糊提示的精度边界未系统分析（极端遮挡/光照条件下性能如何？）

## 对主人的直接启发
AerialVLA 的核心 insight 是：**模糊但自主 > 精确但依赖外部**。龙虾项目目前可能过度依赖精确 oracle（如标定的目标位置），建议探索：①双视角配置（如果 D435i 可兼作前向+下视）；②接受粗略方向提示作为中间表征，降低精确建图要求。
