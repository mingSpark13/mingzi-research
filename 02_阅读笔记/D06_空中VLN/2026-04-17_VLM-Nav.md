# 📖 花火格式笔记

## 🎯 题目

VLM-Nav: Mapless UAV navigation using monocular vision driven by vision-language models

## 📝 三句摘要

1. **问题背景**：很多 UAV 自主导航方案依赖激光雷达、地图构建或特定训练环境，成本高且泛化到未知场景困难。
2. **核心方法**：论文把单目 RGB 图像先转成 DepthAnything-V2 深度图，再交给 Gemini-flash / GPT-4o 判断障碍布局，最后由 FCN 融合 VLM 输出与相对航向角生成动作。
3. **关键结果**：在 AirSim Blocks 和 Downtown West 场景中达到接近 0.98 的任务完成率，说明“单目伪深度 + 通用 VLM”足以构成可工作的低成本 mapless UAV baseline。

## 💎 价值评估

- **🔬 研究价值**：它不是 D06 的最强路线，但很适合补一条“低成本、低工程门槛”的 baseline，专门对照复杂的 explorer / world-model 路线。
- **🚀 实践价值**：如果主人想先做机载快速 demo，而不是先搭完整语义地图，这篇是很好的轻量落地参考。
- **📈 扩展潜力**：可进一步替换成更强深度估计、更轻 VLM，或接到 D06 的 frontier 选择器前面做局部避障与安全壳层。

## 🎯 可落地实验点

**实验设计**：把 VLM-Nav 当成 D06 的轻量机载 baseline，直接和现有 3D frontier 主线做对照。
- 对比基线：[[2026-04-17_AutoFly]]、[[2026-04-17_AerialVLA]]、D06 现有 3D frontier 方案
- 度量指标：任务完成率、碰撞率、机载推理延迟、开放词汇搜索成功率
- 预期结果：VLM-Nav 在局部避障和快速部署上更轻，但在长时搜索、开放词汇目标定位和重规划恢复上弱于显式地图路线
