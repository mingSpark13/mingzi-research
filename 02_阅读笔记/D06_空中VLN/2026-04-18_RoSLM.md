# 📖 花火格式笔记

## 🎯 题目

Ro-SLM: Onboard Small Language Models for Robot Task Planning and Operation Code Generation

## 📝 三句摘要

1. **问题背景**：当前 LLM 机器人依赖云端部署，在网络不稳定或算力受限的无人机/小型地面机器人上无法正常工作。
2. **核心方法**：Ro-SLM 通过 LLM 蒸馏知识合成数据集（含多样化任务指令、对应代码生成），再以 LLM 作为 reward function 微调小型语言模型（SLM），实现机载部署。
3. **关键结果**：在 UAV 操作任务上，Ro-SLM 将原本无法支持机器人任务规划和代码生成的 SLM 提升至接近 LLM 的性能水平。

## 💎 价值评估

- **🔬 研究价值**：证明了"小模型 + 蒸馏 + 机载部署"路线在无人机任务规划上的可行性，为 D06 机载语言规划器提供了直接的 L3 新基线。
- **🚀 实践价值**：无需云端依赖，适合边缘部署，直接回答了"机载小模型能否完成复杂指令规划"的核心问题。
- **📈 扩展潜力**：蒸馏策略可迁移到其他 SLM（Phi-3-mini、Qwen2-1.5B 等）；可与 token 压缩/视觉 token 压缩协同研究。

## 🎯 可落地实验点

**实验设计**：复现 Ro-SLM 并与 D06 基线对比
- 对比基线：云端 GPT-4 / 纯视觉 token 压缩方法（如 ECO-SLAM）
- 度量指标：任务规划成功率、代码生成正确率、断网环境下规划时延
- 预期结果：机载 SLM 规划成功率应 >85%，时延 <200ms
