# 📖 花火格式笔记

## 🎯 题目

**WorldVLN: Autoregressive World Action Model for Aerial Vision-Language Navigation**

首个面向**空中视觉语言导航 (Aerial VLN)** 的**自回归世界动作模型 (World Action Model, WAM)**

## 📝 三句摘要

1. **问题背景**：传统 Vision-Language Navigation (VLN) 方法在复杂 3D 环境中面临挑战，空中导航尤为困难（动态障碍、有限计算资源）；现有方法多为全序列视频生成，缺乏实时闭环 action 能力。

2. **核心方法**：提出 **WorldVLN**，首个面向空中 VLN 的自回归 WAM。不同于生成完整视频的世界模型，WorldVLN 采用**潜在自回归视频骨干**预测短时域世界状态转移，**直接解码为可执行航点动作**。两阶段训练：①监督骨干+action decoder训练 ②action-aware GRPO优化。

3. **关键结果**：在公开室内外 benchmarks 上，WorldVLN 一致超越现有 VLA 基线，**成功率提升 12%+**，且在困难场景上优势更大；支持**零样本迁移到真机无人机部署**。

## 💎 价值评估

- **🔬 研究价值**：
  - 首个将 World Model 扩展到 VLN 任务的工作
  - 自回归闭环 action 预测，解决视频生成 world model 的延迟问题
  - Action-aware GRPO 训练范式，弥合 sim2real 差距

- **🚀 实践价值**：
  - 全面开源：GitHub + HuggingFace 模型权重
  - 支持真机部署（零样本迁移）
  - 提供完整训练 pipeline 和仿真环境

- **📈 扩展潜力**：
  - 可迁移到地面机器人 VLN
  - Action-aware GRPO 可复用到其他 action 任务
  - 潜在扩展到多智能体协同

## 🎯 可落地实验点

**实验设计**：在主人现有无人机项目中试点 WorldVLN 替换 VLN 模块

- **对比基线**：现有 VLN 方法（如 VLNBERT、PREVALENT 等）
- **实现方案**：
  1. 下载 HuggingFace 预训练权重：`EmbodiedCity/WorldVLN`
  2. 部署在线推理服务（`infer/run_server.sh`）
  3. 对比：指令执行成功率、路径长度、任务完成率
  4. 在 IndoorUAV / UAV-Flow 仿真环境验证
- **度量指标**：
  - 任务成功率 (%)
  - 平均路径长度
  - Sim2Real 差距（仿真 vs 真机）
- **预期结果**：WorldVLN 在复杂环境下成功率更高，真机迁移能力强
