# 📖 花火格式笔记

## 🎯 题目

AwareVLN: Reasoning with Self-awareness for Vision-Language Navigation

## 📝 三句摘要

1. **问题背景**：现有 VLN 方法常把 VLM 直接用于端到端动作预测，却缺少对 agent 自身状态、任务进度与场景关系的显式理解，导致推理不透明且长程执行易漂移。
2. **核心方法**：AwareVLN 引入 structural reasoning module，把空间关系与任务进度组织成 self-aware reasoning 机制，并配套 automatic data engine with progress division 提升训练监督质量。
3. **关键结果**：论文在 Habitat 系列 VLN 数据集上显著超过既有 SOTA，并被 CVPR 2026 接收，说明“显式自我状态/进度建模”对长程视觉语言导航是高价值增量。

## 💎 价值评估

- **🔬 研究价值**：它把“agent 是否知道自己走到哪一步了”从隐式能力拉成可建模对象，正好补 D06 里 `progress_score / verifier-first / stage-bounded recovery` 这条主线。
- **🚀 实践价值**：虽然论文主体是地面 VLN，但其 self-awareness 与 progress division 设计很适合迁到空中 VLN，作为 FineCog-Nav 之外的更轻量 planner 增强件。
- **📈 扩展潜力**：后续可直接比较 FineCog-Nav 的细粒度模块化，和 AwareVLN 这种内置自我认知结构，到底谁更适合机载预算受限的无人机导航。

## 🎯 可落地实验点

**实验设计**：把 AwareVLN 的 self-awareness / progress reasoning 思想压进 D06 的 `Semantic Waypoint Packet` 与 verifier 流水线，测试阶段感知是否能减少错误恢复成本。
- 对比基线：FineCog-Nav 模块化 planner、单体 end-to-end VLM planner、无 progress_score 的 packet-first planner
- 度量指标：长指令成功率、阶段切换错误率、repeat-reject rate、恢复成本比 RCR
- 预期结果：显式 progress reasoning 应主要降低 search→approach 的漂移与误升级，而不是单纯提高短程闭环控制精度
