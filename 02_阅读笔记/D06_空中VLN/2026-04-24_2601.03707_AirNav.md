# 📖 花火格式笔记

## 🎯 题目

AirNav: A Large-Scale Real-World UAV Vision-and-Language Navigation Dataset with Natural and Diverse Instructions

## 📝 三句摘要

1. **问题背景**：现有 UAV VLN 数据集大多依赖虚拟环境、指令不自然且规模偏小，导致真实空中语言导航落地受限。
2. **核心方法**：论文提出基于真实城市航拍数据构建的 AirNav 数据集，并在其上训练结合监督微调与强化微调的 AirVLN-R1。
3. **关键结果**：该工作把 UAV VLN 从合成场景推进到真实城市数据设定，并给出初步真实测试，说明语言导航模型具备向真实部署迁移的可行性。

## 💎 价值评估

- **🔬 研究价值**：这篇更像 D06 的基准地基，价值不在新控制器，而在把“真实城市数据 + 自然语言指令 + 初步实飞验证”三件事绑到一起，补上 UAV VLN 长期缺的真实数据底座。
- **🚀 实践价值**：对主人后续做无人机语义导航、空中 agent、VLA on drone 很有参考价值，尤其是数据采集协议、自然指令分布和 SFT→RFT 的训练套路。
- **📈 扩展潜力**：后续很适合顺着它做多轮对话导航、地图记忆增强、world model + VLN 融合，以及从离线 benchmark 走向在线闭环飞行。

## 🎯 可落地实验点

**实验设计**：复刻一个“真实航拍数据上的轻量 UAV VLN baseline”
- 对比基线：纯监督 VLN、AirVLN-R1 风格的 SFT+RFT、再加一个带显式场景记忆的 agent baseline
- 度量指标：任务成功率、路径偏差、指令跟随得分、跨城区泛化性能
- 预期结果：RFT 对复杂自然语言与长尾场景有帮助，但真正拉开差距的会是外部记忆和空间语义建模

**实验设计**：把 AirNav 当 D06 数据底座，验证“无人机 world model + language planner”
- 对比基线：端到端 VLN policy、加 scene graph 的 planner、加 latent world model 的 planner
- 度量指标：长程导航成功率、重规划次数、遮挡/岔路场景恢复率
- 预期结果：带 world model 的方法在长程与歧义指令上更稳，更贴近主人后续研究主线
