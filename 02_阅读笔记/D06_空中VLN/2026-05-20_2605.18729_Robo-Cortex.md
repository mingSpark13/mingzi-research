# 📖 花火格式笔记

## 🎯 题目

Robo-Cortex: A Self-Evolving Embodied Agent via Dual-Grain Cognitive Memory and Autonomous Knowledge Induction

## 📝 三句摘要

1. **问题背景**：具身导航在未见环境里常受“经验失忆”限制，现有轨迹驱动或反应式策略很难把过往成功/失败经验抽象成可迁移策略。
2. **核心方法**：论文提出 Robo-Cortex，用 AKI 机制把多模态轨迹蒸馏成自然语言启发式规则库，再配合短期反思记忆、长期原则记忆和 imagine-then-verify 决策回路持续自进化。
3. **关键结果**：在 IGNav、AR、AEQA 及初步真实机器人实验中，Robo-Cortex 相比强基线在成功率与探索效率上都有稳定提升，并在未见环境启发式迁移下收益更明显。

## 💎 价值评估

- **🔬 研究价值**：它把“导航经验 -> 语言化原则 -> 再利用”这条链打通了，特别适合主人关注的具身导航长期适应与知识累积问题。
- **🚀 实践价值**：若后面做 UAV-VLN 或开放环境巡检，这种短期反思 + 长期原则库的结构很适合沉淀飞行经验和避障启发式。
- **📈 扩展潜力**：可继续和空中视觉语言导航、世界模型 rollout、故障日志自动总结结合，做跨轮次自进化导航代理。

## 🎯 可落地实验点

**实验设计**：在 UAV 语义导航或 object-goal 导航任务中，引入“失败案例语言归纳 + 启发式规则库”机制，验证长期部署时的泛化收益。
- 对比基线：普通 reactive policy、仅短期记忆版本、无启发式迁移版本
- 度量指标：SPL、任务成功率、重复犯错率、未见场景迁移效果
- 预期结果：语言化经验库能降低重复失误，并提升长程开放环境中的探索效率
