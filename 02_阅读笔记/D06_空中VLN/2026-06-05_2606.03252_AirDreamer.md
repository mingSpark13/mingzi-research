# 📖 花火格式笔记

## 🎯 题目

AirDreamer: Generalist Drone Navigation with World Models

## 📝 三句摘要

1. **问题背景**：无人机在陌生拥挤环境里导航时，传统方法强依赖手工感知管线和环境特定规则，跨场景泛化很弱。
2. **核心方法**：论文提出 AirDreamer，在 world-model 驱动的环境理解之上叠加强化学习策略，并用稀疏奖励设计鼓励自主绕障与摆脱局部最优。
3. **关键结果**：它在仿真和真实无人机上都展示出对复杂未知环境的泛化能力，在困难地图上比最强基线高 5.3% 成功率，而且部署时无需额外调参即可完成 sim2real。

## 💎 价值评估

- **🔬 研究价值**：这篇把世界模型从机械臂操控往空中导航方向推进了一步，很贴主人关注的 UAV 自主导航与泛化问题。
- **🚀 实践价值**：如果主人后面做无人机在未知场景中的自主导航，这篇可以直接提供“world model + RL policy + sparse reward”这条可复现主线。
- **📈 扩展潜力**：后续可以接语言目标、主动感知或跨机型迁移，做成更接近空中 VLA / UAV reasoning 的统一导航框架。

## 🎯 可落地实验点

**实验设计**：在现有 UAV 仿真环境中复现 AirDreamer 的 world-model 环境表征，并和端到端导航策略比较陌生场景泛化能力。
- 对比基线：端到端 PPO/SAC、带手工地图先验的方法、无 world model 的 RL 版本
- 度量指标：导航成功率、碰撞率、局部最优逃逸次数、真实部署零调参成功率
- 预期结果：world model 版本在未知复杂场景中更稳，并在 sim2real 时显著减少规则工程依赖
