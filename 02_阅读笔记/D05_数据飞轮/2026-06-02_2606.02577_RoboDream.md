# 📖 花火格式笔记

## 🎯 题目

RoboDream: Compositional World Models for Scalable Robot Data Synthesis

## 📝 三句摘要

1. **问题背景**：真实机器人示教采集成本高、覆盖窄，现有生成式扩数又常停留在视觉表面增强，容易出现 embodiment hallucination，难真正提升下游策略。  
2. **核心方法**：RoboDream 提出以机器人运动渲染为锚点、显式条件化场景与物体先验的 compositional world model，把“轨迹执行”和“环境生成”解耦，支持 retrieval-and-rebirth 与 prop-free teleoperation 两种数据扩增模式。  
3. **关键结果**：论文报告生成的 photorealistic demonstration 能稳定提升下游 manipulation policy 表现，并显著降低真实世界数据需求，属于数据飞轮方向里非常直接的高杠杆工作。

## 💎 价值评估

- **🔬 研究价值**：这篇不是泛泛“生成机器人视频”，而是把 world model 真正压到 **机器人数据生产链** 上，直接回答 D05 最关键的问题：怎么用结构化生成减少真实示教依赖。  
- **🚀 实践价值**：对主人后续做 UE / AirSim / 真实回放混合数据工厂很有启发，可以把已有轨迹重生到新场景、新视角、新物体里，而不必每次重采动作。  
- **📈 扩展潜力**：后续很适合和 F-ACIL、ShapeGen、affordance 标注体系拼接，形成“轨迹资产复用 → 环境重生 → 策略训练”的完整闭环。

## 🎯 可落地实验点

**实验设计**：验证“真实示教 + RoboDream 式生成扩数”是否比纯真实数据或普通视频增强更能提升类别级操作泛化  
- 对比基线：仅真实示教 / 真实示教 + 普通视觉增强 / 真实示教 + compositional world-model 扩数  
- 度量指标：未见物体成功率、未见场景成功率、每单位真实示教带来的性能提升  
- 预期结果：结构化生成扩数方案在真实数据相同预算下能取得更高泛化与更低数据需求
