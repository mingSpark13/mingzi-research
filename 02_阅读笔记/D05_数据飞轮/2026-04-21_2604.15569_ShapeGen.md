# 📖 花火格式笔记

## 🎯 题目

ShapeGen: Robotic Data Generation for Category-Level Manipulation

## 📝 三句摘要

1. **问题背景**：类别级操作想在未见实例上泛化，最大瓶颈是缺少覆盖充足几何变化的大规模训练数据，手工采集既贵又慢。
2. **核心方法**：ShapeGen 走一条 **simulator-free + 3D-aware** 数据生成路线，把物体形状分解、重组并生成多样化几何实例，再合成对应操作数据，用更低成本补齐类别内形状覆盖。
3. **关键结果**：论文主张该方法能显著提升 category-level manipulation 的泛化，说明“先扩形状分布、再学策略”是数据飞轮里很值钱的一环。

## 💎 价值评估

- **🔬 研究价值**：它直接补 D05 里长期缺的一块——**如何系统扩展 object factor**。相比只做场景随机化或轨迹增强，ShapeGen 把“物体几何多样性”单独抽出来，特别适合和 F-ACIL 的 object/action/environment 因子化框架拼起来。
- **🚀 实践价值**：对主人后续 UE/AirSim 数据工厂很有启发：不该只扩环境背景和任务脚本，还要有一层专门的 **资产形状生成/变体采样** 模块，否则学到的策略很容易卡在 seen geometry。
- **📈 扩展潜力**：后续可继续接到 affordance 标注、接触先验甚至跨载体重定向上，形成“形状生成 → 操作轨迹生成 → 策略训练”的完整闭环。

## 🎯 可落地实验点

**实验设计**：验证“只做环境随机化”与“加入形状生成扩数”对类别级操作泛化的影响
- 对比基线：固定资产训练 / 传统 domain randomization / ShapeGen 风格形状扩数
- 度量指标：未见实例成功率、类别内几何泛化、最少真实示教数
- 预期结果：显式扩展物体形状分布后，未见实例成功率更稳，且真实补数据需求下降
