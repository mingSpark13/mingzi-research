# 📖 花火格式笔记

## 🎯 题目

SAS-Prompt: Large Language Models as Numerical Optimizers for Robot Self-Improvement

## 📝 三句摘要

1. **问题背景**：机器人策略自我改进往往依赖黑盒优化或额外学习器，难解释、调试重，还不容易直接利用过往执行轨迹中的语义信息。
2. **核心方法**：论文提出单一 SAS Prompt（Summarize, Analyze, Synthesize），把 LLM 当作可解释的数值优化器，直接基于历史机器人轨迹做总结、分析和新策略合成，完成迭代式自我提升。
3. **关键结果**：方法在仿真和真实桌球任务中都展示了可行性，说明 LLM 不只是高层对话器，也能承担可解释的策略搜索与行为改写角色。

## 💎 价值评估

- **🔬 研究价值**：这篇很适合放进“LLM 直接参与闭环改策略”分支，它不是让 LLM 只出语言计划，而是把 prompt 变成可解释搜索器。
- **🚀 实践价值**：对明子的数据飞轮与 VLA 调优有启发，尤其适合做失败轨迹复盘、规则修补、探索方向生成这类低成本外环优化。
- **📈 扩展潜力**：如果把 SAS 的 summarize/analyze/synthesize 结构化输出接进实验记录和自动回灌系统，就能做半自动研究员式训练闭环。

## 🎯 可落地实验点

**实验设计**：构造一个“策略失败日志 → LLM 复盘 → 新参数/新规则建议 → 再评测”的自动外环，对比无复盘与 SAS 风格复盘的迭代收益。
- 对比基线：人工经验调参、无解释的黑盒搜索、普通 chain-of-thought 建议
- 度量指标：每轮成功率提升、收敛轮数、建议可解释性、失败模式覆盖率
- 预期结果：SAS 风格结构化 prompt 能更稳定地产出可执行修正建议，并缩短人工分析时间
