# 📖 花火格式笔记

## 🎯 题目

NoiseGate: Learning Per-Latent Timestep Schedules as Information Gating in World Action Models

## 📝 三句摘要

1. **问题背景**：World Action Model 往往把所有未来 latent frame 绑定到同一个去噪时间步，默认每个 latent 对动作生成同等可靠，这个假设其实很粗暴。
2. **核心方法**：论文提出 NoiseGate，为每个 latent frame 学习独立 timestep schedule，把噪声强度当成信息门控策略，动态控制未来观测对动作 token 的 Key/Value 贡献。
3. **关键结果**：作者把该机制接到 joint video-action generation 的 WAM 范式中，用更细粒度的信息可靠性建模改善动作生成质量与未来观测利用效率。

## 💎 价值评估

- **🔬 研究价值**：这篇很贴 D05/D01 交界——它不是单纯换 backbone，而是在 WAM 里把“未来信息何时可信”显式建模，适合我们关心的 packet/interface honesty 线索。
- **🚀 实践价值**：如果后续做 video-conditioned policy 或 world model policy，这种 per-latent reliability gating 很可能比一刀切 shared timestep 更稳。
- **📈 扩展潜力**：可继续追问它能否和 flow matching、planner-time vs consume-time attribution 一起形成更可解释的生成控制接口。

## 🎯 可落地实验点

**实验设计**：在自家 video-conditioned action model 上复现 shared timestep vs per-latent timestep schedule，对比未来信息门控收益。
- 对比基线：共享 timestep 的 WAM / 仅动作扩散策略 / 带 NoiseGate 的 joint video-action model
- 度量指标：动作成功率、预测帧一致性、long-horizon error accumulation、OOD 鲁棒性
- 预期结果：per-latent 门控能减少错误未来帧对动作头的污染，特别是在长时预测和多步决策中更明显
