# 📖 花火格式笔记

## 🎯 题目

ComSim: Building Scalable Real-World Robot Data Generation via Compositional Simulation

## 📝 三句摘要

1. **问题背景**：机器人大模型想吃到够多高质量数据，但纯真实采集太贵、纯经典仿真又不够像，导致规模和真实性长期拉扯。
2. **核心方法**：ComSim 提出 **classical simulation + neural simulation** 结合的 **compositional simulation**，走一条小量真实种子驱动的 closed-loop real-sim-real 扩数路线，自动生成更贴近真实世界的 action-video pairs。
3. **关键结果**：它给出一个很像 D05 主线骨架的系统答案：不是在“真实 or 仿真”二选一，而是通过组合式仿真把真实性、可控性和规模三者一起往上推。

## 💎 价值评估

- **🔬 研究价值**：这篇和 D05 方向高度直连，几乎就是把“混合仿真数据飞轮”写成了论文版系统架构。它尤其补强了现有路线里 **real-sim-real 闭环** 的工程可行性。
- **🚀 实践价值**：对主人现在的 UE/AirSim/真实视频混合路线很有参考价值：后续不一定非要追求一个全能仿真器，而可以用 **经典几何/物理 + 神经渲染/神经补全** 的组合管线去扩数据。
- **📈 扩展潜力**：可自然接到 world model、自动重置、语义规划甚至跨 embodiment 配对数据生成，是 D05 后续很值得保留的一篇主线候选。

## 🎯 可落地实验点

**实验设计**：比较纯真实少样本、纯仿真扩数、ComSim 风格 real-sim-real 组合扩数对下游策略的效果
- 对比基线：真实小数据训练 / 纯仿真训练 / 组合式仿真扩数训练
- 度量指标：真实部署成功率、数据成本、域外泛化、动作视频一致性
- 预期结果：组合式仿真在成本-性能比上优于纯真实和纯仿真两端
