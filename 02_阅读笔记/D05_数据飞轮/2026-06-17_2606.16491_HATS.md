# 📖 花火格式笔记

## 🎯 题目

HATS: A Human-Agent Teleoperation System for Multi-Arm Data Collection

## 📝 三句摘要

1. **问题背景**：多臂协作示教既难靠单人完成，也不适合长期依赖双人专家协同，数据采集成本和协调负担都很高。  
2. **核心方法**：HATS 让人类直接控制两条主臂，再由 MLLM agent 无训练接管两条辅助臂与局部子任务，同时允许语音纠偏，构成单人+智能体的多臂遥操作采集系统。  
3. **关键结果**：论文显示 HATS 的采集效率和成功率接近双人专家团队，且下游策略训练证明数据质量有效，是 D05 数据飞轮里非常实用的“降本采集”方案。

## 💎 价值评估

- **🔬 研究价值**：它不只是做 teleop 硬件，而是重新设计了“人负责关键自由度、agent 负责辅助臂”的数据采集分工，对复杂协作任务的 demonstration pipeline 很有启发。  
- **🚀 实践价值**：主人如果后面做多机械臂、空地协同或复杂 manipulation 采集，这篇能直接借鉴成“主操作者 + 助手机器人/agent”的采数框架，明显省人。  
- **📈 扩展潜力**：可以继续和 RoboDream、自动标注、仿真重放拼起来，形成“低成本采集 → 智能辅助扩展 → 下游策略训练”的完整飞轮。

## 🎯 可落地实验点

**实验设计**：比较单人直控、双人协作、单人+agent 辅助三种示教模式的数据成本与下游收益
- 对比基线：单人 teleop / 双人 teleop / HATS 式单人+agent teleop
- 度量指标：单位小时有效轨迹数、任务成功率、碰撞次数、下游策略成功率
- 预期结果：HATS 式协同在人力成本接近单人的同时，数据质量逼近双人专家设置
