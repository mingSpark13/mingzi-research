# 📖 花火格式笔记

## 🎯 题目

GASE — 基于 3D Gaussian Splatting 的全自动 Embodied 仿真环境重建系统

## 📝 三句摘要

1. **问题背景**：真实世界训练 embodied agent 成本极高,现有基于 3DGS 的仿真场景重建在数据采集效率、前景物体分割、sim-to-real gap 三个关键环节上仍有明显短板,导致实际部署困难。
2. **核心方法**：GASE 用全景相机阵列做多视角快速扫描,提出基于相机位姿的 2D 域前景物体鲁棒提取策略 + 高保真场景修复,前后景独立重建后无缝导入物理仿真器,实现"扫一圈即可用"的端到端管线。
3. **关键结果**：分割精度比现有 3DGS 方法高 10%+,inpainting 质量 SOTA,真实机器人操作与导航任务上 sim-to-real 性能差距 < 10%,几乎逼近纯真实数据训练。

## 💎 价值评估

- **🔬 研究价值**：把"3DGS 重建 + 仿真生成"这条线从"学术验证"推到"工程可用",展示了用 GS 做 embodied 仿真环境的真正可行路径(分割精度 + 10% 是关键质量瓶颈突破)。
- **🚀 实践价值**：主人做空地协同、空中操作项目时,常常需要"扫一个真实场地→在仿真里复刻→做数据飞轮",GASE 直接给出完整管线参考,省去自建 pipeline 的工程量。
- **📈 扩展潜力**：可以与 FlyMirage(LLM 场景生成)互补:GASE 负责"真实→仿真复刻",FlyMirage 负责"文本→仿真生成",两条路合起来就是"主动数据工厂 + 真实迁移"的完整飞轮。

## 🎯 可落地实验点

**实验设计**：在主人实际无人机操作场地复现 GASE 管线,测真实可用性
- 对比基线：iGS / GaussianShader / 普通 COLMAP+3DGS 重建
- 度量指标：单次场景扫描时间、GS 重建 PSN/SSIM、前景物体 iou、Isaac Sim 导入后真实机械臂策略零样本成功率
- 预期结果：前景分割 IoU 比 iGS 高 10%,真实场景策略训练 sim-to-real gap 控制在 10% 以内
