# 📖 花火格式笔记

## 🎯 题目

Agent-Centric Visual Reinforcement Learning under Dynamic Perturbations

## 📝 三句摘要

1. **问题背景**：视觉强化学习在动态噪声、腐蚀类型切换这类非平稳扰动下会明显失效，现有方法缺少针对这种真实部署分布漂移的系统基准与稳健机制。
2. **核心方法**：论文先提出带 Markov-switching 扰动的 VDCS 基准，再用 ACO-MoE 把“任务相关前景恢复”和“扰动去耦”做成 agent-centric mixture-of-experts 表征，在进入 RL 策略前先清洗视觉输入。
3. **关键结果**：ACO-MoE 在动态扰动下恢复了 95.3% 的 clean performance，还在 DMControl 泛化基准上拿到 SOTA，说明“先分离扰动、再学控制”比直接重建式表征更稳。

## 💎 价值评估

- **🔬 研究价值**：这篇把非平稳视觉退化正式做成 benchmark，很适合主人后面研究真实机器人/无人机部署时的鲁棒性问题，不再只看静态 domain randomization。
- **🚀 实践价值**：对 D05 数据飞轮很有用，因为它提示数据闭环不该只拼更多数据，还要专门构造“扰动课程+恢复模块”，让系统能吃下持续变化的脏数据。
- **📈 扩展潜力**：可迁到无人机第一视角、户外感知退化、低照度/雨雾/压缩噪声场景，把 agent-centric restoration 作为飞行控制或导航前级模块。

## 🎯 可落地实验点

**实验设计**：给主人现有 UAV/VLA 训练流加一个“动态视觉扰动课程 + 前景恢复模块”
- 对比基线：原始视觉策略 / 静态增强 / 动态扰动课程 / 动态扰动课程+ACO式前端
- 度量指标：OOD 成功率、恢复后控制稳定性、轨迹偏差、样本效率
- 预期结果：动态扰动课程会显著提前暴露脆弱点，而 agent-centric 前端能减少视觉污染向控制层传播
