# 📖 花火格式笔记

## 🎯 题目

GenerativeMPC: VLM-RAG-guided Whole-Body MPC with Virtual Impedance for Bimanual Mobile Manipulation

## 📝 三句摘要

1. **问题背景**：双臂移动操作既要理解语义上下文，又要在接触任务中保持安全柔顺，纯端到端方法难解释，经典控制又缺少场景语义。
2. **核心方法**：论文提出 GenerativeMPC，用 VLM-RAG 把视觉和语言上下文转成 whole-body MPC 的速度/安全约束，并联动虚拟阻抗参数实现情境自适应柔顺控制。
3. **关键结果**：方法在双臂移动操作场景中展示了更强的语义可解释性与安全控制耦合能力，给“生成式高层 reasoning + 经典可验证控制”提供了清晰范式。

## 💎 价值评估

- **🔬 研究价值**：它不是又一个黑箱 VLA，而是把生成模型输出落到可解释控制参数上，和明子关心的 verifier / 可验证接口路线很贴。
- **🚀 实践价值**：适合人机交互、双臂搬运、移动操作等安全敏感任务，尤其适合需要 compliance 的场景。
- **📈 扩展潜力**：可延展到 UAV-manipulation、全身控制、具身 agent 的高层约束生成，甚至接入 PMI 风格接口。

## 🎯 可落地实验点

**实验设计**：做“语言-视觉上下文 → 控制约束”的可验证接口原型
- 对比基线：固定 MPC 约束 + 手工阻抗参数方案
- 度量指标：碰撞率、任务完成率、控制抖动、解释一致性
- 预期结果：动态约束生成在复杂接触场景下更安全，且更容易通过 verifier 做异常拦截
