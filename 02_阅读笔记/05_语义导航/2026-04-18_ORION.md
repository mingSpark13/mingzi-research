# 🎯 题目

ORION: A Holistic End-to-End Autonomous Driving Framework by Vision-Language Instructed Action Generation

# 📝 三句摘要

1. **问题背景**：传统模块化自动驾驶（感知→规划→控制）存在级联误差，且难以处理自然语言指令（如"在便利店门口停下"）；端到端方法虽统一但缺乏语言理解能力。
2. **核心方法**：ORION 以 VLA 架构统一处理多模态传感器输入和自然语言指令，直接输出驾驶动作，实现感知-决策-控制全链路闭环；框架可与现有系统（Autopilot/Apollo）即插即用。
3. **关键结果**：展示了 VLA 架构在自动驾驶落地方的可行性，Vision-Language Instructed Action 范式可迁移到无人机指令跟随任务，与龙虾项目的语言导航目标高度相关。

# 💎 价值评估

- **🔬 研究价值**：证明了 VLA 架构在自动驾驶落地方的可行性，与 Alpamayo-R1 等世界模型工作形成互补（一个侧重世界模型预测，一个侧重 VLA 动作生成）。
- **🚀 实践价值**：即插即用的框架设计降低了工程部署门槛；语言指令接口为无人机指令跟随提供了可参考的交互范式。
- **📈 扩展潜力**：VLA + 语言指令框架可迁移到无人机 VLN 任务，实现"用自然语言指令控制无人机完成复杂任务"的愿景。

# 🎯 可落地实验点

**实验设计**：将 ORION 的 Vision-Language Instructed Action 范式迁移到无人机指令跟随任务
- 对比基线：纯视觉端到端控制（无语言指令）、模块化感知→规划→控制
- 度量指标：指令跟随准确率、任务完成率、泛化到未见指令的能力
- 预期结果：语言指令接口显著提升任务可解释性和零样本指令泛化能力
