# HTNav: 城市空中VLN混合分层框架

- **arXiv**: 2604.08883
- **方向**: D06 空中视觉语言导航
- **阅读深度**: 摘要级深挖（新候选入库）
- **日期**: 2026-04-16

## 核心贡献

HTNav 提出一种**混合 IL+RL 分层导航框架**，核心三件套：

1. **阶段化训练机制 (Staged Training)**：IL 打底保证基础导航稳定性，RL 增强环境探索能力
2. **分层决策机制 (Tiered Decision-Making)**：宏观路径规划 ↔ 细粒度动作控制协同
3. **地图表征学习模块 (Map Representation Learning)**：增强对开放域空间连续性的理解

## 与龙虾/D06的关系

- **方法定位**：HTNav 是 APEX + AirNav 主骨架之外的**第三条路**——显式分层的 IL-RL 混合框架
  - APEX = 动态时空语义建图 + RL 动作决策（解耦记忆 explorer）
  - AirNav = 真实城市 benchmark + SFT/RLFT 训练配方
  - **HTNav = 分层 IL-RL + 地图表征学习**（强调稳定基础策略 + 空间连续性）
- **与龙虾 GoalSearch 的关联**：分层决策（宏观规划+微观控制）与龙虾"UE仿真+视觉语言导航"契合，阶段化训练适合在仿真中先学基础策略再强化探索

## 关键发现

HTNav 的分层结构值得借鉴：它的 **"宏规划+微控制"协同** 思路与龙虾 GoalSearch 的"全局探索+局部确认"双阶段流程一致，可作为后续工程实现的参考架构。
