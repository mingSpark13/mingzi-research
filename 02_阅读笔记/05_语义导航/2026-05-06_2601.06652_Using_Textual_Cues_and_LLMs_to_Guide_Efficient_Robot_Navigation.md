# 📖 花火格式笔记

## 🎯 题目

Using Textual Cues and LLMs to Guide Efficient Robot Navigation

## 📝 三句摘要

1. **问题背景**：传统语义导航依赖显式目标检测或密集探索，面对部分观察和开放环境时效率差，缺少把场景文字线索转成导航先验的能力。
2. **核心方法**：论文用 LLM 从环境中的文本提示与局部观察中推断目标高概率区域，把语言常识转成导航策略先验，从而在不完整地图条件下指导机器人做更高效的目标搜索。
3. **关键结果**：web_search 命中度很高，摘要显示该方法能在语义导航框架里显著提升搜索效率，属于“把 LLM 常识直接接进导航决策”的实用路线，值得纳入 D03/D06 交叉关注。

## 💎 价值评估

- **🔬 研究价值**：不是简单把 LLM 当对话壳，而是把文本 cue 变成导航启发式，正好契合“语义先验如何减少盲搜”这个长期问题。
- **🚀 实践价值**：对室内服务机器人、无人机室内巡检、开放词汇找目标任务都很实用，尤其适合地图不完整、目标不在视野中的场景。
- **📈 扩展潜力**：可以和空中 VLN、主动感知、场景图谱推理结合，形成“高层语义猜测 + 低层避障执行”的双层导航系统。

## 🎯 可落地实验点

**实验设计**：把文本 cue 推理模块接到现有语义导航或 UAV VLN 管线中，对比“无语义先验 / 规则先验 / LLM 文本先验”三种搜索策略。
- 对比基线：frontier exploration、CLIP-based semantic search、规则式 room prior
- 度量指标：到达成功率、平均路径长度、首次发现目标时间、token 调用开销
- 预期结果：LLM 文本先验能在开放环境下减少无效探索，并在长尾目标上优于规则启发式
