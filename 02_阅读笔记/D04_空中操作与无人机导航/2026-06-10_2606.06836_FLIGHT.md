# 📖 花火格式笔记

## 🎯 题目

**FLIGHT / Think Like a Pilot** — 面向长程细粒度指令的 UAV 导航 Benchmark 和异频异步 Pilot VLA

## 📝 三句摘要

1. **问题背景**：语言引导的 UAV 任务要求执行"多阶段语义指令+平滑物理可行连续飞行动作"，而现有 UAV VLN 基准仅做粗粒度/离散动作，UAV VLA 任务又只关注短原子动作，长程细粒度连续控制是空位。
2. **核心方法**：作者提出 **FLIGHT benchmark**（包含 Fine-grained VLN 与 Long-horizon Flow 两个 split，多阶段指令+密集 6-DoF 轨迹标注），以及 **FLIGHT VLA 异频异步架构**——低频 Streaming Pilot VLM 负责任务状态推理与子目标预测，高频扩散动作模型负责连续 6-DoF 控制，二者用显式 Pilot Reasoning 文本监督解耦。
3. **关键结果**：在 FLIGHT benchmark 闭环评测中，FLIGHT VLA 在多阶段完成度、子目标贴合度、末端控制精度上稳定超过代表性 VLN/VLA 基线；其训练出的 Streaming Pilot Reasoning VLM 同时也提升了 UAV 视频推理能力。

## 💎 价值评估

- **🔬 研究价值**：直接对准"单纯语言→轨迹不够，需要长程推理/执行"这一痛点，是首份同时给多阶段指令标注和密集 6-DoF 轨迹的 UAV VLA 基准。Pilot Reasoning 显式监督的解耦 VLM+扩散动作模型是范式新意。
- **🚀 实践价值**：对 AirSpark 的"主动数据工厂"很关键——FLIGHT 可以作为长程连续任务评测的外部参考 benchmark，避免自造基准无外部标尺。
- **📈 扩展潜力**：异频异步 Streaming Pilot VLM + 扩散动作模型的范式理论上可移植到 UAV 跟踪、抓取接近、移动目标拦截等多类任务；其 Pilot Reasoning 文本质上是把"思考过程"显式监督出来，可与 AHEAD 的 future-token wrapper 路线形成对照。

## 🎯 可落地实验点

**实验设计**：在 AirSpark 的园区低空长程任务上，借鉴 FLIGHT 的"低频 VLM 状态推理 + 高频扩散动作"双流架构，做异频异步 pilot-policy 消融。
- **基线**：单频端到端 VLA（OpenVLA-7B 直接微调）/ 仅离散 waypoint VLA / 同步 VLM+扩散（无 reasoning 文本监督）
- **数据集**：用 AirSpark 自采长程任务 episodes + 复刻 FLIGHT 的多阶段指令标注
- **度量**：长程任务成功率（≥3 个子目标）、子目标贴合度、轨迹平滑度、控制频率（Pilot VLM 1Hz vs 动作头 50Hz）、推理显存峰值
- **预期**：异步架构在长程任务上比单频 VLA 提升 ≥15% 子目标完成度，并显著降低 7B VLM 推理频率带来的计算开销
