# 📖 花火格式笔记

## 🎯 题目

**UAV-VL-R1 / HRVQA-VL v2** — 通过 SFT + 多阶段 GRPO 泛化 UAV 视觉推理

## 📝 三句摘要

1. **问题背景**：VLM 在自然图像上泛化强，但在 UAV 图像上退化——UAV 图像高分辨率、空间语义复杂、有严格实时性约束。通用 VLM 难以直接迁移。
2. **核心方法**：作者构建 **HRVQA-VL**——50,019 个高分辨率 UAV 视觉问答标注样本，覆盖 **8 类 UAV 视觉推理任务**。训练 pipeline 用 **SFT + 多阶段 GRPO** 联合后训练。
3. **关键结果**：相对 base VLM 在 UAV 视觉推理任务上显著提升，特别是 VQA、场景描述等任务上。验证 SFT+GRPO 对 UAV 域迁移的有效性。

## 💎 价值评估

- **🔬 研究价值**：把"RL 后训练（GRPO）"用在 VLM 跨域泛化上，是 VLM + 强化学习+UAV 域的结合。
- **🚀 实践价值**：对 AirSpark 主动语言描述生成与场景问答的"硬实时性"约束有借鉴——VLM 需在 UAV 上运行必须轻量化+高效。
- **📈 扩展潜力**：多阶段 GRPO 训练 pipeline 可推广到其他 UAV 任务。

## 🎯 可落地实验点

**实验设计**：把 UAV-VL-R1 的 SFT + 多阶段 GRPO pipeline 移植到 AirSpark 的 VLM 后训练。
- **场景**：AirSpark 园区场景
- **基线**：仅 SFT / 仅 GRPO
- **度量**：UAV VQA 准确率、推理延迟、模型大小
- **预期**：SFT+多阶段 GRPO 比单阶段在 VQA 准确率上提升 ≥10%
