# 📖 花火格式笔记

## 🎯 题目

**WildRoadBench** — 野外航空道路损伤 grounding 基准

## 📝 三句摘要

1. **问题背景**：UAV 道路损伤检测是巡检/遥感的重要任务，但缺乏标准化的 grounding benchmark，难以公平对比 VLM 与 agent 方法。
2. **核心方法**：作者发布 **WildRoadBench**——同一图像集 + 同一 AP50 指标下，两个评测协议：**VLM TRACK**（直接视觉 grounding 测量，VLM 能否定位损伤）、**AGENT TRACK**（LLM 驱动 agent 自主研究+工程能力评估）。数据由专业 UAV 标注。
3. **关键结果**：为 UAV 道路损伤检测提供统一 benchmark，可同时评测 VLM 视觉 grounding 与 LLM agent 工程能力两个轴。

## 💎 价值评估

- **🔬 研究价值**：把"VLM grounding" 与 "LLM agent" 放在同一 benchmark 上双轴评估，是评测设计的新尝试。
- **🚀 实践价值**：对 AirSpark 巡检/区域目标描述任务有参考价值——可借鉴其双轴评测协议。
- **📈 扩展潜力**：双轴评测协议可推广到其他巡检类任务。

## 🎯 可落地实验点

**实验设计**：把 WildRoadBench 的"双轴评测"思想移植到 AirSpark 园区巡检评估。
- **场景**：AirSpark 园区巡检（消防通道/路灯/井盖/绿化等）
- **基线**：单轴 VLM grounding / 单轴 agent
- **度量**：VLM grounding AP50、agent 任务完成率、跨协议一致性
- **预期**：双轴评测能更全面反映 VLM/agent 真实能力
