# 📖 花火格式笔记

## 🎯 题目

AgenticRL: Self-Refining Agentic Reinforcement Learning for Vision-Conditioned UAV Navigation

## 📝 三句摘要

1. **问题背景**：无人机导航里的强化学习很依赖人工奖励设计和反复调参，导致部署慢、泛化差。
2. **核心方法**：论文让多模态 GPT agent 参与奖励函数生成、训练后诊断和闭环修正，再在推理时依据图像与语言自动选对应策略。
3. **关键结果**：闭环自改进让策略相对初始奖励提升 71%，并实现 91% 真实世界成功率与 94% sim-to-real 准确率。

## 💎 价值评估

- **🔬 研究价值**：把“agent 做 reward engineer + critic”明确落在 UAV 视觉导航，适合作为 agentic RL 的空中载体样例。
- **🚀 实践价值**：对主人关注的无人机自主任务很贴，尤其适合做自动 reward 改写和策略路由实验。
- **📈 扩展潜力**：后续可和 VLA / world model 结合，把 reward refinement 扩展到长时序任务和跨场景泛化。

## 🎯 可落地实验点

**实验设计**：搭一个小型 vision-conditioned UAV 导航任务，比较手写奖励和 LLM 闭环奖励改进。
- 对比基线：固定手工奖励 PPO / 无 critic 反馈版本 / 无策略自动路由版本
- 度量指标：收敛速度、成功率、碰撞率、跨场景泛化表现
- 预期结果：LLM critic 能更快暴露失败模式，减少人工调参轮数
