# 📖 花火格式笔记

## 🎯 题目

AgenticDiffusion: Agentic Diffusion-based Path Planning for Vision-Based UAV Navigation

## 📝 三句摘要

1. **问题背景**：室内无人机在有限视野下做长程导航时，单视角方法容易在遮挡、目标可见性和全局结构理解上失效。
2. **核心方法**：论文把语言引导推理、开放词汇目标定位、多视角扩散路径规划和 NMPC 串成统一管线，先选最有信息量的视角，再生成可执行轨迹。
3. **关键结果**：在 4 类真实 UAV 场景、40 次试验里任务成功率 80%，扩散规划器轨迹生成成功率 100%。

## 💎 价值评估

- **🔬 研究价值**：把多视角观测和 diffusion planner 真正落到 UAV 导航闭环里，对“看不全环境时怎么规划”这类问题很有参考价值。
- **🚀 实践价值**：开放词汇目标定位 + NMPC 的组合很适合做室内搜寻、巡检和目标接近任务原型。
- **📈 扩展潜力**：可继续接到具身语言指令、多机协同或 world model 先验，往空中具身智能方向延展。

## 🎯 可落地实验点

**实验设计**：复现“多视角选择 + 视觉扩散规划”的 UAV 导航基线，并和单视角规划器对比。
- 对比基线：单视角扩散规划 / 传统 A* + 局部避障 / 无视角切换版本
- 度量指标：任务成功率、到达时间、重复探索率、轨迹碰撞率
- 预期结果：多视角版本在遮挡和长走廊类任务上明显减少无效探索并提升成功率
