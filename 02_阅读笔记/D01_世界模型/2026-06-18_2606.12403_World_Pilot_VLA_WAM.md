# 📖 花火格式笔记

## 🎯 题目

World Pilot: Steering Vision-Language-Action Models with World-Action Priors

## 📝 三句摘要

1. **问题背景**：VLA 模型通过大规模 image-text 预训练获得语义 grounding，但 manipulation 是接触密集型动态过程，静态预训练无法捕捉物理 dynamics，导致 OOD（视角变化、几何形变、可变形物体、位姿变化）性能崩塌。
2. **核心方法**：World Pilot = VLA + World-Action Model (WAM) priors，通过两条互补路径注入：(1) Latent Steering — 用场景演化的 latent 条件化感知层；(2) Action Steering — 用预期轨迹作动作生成器的 motion prior。即使 WAM 仅用 video 预训练（未做 action post-training），scene-evolution prior 仍能工作。
3. **关键结果**：在 LIBERO-Plus zero-shot OOD 基准上达到 SOTA Total success rate 84.7%，四个真实机器人 manipulation 任务上全部最优；尤其在 viewpoint / geometry / deformable state / pose shift 四个 OOD 维度上提升最大。

## 💎 价值评估

- **🔬 研究价值**：WAM priors 是"D01 世界模型 → D02 VLA 决策"的中间接口设计范式，开创了"perception + action 双路径 steering"新架构，对 VLA + 物理 dynamics 结合是范式级突破。
- **🚀 实践价值**：主人论文 A 核心是"层次化空中 VLA"，可借鉴 WAM 双路径 steering 把世界模型的世界 dynamics 直接注入到 VLA 决策链；scene-evolution latent 对空中操作（气动扰动、光照变化）的鲁棒性是关键。
- **📈 扩展潜力**：与 WAM-RL 联合训练形成"WAM 训练 + VLA steering + RL 提升"完整链路；video 预训练 WAM 可用主人无人机真实飞行视频训练，避免真实机器人数据稀缺。

## 🎯 可落地实验点

**实验设计**：在主人论文 A 的空中 VLA 上接入 WAM priors 双路径
- 对比基线：(a) Vanilla VLA，(b) VLA + single-path latent steering only，(c) VLA + WAM-RL 联合训练（参考 2606.17906）
- 度量指标：LIBERO-Plus OOD 4 维度成功率 / 真实无人机风扰场景成功率 / 推理延迟 overhead
- 预期结果：WAM 双路径 steering 在 OOD 场景下提升 15-25%，latent steering 单独可贡献 8-12%，action steering 单独 5-8%
- 关键挑战：空中场景 video pretraining WAM 缺乏预训练数据，需用主人真实飞行日志 + 仿真视频联合训练
