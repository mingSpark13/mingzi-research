# 📖 花火格式笔记

## 🎯 题目

MemoryVLA++: Temporal Modeling via Memory and Imagination in Vision-Language-Action Models

## 📝 三句摘要

1. **问题背景**：现有 VLA 多依赖当前观测，长时程任务里既记不住历史交互，也无法稳定想象未来状态。
2. **核心方法**：作者把工作记忆、情景记忆库与隐空间世界模型结合，用记忆检索 + future imagination 生成 temporal-aware tokens，再条件化扩散动作专家。
3. **关键结果**：在 5 个仿真基准和 3 类真实机器人任务上都显著提升，真实机器人在一般/记忆依赖/想象依赖任务上分别提高 9%、26%、28%。

## 💎 价值评估

- **🔬 研究价值**：把“记忆检索 + 世界模型想象 + VLA 执行”明确拆成可组合模块，正中主人关心的长时程具身决策痛点。
- **🚀 实践价值**：给出一个能直接迁到 manipulation pipeline 的模板，尤其适合补强历史上下文缺失导致的阶段性失误。
- **📈 扩展潜力**：后续可以把 episodic memory 压缩策略、想象步长调度、失败回放检索都做成可控 ablation。

## 🎯 可落地实验点

**实验设计**：给现有 VLA/WM 控制链增加轻量历史记忆缓存和 latent future rollout，对比是否改善长程任务成功率。
- 对比基线：无记忆 VLA、仅记忆检索、仅 latent imagination
- 度量指标：长程任务成功率、关键状态恢复率、推理延迟
- 预期结果：完整记忆+想象组合在多阶段 manipulation 上最稳，且恢复失败场景能力提升明显
