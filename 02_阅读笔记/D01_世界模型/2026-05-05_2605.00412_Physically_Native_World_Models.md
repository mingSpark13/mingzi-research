# 📖 花火格式笔记

## 🎯 题目

Physically Native World Models: A Hamiltonian Perspective on Generative World Modeling

## 📝 三句摘要

1. **问题背景**：现有世界模型常把重点放在“未来画面像不像”，却没真正解决具身决策最要命的事——预测是否物理可信、动作是否可控、长时 rollout 是否稳定。
2. **核心方法**：这篇从 Hamiltonian dynamics 视角重构世界模型，把观测编码进结构化 latent phase space，再通过含控制项、耗散项和残差项的动力学演化生成未来轨迹，用于规划与控制。
3. **关键结果**：论文更像方法论纲领而非大规模 benchmark paper，但它清晰提出世界模型下一阶段的评判标准应从视觉逼真转向物理可解释、动作条件化和长程稳定，对具身智能路线很有指向性。

## 💎 价值评估

- **🔬 研究价值**：这篇直接命中 D01 方向的核心矛盾，把“世界模型是什么”重新拉回控制与物理层，而不是只做视频生成包装，适合作为后续选题的原则纸。
- **🚀 实践价值**：对明子做机器人/无人机世界模型很有启发：latent dynamics 里显式建 control 与 dissipation，可能比纯视觉生成路线更适合真实控制闭环。
- **📈 扩展潜力**：后续可以往 contact、friction、deformable objects 这些非保守系统扩，再和 MPC 或 flow-matching action model 融合，做 physically-grounded planning。

## 🎯 可落地实验点

**实验设计**：做一个简化版 action-conditioned latent world model，对比普通 RSSM/视频世界模型与带 Hamiltonian 结构约束的 latent dynamics。
- 对比基线：标准 latent world model / action-conditioned RSSM / Hamiltonian-structured latent world model
- 度量指标：长时 rollout 误差、能量漂移、规划成功率、OOD 扰动稳定性
- 预期结果：Hamiltonian 结构在长时预测稳定性和规划可用性上更强，尤其在低数据和受控动力学任务里优势明显
