# 📖 花火格式笔记

## 🎯 题目

Dream-MPC: Gradient-Based Model Predictive Control with Latent Imagination

## 📝 三句摘要

1. **问题背景**：现有基于 world model 的 MPC 常依赖 MPPI/CEM 这类梯度无关优化，控制维度一高就又慢又贵，而纯梯度式 planning 又常常不如采样法稳定。
2. **核心方法**：Dream-MPC 先用 rollout policy 生成少量候选轨迹，再在 learned world model 中对每条轨迹做梯度上升优化，同时加入不确定性正则，并复用前一时刻优化结果来摊薄实时规划开销。
3. **关键结果**：在 24 个连续控制任务上，Dream-MPC 不仅显著增强底层 policy，还能超过梯度无关 MPC 与多种强基线，说明“少量候选 + latent imagination + gradient refinement”是可行的高效规划路线。

## 💎 价值评估

- **🔬 研究价值**：它把“gradient-based planning 为什么总输给 sampling-based planning”这个老问题往前推了一步，给 D01 的 planner 设计提供了新选项。
- **🚀 实践价值**：如果主人后面要在算力受限的平台上做闭环规划，这种 warm-start + 局部梯度细化路线比大规模采样更像能真正塞进机器人控制频率里的方案。
- **📈 扩展潜力**：后续很适合和 ELVIS 一类 uncertainty-aware world model、或者动作分块/flow matching 策略头拼起来，做更稳的长时域具身控制。

## 🎯 可落地实验点

**实验设计**：在主人的世界模型控制框架中，对比 gradient-based latent MPC 与 MPPI/CEM 在高维动作任务上的效率与稳定性。
- 对比基线：标准 MPPI、CEM planner、policy-only inference、Dreamer 风格 imagined actor
- 度量指标：任务成功率、平均规划时延、单位时间回报、长时域 rollout 偏差、遮挡/扰动下恢复能力
- 预期结果：Dream-MPC 风格方法应在规划时延与任务表现之间取得更优折中，尤其适合高频闭环场景
