# 📖 花火格式笔记

## 🎯 题目

RepWAM: World Action Modeling with Representation Visual-Action Tokenizers

## 📝 三句摘要

1. **问题背景**：现有 world action model 往往直接继承面向像素重建的视频 tokenizer，视觉语义弱、视觉与动作潜空间割裂，导致“预测未来”和“推出可执行动作”之间接口松散。
2. **核心方法**：RepWAM 先构建语义对齐的 visual-action tokenizer，把视觉 token 对齐到视觉基础模型表征，再把 latent action 建成同一语义空间里的状态转移表示，随后在这套表示上联合建模未来视觉状态与连接它们的动作。
3. **关键结果**：在真实操作与仿真基准上，RepWAM 相比重建导向 WAM/VLA 基线表现更稳，在 RoboTwin 2.0 上达到 Easy 89.3、Hard 88.4，说明“先修表示层”确实能抬高世界动作模型上限。

## 💎 价值评估

- **🔬 研究价值**：它把 WAM 的瓶颈从“生成器大不大”转回“潜空间是不是语义化、动作化”，这对 D01 现在关注的世界模型—控制接口非常关键。
- **🚀 实践价值**：如果主人后面做具身世界模型落地，RepWAM 这条路能直接指导我们优先改 tokenizer / latent action 对齐，而不是一上来堆更重的视频生成器。
- **📈 扩展潜力**：这套 shared semantic visual-action space 很适合继续接 VLA、cross-embodiment action abstraction、甚至 D07 的 residual/control verifier，做成统一的可执行中间层。

## 🎯 可落地实验点

**实验设计**：在现有世界模型管线里替换掉 reconstruction-oriented tokenizer，做一个“语义视觉 token + latent action token”版本，比较它对操作成功率与动作可解释性的影响。
- 对比基线：传统视频 tokenizer + IDM、现有 DreamZero/Fast-WAM 类表示、直接 VLA policy
- 度量指标：闭环任务成功率、未来状态预测一致性、latent action 可分性、少样本真机适配效率
- 预期结果：语义化 visual-action latent 能显著提升 instruction-following dynamics 与 action grounding 的稳定性，尤其在多任务 manipulation 上更明显
