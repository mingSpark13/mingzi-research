# 📖 花火格式笔记

## 🎯 题目

EVA: Aligning Video World Models with Executable Robot Actions via Inverse Dynamics Rewards

## 📝 三句摘要

1. **问题背景**：机器人视频世界模型常能生成“看起来合理”的未来画面，但这些 imagined rollouts 往往不满足刚体与运动学约束，导致逆动力学解码出的动作不可执行。
2. **核心方法**：EVA 把 inverse dynamics model 的可执行性反馈做成训练奖励，直接回传给 video world model，在训练期而不是推理期弥合 visual realism 与 control executability 之间的 gap。
3. **关键结果**：相比只做 rejection sampling 或后验筛选的方案，EVA 更高效地提升了生成轨迹的控制可落地性，为 world model 到 action policy 的闭环对齐提供了新抓手。

## 💎 价值评估

- **🔬 研究价值**：这篇直接打中了 D01 最关键的“会想不会做”裂缝，把可执行性显式并入 world model 优化目标，研究味很正。
- **🚀 实践价值**：主人现有若有 video prediction 或 imagined rollout 管线，完全可以加一个 lightweight IDM reward 做可执行性蒸馏，而不必重写整套架构。
- **📈 扩展潜力**：它和 MoLA、PlayWorld 是天然互补关系：PlayWorld 解决数据来源，EVA 解决 rollout 可执行性，MoLA 解决动作解码接口。

## 🎯 可落地实验点

**实验设计**：在现有 world model 训练里加入“IDM reconstruction / executability reward”，对比原始视频损失训练与可执行性对齐训练的 rollout 可控性差异。
- 对比基线：纯视频重建损失 / 推理期 rejection sampling / 现有 imagined action 解码流程
- 度量指标：动作可执行率、任务成功率、轨迹稳定性、额外训练开销
- 预期结果：训练期对齐可比后处理更省采样成本，并显著降低 rollout-to-action 失配
