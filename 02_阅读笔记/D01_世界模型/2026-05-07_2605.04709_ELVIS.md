# 📖 花火格式笔记

## 🎯 题目

ELVIS: Ensemble-Calibrated Latent Imagination for Long-Horizon Visual MPC

## 📝 三句摘要

1. **问题背景**：视觉模型式强化学习在长时域规划时很容易因为 latent dynamics 分叉、多峰价值分布和遮挡累积误差而想象崩坏，导致 visual MPC 不敢往深处看。
2. **核心方法**：ELVIS 在 Dreamer 风格 RSSM 上做 latent MPC，把标准 MPPI 换成能保留多模态假设的 Gaussian-mixture MPPI，并用 ensemble critic 给出的 uncertainty-aware lambda-return 同时服务 imagined actor-critic 训练和规划打分。
3. **关键结果**：这套“多假设规划 + 不确定性感知回报”把长时域 latent imagination 变得更稳，目标正是让视觉世界模型第一次真正能撑住实用级长程规划。

## 💎 价值评估

- **🔬 研究价值**：它不只是再堆一个 world model，而是精准打在“长时域 latent planning 为什么不可靠”这个老痛点上，对 D01 很有启发性。
- **🚀 实践价值**：主人若要做无人机或机器人在遮挡、分叉路径环境中的世界模型规划，ELVIS 这种 uncertainty-gated imagination 很适合拿来防止 rollout 幻觉把控制器带沟里。
- **📈 扩展潜力**：后续可以把这套 return 设计接到 action chunking、flow matching 或 packet supervisor 一类下游控制接口上，形成更稳的长程闭环。

## 🎯 可落地实验点

**实验设计**：在主人现有 D01 世界模型框架里加入 ensemble critic + GMM-MPPI，对比长时域规划稳定性
- 对比基线：Dreamer-style RSSM + 标准 MPPI、单峰 critic、固定 lambda-return
- 度量指标：长时域任务成功率、rollout 校准误差、遮挡场景恢复率、规划频率、下游控制抖动
- 预期结果：ELVIS 风格的不确定性感知规划应在分叉未来与部分可观测场景下显著优于单峰 latent MPC
