# 📖 花火格式笔记

## 🎯 题目

HarmoWAM: Harmonizing Generalizable and Precise Manipulation via Adaptive World Action Models

## 📝 三句摘要

1. **问题背景**：现有 world action model 往往在“泛化强但交互不精”与“交互精但受训练分布限制”之间二选一，难以同时兼顾真实机器人零样本泛化与精细操作。
2. **核心方法**：论文提出 HarmoWAM，用世界模型提供时空物理先验，同时驱动 predictive expert 与 reactive expert 两个动作专家，并通过过程自适应门控机制在任务不同阶段自动切换。
3. **关键结果**：在 6 个真实机器人任务、3 类训练未见环境变化下，HarmoWAM 的零样本泛化显著超过已有 VLA 与 WAM，文中报告相对最强基线分别提升 33% 与 29%。

## 💎 价值评估

- **🔬 研究价值**：这篇直接命中 D01 主线里“世界模型不该只会想象，还要真正服务动作生成与部署泛化”的关键矛盾，把 predictive/reactive 两类 WAM 路线统一起来很有代表性。
- **🚀 实践价值**：对主人后续做 deployment-time supervisor 很有启发，尤其是“世界模型提供物理先验 + 不同阶段调用不同动作专家”的结构，适合迁到空中操作或受限平台控制。
- **📈 扩展潜力**：后面如果把它的 gating 逻辑与风险预算、失败分流、remaining-plan packet 结合，能自然延展到 D01 关心的阶段感知监督与执行前筛查。

## 🎯 可落地实验点

**实验设计**：在主人现有 D01 世界模型管线里复现“predictive expert + reactive expert + adaptive gating”三件套，验证它是否比单一路线更适合处理执行前筛查与局部修正。
- 对比基线：纯 predictive WAM、纯 reactive WAM、无 gating 的双专家并联
- 度量指标：训练未见场景成功率、局部交互精度、动作一致性、失败后恢复成功率
- 预期结果：双专家 + 自适应门控应当同时保住 transit 泛化与接触阶段精度，尤其在场景变化和物体位置扰动下更稳
