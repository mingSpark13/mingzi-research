## 🎯 题目

AeroDGS

# AeroDGS: Physically Consistent Dynamic Gaussian Splatting for Single-Sequence Aerial 4D Reconstruction

## 📝 三句摘要

1. 提出 **AeroDGS**，首个针对单目无人机视频的物理约束 4D Gaussian Splatting 框架，解决空中重建中深度模糊和运动估计不稳定的核心难题。
2. 核心方法是 **Monocular Geometry Lifting 模块**：从单目空中序列中重建可靠的静态和动态几何，为动态估计提供稳健基础。
3. **Physics-Guided Optimization 模块**：解析动态物体姿态歧义，确保物理一致的 4D 重建结果。

## 核心贡献

| 贡献 | 内容 |
|------|------|
| **问题定义** | 单目无人机 4D 重建天然是 ill-posed 问题：深度模糊 + 运动估计不稳定 |
| **Monocular Geometry Lifting** | 从单目视频中 lifting 出可靠几何（静态背景 + 动态前景） |
| **Physics-Guided Optimization** | 利用物理先验（城市场景不是任意的）解析动态物体姿态歧义 |
| **物理一致性** | 重建结果满足物理规律（刚体运动、时间连续性等） |

## 方法细节

### 核心难题
- 单目空中影像的深度线索天然模糊（depth ambiguity）
- 动态场景的运动估计不稳定
- 两者耦合导致单目无人机 4D 重建本质上是个 ill-posed 问题

### 解决思路
> "城市场景不是任意的" — 虽然单目影像的深度线索模糊，但城市环境有物理规律约束

1. **Monocular Geometry Lifting 模块**
   - 输入：单目无人机视频序列
   - 输出：可靠的静态几何 + 动态物体几何
   - 关键：利用空中平台的特殊约束（平稳飞行、正交场景结构）辅助几何恢复

2. **Physics-Guided Optimization**
   - 利用物理先验约束优化 Gaussian 的位置/旋转/缩放
   - 解析动态物体的姿态歧义
   - 保证重建结果的物理一致性（刚体、连续性等）

### 技术路线
```
单目无人机视频
    ↓
Monocular Geometry Lifting（几何 lifting）
    ↓
静态背景建模 + 动态前景建模
    ↓
Physics-Guided Optimization（物理约束优化）
    ↓
物理一致 4D 重建结果
```

## 与主人研究的关联

### ✅ 高度相关方向
- **无人机数据采集 + 4D 场景重建**：AirSim UE 场景可以被 AeroDGS 这类方法用于事后场景重建和数据增强
- **实习 3DGS 方向**：AeroDGS 是 4DGS 在无人机场景的具体应用，与主人实习 3DGS 方向完全对齐
- **数据飞轮（D05）**：4D 重建可以作为仿真数据生成的场景来源

### 💡 可能的结合点
1. **AirSim 仿真数据 → AeroDGS 重建 → 训练数据增强**
   - 用 AirSim 采集无人机视频 → AeroDGS 做 4D 重建 → 生成更多视角的仿真数据
2. **真实无人机视频 → AeroDGS 场景理解 → 世界模型输入**
   - 4DGS 重建的场景可以直接作为 VLA/世界模型的输入
3. **单目深度估计增强**
   - AeroDGS 的 Monocular Geometry Lifting 可以帮助提升单目深度估计精度

## 🎯 可落地实验点

1. **场景重建质量对比**：在 AirSim 环境中用 AeroDGS 重建 4D 场景 vs 传统单目重建方法，评估重建精度
2. **数据增强验证**：用 AeroDGS 重建结果生成新视角数据，训练跟随模型，评估数据增强收益
3. **深度估计提升**：将 AeroDGS 的几何 lifting 模块作为深度估计网络的辅助监督
