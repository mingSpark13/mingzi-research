# 📖 花火格式笔记

## 🎯 题目

World Model for Robot Learning: A Comprehensive Survey

## 📝 三句摘要

1. **问题背景**：机器人世界模型近两年同时向策略学习、可控视频生成、仿真评测和数据生成四条线快速扩张，但文献分散，缺少面向机器人学习的统一整理。
2. **核心方法**：论文按“world model 如何耦合 policy、如何充当 learned simulator、如何演化成 robotic video world model”三条主线，系统梳理架构、应用域、数据集、基准和评测协议。
3. **关键结果**：这篇综述把 reactive VLA 的局限、world model 的增益来源，以及从 imagination-based 生成走向 controllable foundation-scale world model 的趋势讲清楚了，适合作为 D01 方向总览锚点。

## 💎 价值评估

- **🔬 研究价值**：它不是泛泛综述，而是把世界模型在 policy learning / planning / simulation / evaluation / data generation 中的角色拆得很清，对主人后续梳理 D01 技术路线很有用。
- **🚀 实践价值**：可以直接拿来反查基线、数据集和 benchmark，帮助后续筛选该补哪类世界模型论文、哪些任务最适合做实验切口。
- **📈 扩展潜力**：适合作为 D01 知识树入口页，再向视频世界模型、动作条件预测、VLA-world-model 接口设计分叉深挖。

## 🎯 可落地实验点

**实验设计**：从综述中挑 2 类 world model pipeline，做“显式 world model + planner”与“纯 reactive VLA”在长程操作任务上的对比复现。
- 对比基线：reactive VLA、video world model + inverse dynamics、latent world model + planner
- 度量指标：长程任务成功率、rollout error accumulation、规划深度收益、数据效率
- 预期结果：显式 world model 在长时序任务和误差可分析性上更强，但动作落地质量受接口设计限制
