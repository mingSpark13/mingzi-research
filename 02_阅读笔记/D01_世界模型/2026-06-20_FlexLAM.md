# 📖 花火格式笔记

## 🎯 题目

**FlexLAM: Resolving the Bottleneck Trade-off in Latent Action Learning**

## 📝 三句摘要

1. **问题背景**：Latent Action Model（LAM）通过无标签视频自监督学习动作表征，但现有方法存在"表征容量 vs 物理一致性"瓶颈——要么 LAM 学到过于抽象的语义无法预测物理细节，要么太精细失去跨任务迁移能力。
2. **核心方法**：FlexLAM 通过灵活的 latent action 表征框架，引入"瓶颈解耦"机制，把"高层语义动作"和"底层物理动作"分离建模，兼顾跨任务泛化与下游物理预测精度。
3. **关键结果**：在多个 VLA 下游基准（LIBERO / RoboSuite / 真实机器人）上同时取得高层语义和底层物理的最佳平衡，跨本体迁移能力优于 LAPA / IGOR 等同类方法。

## 💎 价值评估

- **🔬 研究价值**：直接命中 D01 当前"latent action 学习"创新口，是 LA 方向的最新进展；为 LAM 设计提供了"分层瓶颈"的工程范式。
- **🚀 实践价值**：分层 latent action 可直接迁移到主人 D01 项目中作为预训练目标，特别是在 D04 跨载体场景下，"高层语义动作"对形态变化更鲁棒。
- **📈 扩展潜力**：①与 D06 空中VLN 结合，看分层 LA 是否对"导航语义动作"和"飞行控制动作"自然解耦；②与 D05 数据飞轮结合，作为大规模无标签无人机视频的自监督预训练目标。

## 🎯 可落地实验点

**实验设计**：在主人已有 D01 latent action 实验基础上，复现 FlexLAM 的"分层瓶颈"机制，对比单层 LAM 在以下场景：①同本体跨任务迁移 ②跨本体（D04）迁移 ③真实无人机视频自监督预训练 → D06 空中VLN 下游任务迁移。

- 对比基线：单层 LAPA / IGOR / LAPA-finetune
- 度量指标：跨任务 / 跨本体成功率、预训练数据效率
- 预期结果：分层 LA 在跨本体场景应比单层 LAM 提升 10-15% 成功率
