# 📖 花火格式笔记

## 🎯 题目

Learning Actionable Manipulation Recovery via Counterfactual Failure Synthesis

## 📝 三句摘要

1. **问题背景**：现有机器人失败恢复大多依赖高风险真机采集或仿真扰动，且常只给粗粒度失败诊断，难以直接生成可执行修正动作。
2. **核心方法**：Dream2Fix 直接在生成式世界模型里对成功演示做动作扰动，合成逼真的反事实失败轨迹，再用任务有效性、视觉连贯性和运动学安全三重校验筛出高质量“失败-修正”配对数据，随后微调 VLM 同时预测失败类型与恢复轨迹。
3. **关键结果**：该方法构建了 12 万+ 高保真配对样本，在真实机器人实验中把纠错准确率从 19.7% 提升到 81.3%，并实现零样本闭环失败恢复。

## 💎 价值评估

- **🔬 研究价值**：它把 D01 当前的“已知失败 vs 未知异常”分诊进一步推进成“已知失败如何恢复”的可执行路线，证明生成式世界模型不只会评估和筛选，还能直接产出恢复 supervision。
- **🚀 实践价值**：对空中操作尤其有价值，因为真机故障样本昂贵且危险，Dream2Fix 这种从成功轨迹反推失败样本的方式，很适合先在仿真或离线回放里补齐 recovery 数据。
- **📈 扩展潜力**：后续可把它和 D01 的 verifier / anomaly triage 结合，形成 `分诊 → 恢复建议 → 回退策略` 三层闭环，而不是只做 pass/fail 过滤。

## 🎯 可落地实验点

**实验设计**：在主人后续的空中抓取或接近-对位任务里，先收集少量成功轨迹，再用已有世界模型对动作做受约束扰动，合成“接近位姿偏移 / 局部碰撞 / 接触失败 / 退出路径错误”等反事实失败片段，训练轻量恢复头预测修正轨迹。
- 对比基线：仅做 F1/F2 分诊；分诊 + 手工 recovery rules；分诊 + Dream2Fix 式反事实恢复器
- 度量指标：恢复成功率、异常漏放率、恢复后额外碰撞率、恢复时延
- 预期结果：反事实恢复器在已知失败场景显著优于纯规则回退，同时不明显增加异常场景的误恢复风险
