# 📖 花火格式笔记

## 🎯 题目

World Action Verifier: Self-Improving World Models via Forward-Inverse Asymmetry

## 📝 三句摘要

1. **问题背景**：通用世界模型需要在大量次优甚至随机动作下依然可靠，但真实机器人动作标注数据稀缺，导致 under-explored 区域最难校验。
2. **核心方法**：作者提出 World Action Verifier，用“视频语料子目标生成器 + 稀疏逆动力学模型 + 前向世界模型”构成 forward-inverse 非对称校验环，把 state plausibility 与 action reachability 分开验证。
3. **关键结果**：在 MiniGrid、RoboMimic、ManiSkill 共 9 个任务上，WAV 让世界模型样本效率提升约 2×，并把下游策略性能再抬高 18%。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯把世界模型做更大，而是把“世界模型什么时候值得信”显式化，正中 D01 现在的可验证性与 consume-honesty 研究口。
- **🚀 实践价值**：它把动作标注稀缺问题转成“用无动作视频验证 plausibility、用低维特征验证 reachability”，很适合低成本扩充机器人世界模型数据闭环。
- **📈 扩展潜力**：可以直接启发 D01 的 packet/address/consume 线路：先验可行状态、可执行动作、前向 rollout 三者做一致性审计，比只看最终预测误差更稳。

## 🎯 可落地实验点

**实验设计**：给 D01 的 delayed-consumption world model 增加一层 WAV 式 verifier。
- **对比基线**：原始 D01 world model；仅前向误差筛样本；加入 verifier 的 world model
- **度量指标**：sample efficiency、consume-time honesty、identity preservation、下游任务成功率
- **预期结果**：verifier 版本应更早发现“看起来合理但动作不可达”的伪正例，并降低 delayed-consumption 场景中的 referent drift
