# 📖 花火格式笔记

## 🎯 题目

Hierarchical Latent Action Model

## 📝 三句摘要

1. **问题背景**：现有 latent action models 多只建模短时帧间过渡，能抓低层运动，却难以发现无动作视频里的长时程高层技能结构。
2. **核心方法**：HiLAM 以预训练 LAM 作为低层 latent action 提取器，再把动作序列聚合成高层 latent skills，从而进行层次化的长期时序建模。
3. **关键结果**：论文表明 HiLAM 相比基础 LAM 能更稳地发现动态技能模式，在 actionless video 到交互世界模型/控制的桥接上更强。

## 💎 价值评估

- **🔬 研究价值**：它补的是“latent action 只有短视野”这个关键缺口，对世界模型从短期预测走向长期 skill abstraction 很重要。
- **🚀 实践价值**：对主人做长时域机器人策略、skill discovery、以及无标注视频预训练都有直接启发，尤其适合和 VLA/world model 融合。
- **📈 扩展潜力**：后续可把层次 latent skill 接到动作分块、goal-conditioned planning、跨 embodiment skill transfer 上，形成更强的统一策略先验。

## 🎯 可落地实验点

**实验设计**：在主人现有 world model / policy pipeline 中加入“高层 latent skill token”中间层
- 对比基线：普通 LAM / 直接 action chunk policy
- 度量指标：长时域任务成功率、skill 可分性、少样本适配效率
- 预期结果：层次 latent skill 能提升长期任务稳定性，并降低对精确动作标签的依赖
