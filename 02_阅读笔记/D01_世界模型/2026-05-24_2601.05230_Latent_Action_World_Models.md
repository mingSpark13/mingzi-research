# 📖 花火格式笔记

## 🎯 题目

Learning Latent Action World Models In The Wild

## 📝 三句摘要

1. **问题背景**：传统 world model 往往依赖显式动作标注，难以直接吃下海量 in-the-wild 视频，也很难跨不同 embodiment 学到可复用的动作结构。
2. **核心方法**：论文从无动作标签视频中学习连续且受约束的 latent action，并结合世界模型与控制器把已知动作映射到 latent action，形成可规划的通用动作接口。
3. **关键结果**：作者证明 latent action 可在复杂真实视频中捕捉由 agent 引发的环境变化，规划性能接近 action-conditioned baseline，但 learned action 多呈相机坐标系局部化，揭示了“野外视频学动作”的边界与可行性。

## 💎 价值评估

- **🔬 研究价值**：这篇直接打到 D01 很核心的“没有动作标签还能不能学 world model”问题，而且不是玩具仿真而是 in-the-wild 视频，属于把 latent action 路线往真实世界推进的重要一步。
- **🚀 实践价值**：对主人后续如果想利用公开视频、无人机录像或互联网机器人视频扩充 pretraining pool 很有启发，尤其是把 latent action 当作跨数据源统一接口这一点。
- **📈 扩展潜力**：后续可把这套 latent interface 与 embodiment-aware token、物理一致性约束或下游 planner 结合，继续逼近真正可迁移的世界模型控制前端。

## 🎯 可落地实验点

**实验设计**：在现有 D01 管线中做“有动作标签 vs 无动作标签 latent action pretraining”对比，评估 latent action 是否能作为跨数据源统一控制接口。
- 对比基线：action-conditioned world model、VAE-style latent world model、无约束离散 latent action
- 度量指标：未来预测误差、下游规划成功率、跨数据源迁移表现、latent action 与真实控制量的可对齐性
- 预期结果：连续受约束 latent action 会比离散量化 latent 更稳，更适合作为野外视频到机器人控制之间的桥梁
