# 📖 花火格式笔记

## 🎯 题目

AirDreamer: Generalist Drone Navigation with World Models

## 📝 三句摘要

1. **问题背景**：无人机在陌生拥挤环境中导航，传统方法强依赖手工感知管线和环境特定规则，跨场景泛化很差。
2. **核心方法**：论文把基于世界模型的环境理解和强化学习导航策略结合起来，并设计无手工 shaping 的稀疏奖励来鼓励有效偏航与脱困行为。
3. **关键结果**：在仿真与真实无人机上都表现出较强泛化，困难地图上比最强基线高 5.3% 成功率，且实现了无需部署调参的 sim-to-real 迁移。

## 💎 价值评估

- **🔬 研究价值**：把世界模型从机械臂/视频想象进一步推进到无人机导航，正中明子关心的 UAV + world model 交叉点。
- **🚀 实践价值**：对陌生环境飞行、复杂避障和真实部署很有参考意义，尤其适合作为轻地图依赖导航基线。
- **📈 扩展潜力**：后续可继续接多模态目标指令、长程任务规划和空地协同导航，适合纳入 D01→D06 的桥接脉络。

## 🎯 可落地实验点

**实验设计**：复现“世界模型表征 + 稀疏奖励 RL”在无人机单目导航中的收益
- 对比基线：纯端到端 RL、传统手工感知+规划、去掉 world model 的 AirDreamer 退化版
- 度量指标：成功率、碰撞率、平均航程、部署后零调参成功率
- 预期结果：world model 提供更稳的结构表征，在陌生复杂地图中显著降低局部最优和碰撞
