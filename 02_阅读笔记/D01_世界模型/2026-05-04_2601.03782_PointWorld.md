# 📖 花火格式笔记

## 🎯 题目

PointWorld: Scaling 3D World Models for In-The-Wild Robotic Manipulation

## 📝 三句摘要

1. **问题背景**：机器人操作想要像人一样“看一眼就预判动作后果”，需要能把视觉状态和控制动作统一起来的世界模型，而传统关节空间表征很难跨载体泛化。
2. **核心方法**：论文提出 PointWorld，把机器人动作表示成 3D point flow，在共享 3D 空间里联合建模 RGB-D 观测与低层动作，并用真实+仿真共约 200 万条轨迹做大规模预训练。
3. **关键结果**：PointWorld 可实时推理并直接接入 MPC，在单臂 Franka 与双臂 humanoid 混合数据上学到可迁移的 3D 动态预测能力，真实机器人无需额外示教或后训练就能完成 pushing、deformable/articulated object manipulation 和 tool use。

## 💎 价值评估

- **🔬 研究价值**：这篇把“世界模型的动作接口”从 embodiment-specific 关节空间抬升到几何 point flow，等于把 D01 世界模型和 D04 跨载体泛化硬生生接上了，味道很正。
- **🚀 实践价值**：对主人后面做跨平台具身预测、空地共用策略先验、甚至 UAV+机械臂一体控制，都有很直接的参考——先统一几何作用，再谈具体执行器。
- **📈 扩展潜力**：可沿着 point flow → latent action / PMI 接口继续做，把不同机器人动作抽象成共享可验证的“对世界施加的几何变化”。

## 🎯 可落地实验点

**实验设计**：在主人现有机械臂或仿真操作任务中，把 action 表征从 joint delta 改成末端/接触点 3D flow，比较世界模型跨任务迁移能力。
- 对比基线：关节空间 action-conditioned world model；像素/latent 视频预测模型；不接 MPC 的纯 policy rollout
- 度量指标：多步预测误差、MPC 成功率、跨平台迁移成功率、真实部署是否需要额外微调
- 预期结果：3D flow 表征在跨 embodiment 与接触丰富任务里更稳，MPC 规划更容易利用预测结果
