# 📖 花火格式笔记

## 🎯 题目

H-WM: Robotic Task and Motion Planning Guided by Hierarchical World Model

## 📝 三句摘要

1. **问题背景**：传统任务与运动规划擅长长时程逻辑推理，但与视觉感知脱节，难以为复杂机器人任务提供统一中间表示。
2. **核心方法**：论文提出 H-WM，用分层世界模型联合预测逻辑状态转移与视觉状态转移，把符号规划的可执行性与感知预测结合起来。
3. **关键结果**：方法面向复杂长时程机器人任务，为 VLA 类策略提供中间指导信号，价值在于把“规划层”与“感知层”真正接上。

## 💎 价值评估

- **🔬 研究价值**：对“世界模型+长时程规划+感知 grounding”这条线很关键，适合主人关注的 embodied/规划融合方向。
- **🚀 实践价值**：可作为无人机/机器人任务规划中的中间层设计参考，尤其适合需要显式逻辑约束的场景。
- **📈 扩展潜力**：后续可结合 VLA、扩散策略或 flow matching 动作建模，做分层规划-控制闭环。

## 🎯 可落地实验点

**实验设计**：做“高层符号子目标 + 低层视觉策略”的两层系统原型
- 对比基线：纯端到端 VLA / 纯符号 TAMP
- 度量指标：任务成功率、长时程步骤完成率、重规划次数
- 预期结果：分层世界模型在长时程任务上更稳，尤其在中间状态易漂移时更有优势
