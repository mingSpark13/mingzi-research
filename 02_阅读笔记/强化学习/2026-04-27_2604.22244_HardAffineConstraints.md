# 📖 花火格式笔记

## 🎯 题目

Learning Control Policies to Provably Satisfy Hard Affine Constraints for Black-Box Hybrid Dynamical Systems

## 📝 三句摘要

1. **问题背景**：黑盒混合动力系统里既有未知非线性动力学，又有碰撞/重置导致的状态跳变，传统 safe RL 往往只能软惩罚违规，难以给出硬约束安全保证。
2. **核心方法**：论文把策略限制成在约束边界附近具有排斥性的仿射策略，并额外为 reset 前区域构造第二层排斥结构，从而在未知动力学和仿射 reset map 下给出闭环不越界的充分条件。
3. **关键结果**：在 constrained pendulum 和 paddle juggler 等混合系统上，该方法比 reward shaping 与 learned-CBF 基线学到更高质量策略，同时始终满足安全约束。

## 💎 价值评估

- **🔬 研究价值**：它不是泛泛讲 safe RL，而是专盯“黑盒混合系统 + 硬约束”这个很刁钻的控制问题，理论条件与实验验证都比较完整。
- **🚀 实践价值**：对带接触/碰撞/模式切换的机器人控制很有参考意义，尤其适合需要“绝不能越界”的安全场景，而不只是平均意义上少犯错。
- **📈 扩展潜力**：后续可继续看这种结构化安全策略能否迁到腿足接触控制、空中操作碰撞恢复、或 IsaacSim 中的安全训练框架。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 机械臂接触任务中加入“边界排斥仿射安全层”，对比普通 reward shaping safe RL 的越界率与收敛速度。
- 对比基线：reward shaping、action clipping、learned CBF safety shield
- 度量指标：约束违规次数、成功率、收敛步数、接触任务完成质量
- 预期结果：结构化安全层在强接触/重置场景下违规率更低，且比纯惩罚式方法稳定
