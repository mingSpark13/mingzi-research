# 📖 花火格式笔记

## 🎯 题目

Unified Noise Steering for Efficient Human-Guided VLA Adaptation

## 📝 三句摘要

1. **问题背景**：扩散式 VLA 在真机适配时很强，但 on-robot RL 成本高，纯 noise-space RL 又常被探索低效拖慢。
2. **核心方法**：论文提出 UniSteer，把人类在 action space 给出的纠正动作，经 frozen flow-matching decoder 近似反演成 noise target，再和 noise-space RL 共享同一个噪声 actor 联合优化。
3. **关键结果**：在四个真实操作适配任务上，平均 66 分钟就把成功率从 20% 拉到 90%，比纯 noise-space RL 和 action-space human-in-the-loop baseline 都更高效。

## 💎 价值评估

- **🔬 研究价值**：这篇把“人类纠偏”与“噪声空间策略优化”真正打通了，不再是两条割裂监督流，对扩散/VLA 适配很关键。
- **🚀 实践价值**：主人后面做 VLA、机械臂、空中操作自适应时都能直接借：真机少量交互 + 低频人工纠偏，比全量 teleop 微调便宜很多。
- **📈 扩展潜力**：如果把 human corrective action 换成 agent guidance、规则控制器甚至 MPC teacher，就能往“自动纠偏版 UniSteer”继续推。

## 🎯 可落地实验点

**实验设计**：在机械臂或 UAV-manipulation 任务上，给预训练 flow-matching policy 加入“动作纠偏→噪声目标”的近似反演模块，比较适配效率。
- 对比基线：纯 noise-space RL、LoRA/全参 VLA 微调、直接 action cloning 的 human-in-the-loop adaptation
- 度量指标：成功率提升速度、真机交互步数、纠偏次数、部署稳定性
- 预期结果：UniSteer 风格的统一噪声监督会显著提升少样本适配效率，并降低人工纠偏总量
