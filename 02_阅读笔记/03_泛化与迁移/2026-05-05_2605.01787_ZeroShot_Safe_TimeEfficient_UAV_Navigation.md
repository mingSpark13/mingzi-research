# 📖 花火格式笔记

## 🎯 题目

Zero-Shot, Safe and Time-Efficient UAV Navigation via Potential-Based Reward Shaping, Control Lyapunov and Barrier Functions

## 📝 三句摘要

1. **问题背景**：现有 UAV 强化学习导航常常把“能到达”放在第一位，却牺牲了任务时间和形式化安全保证，导致从简单训练环境迁到复杂场景时不够稳。
2. **核心方法**：论文把 Potential-Based Reward Shaping 与 CLF/CBF 约束结合起来，先在简化环境训练 RL 策略，再在复杂场景部署时叠加 CLF-CBF-QP safety filter，无需再训练。
3. **关键结果**：实验表明该方法在复杂环境里能显著缩短任务时间，同时保持更强避障与安全约束满足能力，属于“策略负责效率，安全壳负责托底”的清晰路线。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯再做一个 UAV RL baseline，而是把 reward shaping、稳定性约束和 barrier safety 组合成一套可解释部署框架，正中 D03/D06 交界处“训练策略怎么安全落地”的缺口。
- **🚀 实践价值**：对明子的无人机任务很有参考性，后续无论做低空导航、人体跟随还是障碍密集场景穿越，都可以把安全过滤器当成部署层保险丝。
- **📈 扩展潜力**：可以继续接入语言导航或视觉策略前端，让 VLA / VLN 负责高层意图，CLF-CBF-QP 负责最后一层安全执行，形成更完整的空中智能闭环。

## 🎯 可落地实验点

**实验设计**：在 AirSim / Isaac Sim 中复现“简化环境训练 + 复杂环境零样本部署 + CLF/CBF 安全过滤”的导航框架。
- 对比基线：纯 RL 导航、带 PBRS 的 RL、带 PBRS + CLF-CBF-QP 的完整方法
- 度量指标：任务成功率、到达时间、碰撞率、安全约束违反次数、零样本迁移性能
- 预期结果：完整方法在复杂场景里会比纯 RL 更稳，尤其在障碍密集和路径约束明显时优势更大
