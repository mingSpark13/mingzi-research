# 📖 花火格式笔记

## 🎯 题目

GaussFly: Contrastive Reinforcement Learning for Visuomotor Policies in 3D Gaussian Fields

## 📝 三句摘要

1. **问题背景**：单目 RGB 无人机策略学习虽然传感器轻量，但直接从高维图像到动作的端到端学习样本效率低，而且仿真与真实场景的纹理、光照和噪声差异会造成严重 sim-to-real gap。
2. **核心方法**：GaussFly 提出一个 **real-to-sim-to-real** 闭环，先用带显式几何约束的 **3D Gaussian Splatting** 重建真实训练场景，再基于重建场景渲染多视角 RGB 图像，用 **contrastive representation learning** 预训练视觉编码器，最后冻结编码器并将低维特征送入 RL policy 学控制。
3. **关键结果**：论文声称在仿真和真实环境中都优于 baseline，既提高了样本效率和最终性能，又能在复杂纹理的未见真实场景中实现 **zero-shot transfer**。

## 💎 价值评估

- **🔬 研究价值**：这篇不是经典“规划避障”论文，而是把 **3DGS 场景表示 + 对比表征学习 + UAV visuomotor RL + sim2real** 串成一个完整闭环，明确回答了“如何让单目无人机不直接吃 raw RGB 也能学到可迁移策略”的问题。
- **🚀 实践价值**：对主人当前 AirSpark / 3DGS / 无人机策略训练很有参考价值，尤其是“真实场景先重建成高保真可渲染中间层，再在其中训练策略”的路线，可直接启发离线地图驱动的训练管线。
- **📈 扩展潜力**：后续可以继续往两边扩展，一边接入更强的几何/动力学先验，另一边接入世界模型或 VLA 高层决策，形成“高保真场景表示 + 策略学习 + 长程任务”的统一栈。

## 🎯 可落地实验点

**实验设计**：验证“直接 RGB 端到端 RL”与“3DGS 场景 + 对比表征 + RL”在无人机 sim2real 上的差异
- 对比基线：raw RGB end-to-end RL / depth 或显式中间表征 / GaussFly 风格的 frozen encoder + RL
- 度量指标：样本效率、真实场景成功率、zero-shot 泛化、对纹理变化和光照变化的鲁棒性
- 预期结果：3DGS + 对比表征路线在真实迁移和复杂纹理场景上更稳，训练更省样本
