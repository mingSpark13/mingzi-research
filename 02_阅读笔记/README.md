# 阅读笔记 📝

## 模板

新建论文笔记时复制 `_TEMPLATE.md`。

## 必须章节

1. YAML frontmatter（完整字段）
2. 🎯 题目
3. 📝 三句摘要（问题/方法/结果）
4. 💎 价值评估（列表格式，不用表格）
5. 🎯 可落地实验点（≥1个）
6. 🔗 知识图谱（标准概念 wikilink，≥3个，指向当前知识 Wiki 概念页）
7. 🔗 相关链接（论文 wikilink，≥1个）
8. 📌 待探索问题（≥2个）

## 放置规则

- 这里只放**论文阅读笔记**。
- B站视频解读、工具介绍、面试题、主人自己的讨论纪要，不放这里。
- 若内容不是对一篇正式论文/预印本的阅读总结，应移动到更合适的目录（如 `40_工作开发/` 或 `10_与花火的对话/`）。

## 知识图谱规则

- `## 🔗 知识图谱` 只允许使用**标准概念名**。

## 命名

```
YYYY-MM-DD_标题.md
```

## 链接

```markdown
[[YYYY-MM-DD_标题]]    ← 论文互引
[[VLA架构]]           ← 引用知识图谱概念（直接用概念页文件名）
```
