# 📖 花火格式笔记

## 🎯 题目

Safe Whole-Body Loco-Manipulation via Combined Model and Learning-based Control

## 📝 三句摘要

1. **问题背景**：腿足机器人在移动中做接触式操作时，很难同时兼顾全身协调、接触顺应性和安全约束。
2. **核心方法**：论文把机械臂的模型式导纳控制和四足底盘的强化学习策略结合起来，再用 Reference Governor 做安全约束、用神经增强卡尔曼滤波估计底座速度。
3. **关键结果**：方法在 Unitree Go2+6DoF 机械臂的仿真与实机上都实现了稳定的交互驱动速度跟踪和安全顺应控制，说明 model-based + learning-based 混合方案适合腿足操作一体化任务。

## 💎 价值评估

- **🔬 研究价值**：它不是单纯跑步或单纯操作，而是把 loco-manipulation 中最难的“接触安全 + 全身协调”放到统一控制栈里，适合补齐腿足机器人从导航走向任务执行的关键一环。
- **🚀 实践价值**：对需要推门、扶物、牵引、辅助搬运这类真实接触任务很有参考性，尤其适合 Unitree 这类低成本腿足平台快速验证。
- **📈 扩展潜力**：后续可往视觉条件、长程任务规划、多接触切换和人机协作安全边界继续扩展，也能与 D04 跨载体泛化里的统一动作接口思路做对接。

## 🎯 可落地实验点

**实验设计**：在 Go2 或仿真四足平台上复现“移动中末端受力引导”的 whole-body 控制任务，比较纯模型控制、纯 RL、混合控制三种方案。
- 对比基线：纯导纳+逆解、纯 RL locomotion-manipulation policy、本文 hybrid controller
- 度量指标：末端速度跟踪误差、接触力超限率、任务成功率、底盘姿态稳定性
- 预期结果：混合方案在安全性和跟踪性能上优于纯 RL，在动态场景鲁棒性上优于纯模型法
