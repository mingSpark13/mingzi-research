# 📖 花火格式笔记

## 🎯 题目

MeMix: Training-Free Plug-and-Play Module for Streaming 3D Reconstruction

## 📝 三句摘要

1. **问题背景**：从单目图像序列实时重建高质量几何和外观仍是挑战——现有流式重建方法面临灾难性遗忘（后续帧覆盖前面关键信息）和内存随帧数线性增长的问题。
2. **核心方法**：MeMix 将 recurrent state 重构为 Memory Mixture，分区为多个独立 memory patches，仅更新最低对齐度的 patches，同时精确保留其他 patches，实现选择性更新。
3. **关键结果**：O(1) 推理内存（不随帧数增长）、无需微调或额外可学习参数，在 ScanNet、7-Scenes、KITTI 等基准上降低重建完整性误差 15.3%（最高 40.0%）。

## 💎 价值评估

- **研究价值**：★★★☆☆ — 训练-free 即插即用设计有工程价值，但方法本身创新性有限
- **实践价值**：★★★★☆ — 对 UE 仿真数据采集、无人机长序列场景重建有直接应用价值
- **扩展潜力**：可与 3DGS 类方法结合用于实时场景建图

## 🎯 可落地实验点

**实验设计**：在 UE 仿真器中验证 MeMix 对长序列城市场景重建的内存稳定性
- 对比基线：原始 3DGS（内存随帧数 O(N) 增长）
- 度量指标：重建完整性（Completion）、精度（Accuracy）、峰值内存占用
- 预期结果：MeMix 在 500+ 帧序列保持 O(1) 内存，重建质量与原始 3DGS 持平
