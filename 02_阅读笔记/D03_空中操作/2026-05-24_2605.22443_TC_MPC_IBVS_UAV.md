# 📖 花火格式笔记

## 🎯 题目

Terminal Constraint Model Predictive Control for Image-Based Visual Servoing of UAVs with Kalman Filter-Based Moment Loss Compensation

## 📝 三句摘要

1. **问题背景**：传统 UAV 图像视觉伺服在接近目标时常因状态/输入约束破坏闭环稳定性，激烈机动时视觉矩特征丢失又会直接导致控制中断。
2. **核心方法**：论文将终端约束 MPC 引入 IBVS 误差动力学，并用 Kalman filter 在短时视觉退化期间预测图像矩状态，维持控制连续性。
3. **关键结果**：实验证明该方法在受约束和特征暂时缺失的情况下仍能保持收敛与稳定，明显提升了真实 UAV 视觉伺服的可部署性。

## 💎 价值评估

- **🔬 研究价值**：它抓住了 UAV 视觉伺服最常见的两个失效点——约束稳定性和短时观测丢失——属于很务实的控制层补强。
- **🚀 实践价值**：对主人做无人机贴近观察、空中抓取对位、近壁操作前的视觉闭环非常有用，尤其适合作为空中 manipulation 系统的末端逼近控制器。
- **📈 扩展潜力**：后续可把 KF 预测换成学习式观测补全，或把 MPC 目标扩到接触力/抓取可行性，进一步接上空中接触操作链路。

## 🎯 可落地实验点

**实验设计**：评估终端约束 MPC 与 moment loss compensation 对 UAV 近目标视觉伺服稳定性的改善。
- 对比基线：经典 IBVS、普通 MPC-IBVS、TC-MPC without KF、TC-MPC + KF
- 度量指标：最终像平面误差、特征丢失期间控制连续性、约束违背次数、任务成功率
- 预期结果：TC-MPC + KF 在目标附近更稳，短时遮挡或抖动下不易失控，整体任务成功率最高
