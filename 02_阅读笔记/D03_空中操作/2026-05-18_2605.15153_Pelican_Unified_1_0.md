# 📖 花火格式笔记

## 🎯 题目

Pelican-Unified 1.0: A Unified Embodied Intelligence Model for Understanding, Reasoning, Imagination and Action

## 📝 三句摘要

1. **问题背景**：现有具身系统常把理解、推理、未来生成与动作控制拆成多个孤立模块，导致表征割裂、训练目标不一致。
2. **核心方法**：Pelican-Unified 1.0 用单一 VLM 统一场景理解与链式推理，再接一个 Unified Future Generator，在同一去噪过程中联合生成未来视频与未来动作。
3. **关键结果**：单 checkpoint 同时在 8 个 VLM benchmark、WorldArena 与 RoboTwin 上取得强结果，说明统一范式没有明显牺牲理解、推理和行动能力。

## 💎 价值评估

- **🔬 研究价值**：这篇论文把“理解-想象-行动共表征”做成了清晰的统一训练范式，对主人关注的具身基础模型与 world-model + action 一体化很有参考价值。
- **🚀 实践价值**：如果后面要做无人机/机械臂联合决策，Pelican 的 latent future + action joint generation 思路很适合迁移到长程任务规划和视觉前瞻控制。
- **📈 扩展潜力**：可继续拆看其 unified future generator 是否能替换成 flow matching / diffusion action head，也能对接更强的可验证 planning token 或 safety critic。

## 🎯 可落地实验点

**实验设计**：做一个轻量版 unified latent planner，把语言推理 token、未来视觉预测和动作 chunk 放进同一训练图里，验证是否优于串联式 planner+policy。
- 对比基线：VLM planner + 单独 diffusion policy、world model + MPC、普通 VLA
- 度量指标：长时程成功率、replan 次数、未来预测误差、动作执行稳定性
- 预期结果：统一 latent 能减少模块间误差传递，在复杂长程任务中更稳
