# 📖 花火格式笔记

## 🎯 题目

Learning Cross-Coupled and Regime Dependent Dynamics for Aerial Manipulation

## 📝 三句摘要

1. **问题背景**：空中机械臂在载荷变化、机械臂重构和气动迟滞下会出现强耦合、时变且非平稳的残差动力学，导致解析模型和离线模型部署后快速失真。
2. **核心方法**：论文提出结构化 encoder-decoder 残差动力学学习框架，用非线性 latent encoder 建模跨变量耦合与时序依赖，再用可闭式贝叶斯更新的轻量线性 latent decoder 做在线自适应。
3. **关键结果**：在真实空中操作平台上，该方法提升残差预测精度与工况切换时的适应速度，并显著改善基于 MPC 的轨迹跟踪稳定性。

## 💎 价值评估

- **🔬 研究价值**：把“耦合时序残差 + 在线 regime adaptation”显式统一到空中操作动力学里，对 D03 很贴，能补强空中平台从感知走向精细接触控制的动力学建模缺口。
- **🚀 实践价值**：如果主人后面做 UAV 挂载机械臂或载荷扰动场景，这篇可直接提供一个比纯辨识或静态补偿更稳的控制建模思路。
- **📈 扩展潜力**：很适合和 world model / planner 结合，向“动作条件预测 + 自适应 MPC + 接触任务”延展，也能迁到空地异构平台的 residual adapter。

## 🎯 可落地实验点

**实验设计**：做一个 UAV-manipulator 的残差自适应控制对比实验，验证 regime 切换下 tracking 稳定性。
- 对比基线：固定参数动力学模型、纯离线 learned residual、无 covariance inflation 的在线更新
- 度量指标：位置/姿态跟踪误差、切换后恢复时间、MPC 重规划频率、不同载荷下成功率
- 预期结果：结构化 latent residual + 贝叶斯在线更新在载荷变化和机械臂姿态突变时最稳
