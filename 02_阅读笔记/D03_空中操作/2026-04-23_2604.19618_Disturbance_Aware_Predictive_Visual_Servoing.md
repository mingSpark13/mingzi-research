# 📖 花火格式笔记

## 🎯 题目

Autonomous UAV Pipeline Near-proximity Inspection via Disturbance-Aware Predictive Visual Servoing

## 📝 三句摘要

1. **问题背景**：近距离管道巡检要求无人机在复杂地形和扰动下稳定贴近飞行，传统方法容易受视觉低频、噪声和风扰影响。
2. **核心方法**：论文提出把四旋翼动力学与图像特征运动学统一进视觉伺服 MPC，并结合扩展状态卡尔曼滤波与图像特征预测估计总扰动。
3. **关键结果**：真实实验里，直管巡检的图像方向与横向偏差 RMSE 分别下降 52.63% 和 75.04%，且在风扰与弯管任务上基线失败而该方法成功完成。

## 💎 价值评估

- **🔬 研究价值**：这篇不是大模型路线，但把 disturbance-aware 预测视觉伺服做得很扎实，适合补强无人机高鲁棒控制知识树。
- **🚀 实践价值**：对近距巡检、沿结构飞行、受扰跟踪这类工业无人机任务很实用，也能给主人做鲁棒视觉控制 baseline。
- **📈 扩展潜力**：可往语义目标、长程任务规划或 aerial manipulation 接，形成“高层语义 + 低层抗扰控制”的分层方案。

## 🎯 可落地实验点

**实验设计**：把 ESKF-PRE-VMPC 思路迁到主人现有四旋翼任务，验证在风扰和感知延迟下的稳态误差改善。
- 对比基线：普通 IBVS / VMPC、无扰动补偿 MPC
- 度量指标：轨迹误差、图像特征 RMSE、任务完成率、风扰恢复时间
- 预期结果：加扰动估计后在强扰动和低频视觉更新下更稳、更抗偏航漂移
