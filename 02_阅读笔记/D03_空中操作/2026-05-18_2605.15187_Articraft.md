# 📖 花火格式笔记

## 🎯 题目

Articraft: An Agentic System for Scalable Articulated 3D Asset Generation

## 📝 三句摘要

1. **问题背景**：机器人学习和 articulated object 理解长期受限于高质量可动 3D 资产稀缺，人工建模和标注成本太高。
2. **核心方法**：Articraft 把“生成可动 3D 资产”转成“让 LLM 代理写构建程序”，借助受限 SDK、验证 harness 和测试反馈自动生成带关节与结构约束的资产。
3. **关键结果**：系统生成质量超过现有 articulated asset generator 与通用 coding agent，并构建出覆盖 245 类、超 10K 样本的 Articraft-10K 数据集。

## 💎 价值评估

- **🔬 研究价值**：它不是直接做控制算法，但补的是具身智能很卡脖子的训练资产供给，对数据飞轮和仿真数据扩张很关键。
- **🚀 实践价值**：主人如果后面做空中操作、抓取或跨载体泛化，这类自动生成 articulated assets 的流程很适合拿来扩充仿真场景和物体库。
- **📈 扩展潜力**：可以继续接触觉/材质/物理参数自动补全，也能和大规模 policy pretraining、sim2real domain randomization 联动。

## 🎯 可落地实验点

**实验设计**：基于现有仿真平台，做一个“小型 Articraft 流程”自动生成若干关节物体，用于训练操作策略并测试资产多样性是否提升泛化。
- 对比基线：人工少量建模资产、传统 procedural generation、公开静态资产库
- 度量指标：资产可用率、URDF/关节合法率、下游策略泛化成功率、场景覆盖度
- 预期结果：自动生成的 articulated 资产能更快扩展训练分布，并提升对未见物体结构的适应性
