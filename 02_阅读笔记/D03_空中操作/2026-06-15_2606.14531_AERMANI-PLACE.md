# 📖 花火格式笔记

## 🎯 题目

AERMANI-PLACE: Language Guided Object Placement with Aerial Manipulators

## 📝 三句摘要

1. **问题背景**：现有空中操作放置任务通常要求用户直接给出公制坐标，交互很反人类，也不适合开放场景部署。
2. **核心方法**：AERMANI-PLACE 先用图像编辑模型根据场景图像和自然语言指令生成带视觉标记的目标图，再结合深度观测把标记回投成物理放置点，最后由空中机械臂生成并执行放置轨迹。
3. **关键结果**：在 100 个语言引导放置任务上，方法的目标位置推断成功率达到 87%，迁移到真实空中操作平台后平均成功率仍有 72%。

## 💎 价值评估

- **🔬 研究价值**：它把“语言 → 场景可操作放置点 → 轨迹执行”真正打通了，不只是做语义 grounding，而是把放置动作闭环接上了空中机械臂。
- **🚀 实践价值**：对主人现在关心的空中操作/开放任务交互很有用，尤其适合做“把它放到桌角旁边”这类自然语言目标指定接口。
- **📈 扩展潜力**：后面可以往多轮指令、视觉指点、3D scene graph、动态目标更新上扩展，也能和 D06 的语义导航接口合流。

## 🎯 可落地实验点

**实验设计**：在 AirSpark 或仿真空中抓放任务里复现“语言描述 → 视觉标记 → 深度回投 → 放置轨迹”的四段式接口。
- **对比基线**：显式坐标输入放置 / 直接语言到动作 policy / 只做 2D grounding 不做深度回投
- **度量指标**：放置点偏差、任务成功率、语言泛化成功率、真实部署成功率
- **预期结果**：四段式方案会在开放语言描述和真实环境落地上明显强于直接端到端动作映射
