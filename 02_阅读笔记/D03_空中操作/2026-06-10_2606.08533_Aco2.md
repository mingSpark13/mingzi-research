# 📖 花火格式笔记

## 🎯 题目

**Aco2** — 基于上下文对比元强化学习的自主空中操作

## 📝 三句摘要

1. **问题背景**：UAV 在物流、服务机器人领域对自主载荷获取+投递需求日益增长。现有方法多假设预附着载荷或用专用夹爪，无法处理"不同 payload 引起不同动力学 + 单一策略需在线适应"的端到端空中投递。
2. **核心方法**：作者提出 **Aco2** 框架——四旋翼 + 轻量 hook 连续完成 pickup/transport/delivery，无需人工干预。核心设计：**contextual observation encoder** 从近期交互历史中推断 compact latent context，使策略能在线适应 payload 依赖动力学；**contrastive objective** 把 context embedding 围绕任务相关变化结构化，提升跨 payload 泛化。
3. **关键结果**：完全在仿真中训练（大量 domain randomization），可直接部署到真实四旋翼**无实机微调**。实验显示可无人值守递送对飞行器来说相对重/大的带柄物体，验证空中物流、应急物资、工业运输的实用潜力。

## 💎 价值评估

- **🔬 研究价值**：把 contextual meta-RL + contrastive learning 用在"不同 payload 在线适应"上，是空中操作 + 元强化学习的典范。
- **🚀 实践价值**：对 AirSpark 未来"导航—接近—操作"链路中"操作段"有直接借鉴。无需系统辨识的在线适应是关键工程优势。
- **📈 扩展潜力**：contextual encoder + contrastive objective 可推广到不同机械臂、不同负载形状等场景。

## 🎯 可落地实验点

**实验设计**：把 Aco2 的 contextual contrastive meta-RL 框架移植到 AirSpark 仿真器，对接我们的"操作段"。
- **场景**：AirSpark 仿真器中添加带 hook 的四旋翼 + 多类 payload
- **基线**：固定策略（无 context）/ 系统辨识+MPC / 纯 meta-RL 无 contrastive
- **度量**：跨 payload 适应速度（在线 few-shot 后成功率）、仿真到真实迁移成功率
- **预期**：完整 Aco2 比固定策略在跨 payload 任务上提升 ≥30%
