# 📖 花火格式笔记

## 🎯 题目

PointACT: Vision-Language-Action Models with Multi-Scale Point-Action Interaction

## 📝 三句摘要

1. **问题背景**：现有 VLA 多依赖 2D 视觉表征，难以稳定处理精细几何与空间对齐要求高的操作任务。
2. **核心方法**：论文提出 PointACT，把分层点云表示直接注入动作解码器，用 multi-scale point-action interaction 让动作 token 同时关注局部几何和全局场景。
3. **关键结果**：在 LIBERO 与 RLBench 上相对 SOTA 预训练 VLA 均有稳定提升，RLBench-10Tasks 成功率提升约 10%，说明 3D-aware action decoding 对精细操作很值钱。

## 💎 价值评估

- **🔬 研究价值**：把 3D 几何从“前端附加输入”推进到“动作解码核心接口”，对 VLA 结构改造很直接。
- **🚀 实践价值**：适合给主人后续做点云/多视角感知增强的 manipulation policy 设计提供结构参考。
- **📈 扩展潜力**：可继续接 world model、6D pose 或触觉分支，做更强的空间 grounding 与闭环控制。

## 🎯 可落地实验点

**实验设计**：在现有 VLA/ACT 策略上增加轻量点云 token 分支，验证 3D 几何是否能提升精细抓取与插接成功率
- 对比基线：纯 RGB VLA、RGB+point cloud early fusion、PointACT 式 late interaction
- 度量指标：任务成功率、末端位姿误差、对遮挡/视角变化鲁棒性
- 预期结果：late interaction 方案在精细操作与 OOD 视角下更稳
