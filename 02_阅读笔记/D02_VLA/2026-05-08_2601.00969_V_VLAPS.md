# 📖 花火格式笔记

## 🎯 题目

Value Vision-Language-Action Planning & Search

## 📝 三句摘要

1. **问题背景**：纯 behavior cloning 的 VLA 在分布外状态下容易脆弱失效，单靠 policy prior 引导的测试时搜索也缺少对未来回报的 grounded 估计。
2. **核心方法**：论文提出 V-VLAPS，在固定 Octo backbone 的 latent state 上训练一个轻量 value head，并把它并入 MCTS 节点打分，让搜索不只看先验概率，还看预估回报。
3. **关键结果**：在 LIBERO 操作任务上，V-VLAPS 相比只依赖 VLA prior 的基线把成功率提升了 5 个点以上，同时把平均 MCTS 模拟次数再降 5–15%。

## 💎 价值评估

- **🔬 研究价值**：它补的是 VLA 很实际的一块短板——长程决策和分布外纠错，不是重新训练更大模型，而是给现有 backbone 加可学习 value guidance。
- **🚀 实践价值**：对主人这种想把高层语义与低层控制接起来的路线很有用，value-guided search 比纯 imitation 更适合安全敏感任务。
- **📈 扩展潜力**：后续可把 value head 接到 UAV 导航/空中操作 rollout 上，做 risk-aware action selection 或轻量在线重规划。

## 🎯 可落地实验点

**实验设计**：给现有 VLA/策略模型增加 latent-value head，验证搜索纠错收益
- 对比基线：Octo / OpenVLA + vanilla VLAPS 或纯 BC policy
- 度量指标：长程任务成功率、OOD 场景恢复率、平均搜索步数、推理时延
- 预期结果：小 value head 就能显著减少无效搜索，把高层决策从“盲搜”变成“带回报偏置的搜”
