# 📖 花火格式笔记

## 🎯 题目

CorridorVLA: Explicit Spatial Constraints for Generative Action Heads via Sparse Anchors

## 📝 三句摘要

1. **问题背景**：生成式 VLA 虽然能把视觉语言映射到连续动作，但空间约束大多只藏在隐式特征里，导致动作头对接触偏差和轨迹漂移不够稳。
2. **核心方法**：论文提出 CorridorVLA，用稀疏空间锚点表示增量物理变化，再构造允许小噪声但惩罚越界轨迹的 corridor loss，把显式几何约束直接施加到 flow-matching 动作头上。
3. **关键结果**：在更难的 LIBERO-Plus 基准上，CorridorVLA 对 SmolVLA 和 GR00T 基线带来 3.4%–12.4% 成功率提升，其中 GR00T-Corr 达到 83.21%。

## 💎 价值评估

- **🔬 研究价值**：它把“空间引导”从视觉隐变量里剥出来，做成可解释、可控、可训练的动作生成约束，对生成式 VLA 的稳定性研究很有参考价值。
- **🚀 实践价值**：如果主人后面做无人机或机械臂动作生成，可以把航迹、接触容差、禁飞区边界编码成 corridor，减少策略输出飘掉的问题。
- **📈 扩展潜力**：这套 sparse anchor + corridor 的思路很适合继续扩到 3D waypoint、接触先验、力约束，甚至和 world model 预测轨迹联动。

## 🎯 可落地实验点

**实验设计**：把 CorridorVLA 的显式 corridor 正则迁移到主人的生成式动作模型中，验证是否能提升长程操作和无人机精细机动的轨迹稳定性。
- 对比基线：原始 flow-matching / diffusion action head、仅视觉条件的 VLA、加入 waypoint 监督但无 corridor loss 的版本
- 度量指标：任务成功率、轨迹偏差、接触失败率、越界率、对扰动的鲁棒性
- 预期结果：显式 corridor 约束能明显降低动作漂移，在多步操作和狭窄空间任务中提升成功率
