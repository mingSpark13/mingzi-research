# 📖 花火格式笔记

## 🎯 题目

Evo-Depth: A Lightweight Depth-Enhanced Vision-Language-Action Model

## 📝 三句摘要

1. **问题背景**：现有 VLA 多依赖 2D 视觉表征，遇到精细空间定位、摆放和接触操作时容易因为深度感不足而掉点，而显式 3D 输入又常带来额外传感器和重计算负担。
2. **核心方法**：论文提出 Evo-Depth，用轻量 Implicit Depth Encoding Module 从多视角 RGB 中提取隐式深度特征，再通过 Spatial Enhancement Module 以调制方式注入视觉语言表征，并配合 Progressive Alignment Training 逐步对齐动作学习。
3. **关键结果**：仅 0.9B 参数下，Evo-Depth 在 Meta-World、VLA-Arena、LIBERO、LIBERO-Plus 均取得强表现，真实实验平均成功率达 90%，同时保持最低显存占用和最高推理频率。

## 💎 价值评估

- **🔬 研究价值**：它抓的是 VLA 很实在的短板——空间感知不足——但没有走“堆更大 3D foundation model”的老路，而是给出更轻的隐式深度增强方案。
- **🚀 实践价值**：如果主人后面做轻量 VLA、机械臂部署或边缘端推理，这篇很适合作为“低成本补空间感”的工程参考。
- **📈 扩展潜力**：后面可以继续接触觉、力觉或单目视频时序，把隐式几何增强扩成更完整的多模态空间 grounding。

## 🎯 可落地实验点

**实验设计**：在现有轻量 VLA baseline 上增加一个隐式深度调制支路，验证是否能用极小计算代价换取空间操作成功率提升。
- 对比基线：2D-only VLA、显式深度/点云增强 VLA、使用大型几何 foundation model 的 implicit 3D VLA
- 度量指标：操作成功率、显存占用、推理频率、受扰动场景下的鲁棒性
- 预期结果：轻量隐式深度增强在部署效率基本不变的前提下，能稳定提升空间精度和泛化能力
