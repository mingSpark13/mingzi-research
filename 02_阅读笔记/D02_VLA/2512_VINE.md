## 🎯 题目

VINE: Vision-Language Models for Hierarchical Robot Control

## 📝 三句摘要

1. **问题背景**：VLM 在高层推理上强，但直接输出低层高频动作控制时语义能力被稀释；需要显式的 hierarchical 分离。
2. **核心方法**：VINE 明确提出 System 2（VLM reasoning） + System 1（low-level control）的分层架构，VLM 负责高层任务分解和子目标生成，低层控制器负责实时执行。
3. **关键结果**：在多种 manipulation 任务上验证了分层架构比端到端 VLA 更能保持语义理解能力。

## 💎 价值评估

- **🔬 研究价值**：VINE 是 hierarchical reasoning-control 分离的标志性工作，是主人 PMI 框架的最近邻基础之一。
- **🚀 实践价值**：分层架构可直接用于"高层通用 + 低层躯体特定"的设计思路。
- **📈 扩展潜力**：可与主人的 PMI 接口结合，形成完整的"高层意图 → PMI → 低层执行"链路。

## 🎯 可落地实验点

**实验设计**：以 VINE 的分层架构为基础，加入主人的 PMI 接口层
- 对比基线：端到端 VLA、分层 VLA 无 PMI
- 度量指标：跨 embodiment 迁移能力、语义理解保持率、执行频率
- 预期结果：分层 + PMI 应优于纯分层或端到端方法
