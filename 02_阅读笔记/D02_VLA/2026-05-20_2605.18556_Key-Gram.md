# 📖 花火格式笔记

## 🎯 题目

Key-Gram: Extensible World Knowledge for Embodied Manipulation

## 📝 三句摘要

1. **问题背景**：现有 vision-language-action 与 world-action 控制策略常把语言知识和视觉推理塞进同一主干，导致模态竞争，也让知识扩展必须改动 backbone。
2. **核心方法**：论文提出 Key-Gram，把指令拆成 task-specific key-grams，经哈希检索外部语言记忆，再通过门控和轻量卷积注入隐藏层，实现“语言知识外置、视觉推理留给主干”。
3. **关键结果**：在 RoboTwin2.0、LIBERO/LIBERO-Plus 与真实双臂长程操作上，Key-Gram 对 π0/π0.5 都带来稳定增益，跨域迁移和真实任务收益尤其明显。

## 💎 价值评估

- **🔬 研究价值**：这篇把“可扩展外部知识记忆”明确嫁接到具身操作控制里，正好补主人知识库里 VLA 如何低成本扩充语言先验的关键空位。
- **🚀 实践价值**：如果后面做无人机或机械臂语言控制，这种 O(1) 外部记忆检索思路很适合在不重训大模型的前提下快速注入任务知识。
- **📈 扩展潜力**：后续可把 key-gram memory 与世界模型、在线经验回放或多机器人共享知识库结合，形成真正可持续进化的 embodied policy。

## 🎯 可落地实验点

**实验设计**：在现有 VLA 或 UAV-language control 框架上加一个可插拔语言记忆表，验证外部指令知识能否提升组合泛化和跨场景迁移。
- 对比基线：原始 backbone 微调、LoRA 微调、无记忆版本
- 度量指标：任务成功率、未见指令组合泛化、目标域零样本迁移得分
- 预期结果：外部知识记忆能显著提升长尾指令理解与跨域组合泛化
