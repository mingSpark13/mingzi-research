# 📖 Frequency-Aware Flow Matching for Continuous and Consistent Robotic Action Generation

## 🎯 题目

Frequency-Aware Flow Matching：用于连续一致机器人动作生成的频率感知流匹配

## 📝 三句摘要

1. **问题背景**：现有 flow matching / diffusion policy 依赖**离散化 action chunks**，对**异构控制频率**采样的演示脆弱，并产生**时序不一致动作**，损害控制稳定性。
2. **核心方法**：提出 Frequency-Aware Flow Matching (FAFM)，**显式建模演示数据的频率分布**，让 flow matching 产生**与训练时控制频率解耦的连续一致动作**。
3. **关键结果**：跨频率演示数据训练时显著优于离散 chunks，闭环控制更稳定，长程任务一致性提升（具体数据见 PDF）。

## 💎 价值评估

- **🔬 研究价值**：直接命中 flow matching 的两个核心痛点（频率异构 + 块边界不一致），是从"能跑"到"能稳跑"的关键补丁。
- **🚀 实践价值**：主人项目正在用 π0 风格 flow matching，**混合不同控制频率的演示数据**（人手 30Hz + 机械臂 100Hz）是常态，本方法几乎是"即插即用"。
- **📈 扩展潜力**：与 Legato 的 chunk continuation、SmoothNet 的动作平滑正交，组合后能彻底解决"块边界 spurious switching"问题。

## 🎯 可落地实验点

**实验设计**：在主人 sim2real 实验台上加 FAFM 头
- 对比基线：π0 离散 chunks、π0 + Legato continuation、π0 + FAFM、π0 + FAFM + Legato
- 度量指标：跨频率演示训练一致性、动作块边界跳跃幅度、长程任务方差、控制频率漂移恢复时间
- 预期结果：FAFM 在混合频率数据训练下成功率提升 5-10pp；与 Legato 组合后块边界跳跃幅度再降 30-50%
