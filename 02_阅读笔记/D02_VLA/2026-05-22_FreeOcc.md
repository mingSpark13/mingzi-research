# 📖 花火格式笔记

## 🎯 题目

FreeOcc: Training-Free Open-Vocabulary 3D Occupancy Prediction

## 📝 三句摘要

1. **问题背景**：现有 3D 占据预测方法依赖大规模标注数据或特定类别训练，无法实现零样本开放词汇感知。
2. **核心方法**：提出 FreeOcc，首个**无需训练的开放词汇 3D 占据预测**框架，利用 SLAM（DROID-SLAM）提供相机位姿和点云地图，结合 3DGS 渲染 + 语言查询实现语义占据预测。
3. **关键结果**：在 EmbodiedOcc-ScanNet 上，RGB-D 版本达到 **34.40 IoU / 15.84 mIoU**，超自监督基线两倍以上；在 ReplicaOcc 跨数据集泛化基准上，RGB-D 版本达 **55.65 IoU / 20.90 mIoU**，零样本泛化能力显著。

## 💎 价值评估

- **🔬 研究价值**：首个 training-free 开放词汇 3D 占据预测范式，无需任何任务特定训练或语义标注即可响应任意自然语言查询；为零样本开放世界感知提供了新思路。
- **🚀 实践价值**：直接基于单目/RGB-D 图像序列在线构建语义地图，无需大规模预训练数据，降低了具身场景建图的数据门槛。
- **📈 扩展潜力**：可与下游导航/规划模块结合构建完整"建图→理解→导航"系统；SLAM 主干可替换为 MASt3R-SLAM 或 VGGT-SLAM。

## 🎯 可落地实验点

**实验设计**：在 ROS 机器人平台上集成 FreeOcc，构建"开放词汇语义占据地图 → 基于语言指令的目标导航"完整 pipeline。

- **对比基线**：无 SLAM 基线（纯 3DGS）、固定阈值占据预测
- **度量指标**：语义 IoU/mIoU、导航成功率、平均路径长度
- **预期结果**：在未知类别语言查询（如"找绿色椅子"）下，地图能正确分割目标区域，下游导航任务成功率显著提升
