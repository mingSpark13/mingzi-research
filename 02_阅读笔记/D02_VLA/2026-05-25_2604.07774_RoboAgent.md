# 📖 花火格式笔记

## 🎯 题目

RoboAgent: Chaining Basic Capabilities for Embodied Task Planning

## 📝 三句摘要

1. **问题背景**：现有 VLM 做 embodied task planning 时，常被多轮交互、长程推理和上下文膨胀拖垮，直接 CoT 也很难保证中间推理可靠可控。  
2. **核心方法**：论文提出 RoboAgent，把规划拆成 scheduler + 多个基础 capability，由同一个 VLM 维护各自上下文并按需调用，再用 behavior cloning、DAgger 和 expert-guided RL 三阶段训练整套规划链。  
3. **关键结果**：在 EB-ALFRED、ALFWorld 和 LoTa-WAH 上，RoboAgent 用 Qwen2.5-VL-3B 达到很强泛化表现；其中 EB-ALFRED 平均 SR 67.0，明显超过 REBP 的 35.6 和 RoboGPT-R1 的 55.3，说明“能力拆分式规划”比直接端到端推理更稳。  

## 💎 价值评估

- **🔬 研究价值**：它把 embodied planning 中模糊的“思维链”改写成可监督、可调度、可分解的 capability graph，对长程任务规划路线很有启发。  
- **🚀 实践价值**：主人后续做 VLA/机器人高层决策时，可以直接借这套“共享主模型 + 分能力上下文”的结构，降低长上下文污染和错误累积。  
- **📈 扩展潜力**：若把 scheduler 查询接口接到真实机器人技能库或 PMI 接口，能自然扩成更强的具身任务操作系统。  

## 🎯 可落地实验点

**实验设计**：在现有机器人 agent 中实现一个轻量 capability scheduler，把环境理解、目标定位、子任务检查和动作选择拆成独立上下文槽位
- 对比基线：单一 CoT planner、ReAct planner、固定模板分步 planner
- 度量指标：任务成功率、平均规划步数、错误恢复次数、上下文 token 开销
- 预期结果：分能力调度在长程任务和复杂指代场景下更稳，且更易定位失败原因
