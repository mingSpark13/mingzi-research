# 📖 花火格式笔记

## 🎯 题目

PokeVLA: Empowering Pocket-Sized Vision-Language-Action Model with Comprehensive World Knowledge Guidance

## 📝 三句摘要

1. **问题背景**：现有 VLA 在机器人操作里常卡在模型太重、空间感弱、缺少高层知识先验，导致部署效率和复杂场景成功率都不够稳。
2. **核心方法**：PokeVLA 先用 240 万条多模态样本预训练紧凑型 PokeVLM，把 spatial grounding、affordance 与 embodied reasoning 压进视觉语言表征，再通过多视角目标语义学习、几何对齐和 action expert 注入操作相关动作能力。
3. **关键结果**：它在 LIBERO-Plus 和真实部署里都优于同量级基线，说明“小模型 + 世界知识引导”这条路，确实能把 VLA 做得更轻更稳。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯卷参数量，而是把“世界知识监督 + 空间语义 + 操作动作”打包成轻量 VLA 路线，对 D02/D04 之外的具身通用底座很有参考意义。
- **🚀 实践价值**：如果明子后面想做资源受限平台上的具身策略，这篇给了很实用的方向——先把可迁移的视觉语言知识学扎实，再小规模注入动作专家，而不是一上来就堆超大机器人模型。
- **📈 扩展潜力**：它可自然接到 object-aware 预测、world model 辅助控制、跨载体数据配对等路线，适合当“轻量具身底模”分支继续扩展。

## 🎯 可落地实验点

**实验设计**：在现有具身操作任务里复现“先知识预训练、后动作注入”的两阶段轻量 VLA 路线，比较紧凑模型与直接端到端训练的效果差异。
- 对比基线：直接训练小型 VLA / 仅视觉语言预训练 / PokeVLA 两阶段方案
- 度量指标：任务成功率、参数量、推理延迟、真实扰动下鲁棒性
- 预期结果：两阶段方案在相近算力下获得更高成功率，并在新场景和干扰条件下更稳
