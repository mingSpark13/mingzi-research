# 📖 花火格式笔记

## 🎯 题目

Coarse-to-Control: Action-Token Planning for Vision-Language-Action Models

## 📝 三句摘要

1. **问题背景**：直接从观测到动作的 VLA 在长程多阶段任务上容易把高层意图和低层执行挤进同一次前向传播，早期小错会一路放大。
2. **核心方法**：论文提出 Coarse-to-Control，让模型先在共享 residual-VQ 动作词表里生成 coarse planning tokens，再条件化生成 executable action tokens，把 planning 显式放进 action space 而不是文本或图像空间。
3. **关键结果**：作者报告在 LIBERO 上平均成功率达 97.9%，在 SimplerEnv-WidowX 达 83.3%，并在真实操作任务中稳定优于直接 action generation。

## 💎 价值评估

- **🔬 研究价值**：这篇很适合放进主人当前路线里——它把“中间层”从语言链路挪到动作 token，本质是在做更贴控制面的可执行 planning interface。
- **🚀 实践价值**：如果后续要做长程操作、空中抓取或多阶段任务，这种 coarse plan prefix 很可能比文本 CoT 更稳，也更容易接到现有 action tokenizer 上。
- **📈 扩展潜力**：能继续和 continuous reasoning、world model rollout 或 UAV planner 结合，形成“高层粗规划 + 低层执行”双时域结构。

## 🎯 可落地实验点

**实验设计**：给现有 VLA/DroneVLA 路线增加 coarse action plan prefix，测试多阶段任务和长时序鲁棒性
- 对比基线：π0.5 / DroneVLA / Continuous Reasoning for VLA
- 度量指标：长程任务成功率、早期错误恢复率、action token 一致性、推理时延
- 预期结果：显式 coarse planning 能明显提升多阶段任务稳定性，代价是轻微 token 开销，但整体性价比优于文本 reasoning
