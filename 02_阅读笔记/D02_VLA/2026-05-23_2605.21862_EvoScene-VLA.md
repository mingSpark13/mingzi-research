# 📖 花火格式笔记

## 🎯 题目

EvoScene-VLA: Evolving Scene Beliefs Inside the Action Decoder for Chunked Robot Control

## 📝 三句摘要

1. **问题背景**：chunked VLA 虽然能一次预测多步动作，但后续动作仍只条件于当前观测，无法显式维护“动作已经改变了场景”这一事实。
2. **核心方法**：论文提出 EvoScene-VLA，在 action decoder 内维护 recurrent scene prefix，把上一 chunk 动作导致的几何变化编码成 action-updated scene prior，并用深度/3D teacher 进行几何锚定监督。
3. **关键结果**：在 31 个 RoboTwin 任务上把平均成功率从 87.2% 提升到 89.1%，随机评测从 86.1% 提升到 88.5%，真实 Galaxea R1-Lite 也超过全部 baseline，说明“随动作演化的场景信念”对长程控制很值钱。

## 💎 价值评估

- **🔬 研究价值**：这篇不是单纯堆历史帧，而是把场景状态显式塞进 chunked policy 内循环，补上 action chunking 在接触、遮挡、物体位移场景里的语义断层。
- **🚀 实践价值**：很适合给主人后续做 ACT/VLA、world model、视觉闭环控制结合时参考，尤其适合需要多步接触连续性的任务。
- **📈 扩展潜力**：后续完全可以把 recurrent scene prefix 扩展成更强的隐式世界模型，再迁到空中操作或移动操作平台上做跨 chunk 规划。

## 🎯 可落地实验点

**实验设计**：在现有 ACT/VLA 策略里加入轻量 scene state token，比较纯当前帧控制与 action-updated scene prior 对接触后连续动作的帮助
- 对比基线：普通 ACT、仅时序堆帧的 temporal VLA、EvoScene-VLA 式 scene prefix
- 度量指标：任务成功率、接触后恢复率、遮挡后动作稳定性、跨 chunk 末端位姿误差
- 预期结果：scene prior 在物体位移、接触遮挡、长序列 chunk 执行时能显著减少后续动作漂移
