# 📖 花火格式笔记

## 🎯 题目

Move-Then-Operate: Behavioral Phasing for Human-Like Robotic Manipulation

## 📝 三句摘要

1. **问题背景**：单一策略把“靠近目标”和“精细接触操作”混在一起学，常常顾此失彼，导致高精度 manipulation 训练效率低、成功率也不稳。
2. **核心方法**：论文显式把操作拆成 move 与 operate 两个阶段，用 learnable phase selector 路由到 dual-expert policy，并通过 MLLM 管线自动生成阶段标签，让模型更贴近人类分阶段操作习惯。
3. **关键结果**：在 RoboTwin2 上平均成功率达到 68.9%，比单体 π0 基线高 24%，还能用更少数据和更少训练步数达到更好性能。

## 💎 价值评估

- **🔬 研究价值**：这篇把“分阶段行为先验”做成结构归纳偏置，不是小修小补，而是对长程操作策略设计给出一个很干净的工程范式。
- **🚀 实践价值**：主人后续做机械臂/具身操作时，非常适合拿来做 coarse-to-fine 或双系统架构参考——先移动对齐，再做接触关键动作，天然更稳。
- **📈 扩展潜力**：可继续延展到 aerial manipulation、双臂协作、甚至 VLA + world model 框架里，把阶段切换作为 planner 与 controller 的接口。

## 🎯 可落地实验点

**实验设计**：在 Isaac Sim 机械臂任务里，把 pick-place / 插接 / drawer 开合任务拆成 approach 与 contact 两阶段，对比单策略与双专家策略。
- 对比基线：单一 diffusion/VLA policy、手工状态机阶段切换
- 度量指标：成功率、接触后失败率、训练收敛步数、阶段切换误判率
- 预期结果：双阶段策略在接触密集任务上更稳，样本效率更高，失败更多集中在 phase selector 而非动作生成本身
