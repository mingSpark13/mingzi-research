# 📖 花火格式笔记

## 🎯 题目

Qwen-VLA: Unifying Vision-Language-Action Modeling across Tasks, Environments, and Robot Embodiments

## 📝 三句摘要

1. **问题背景**：现有具身模型常按操作、导航、轨迹预测分别建模，导致跨任务与跨机器人迁移能力碎片化。
2. **核心方法**：Qwen-VLA把多类 embodied decision-making 统一进同一个 VLA 框架，用 DiT action decoder、联合预训练配方和 embodiment-aware prompt 支持多任务多载体。
3. **关键结果**：在 LIBERO、Simpler、RoboTwin、R2R/RxR、ALOHA 和 DOMINO 上都给出强结果，说明统一模型对 OOD 场景和不同机器人形态都有较强泛化力。

## 💎 价值评估

- **🔬 研究价值**：这是近期很强的“统一 VLA”信号，直接对齐主人关注的通用 embodied foundation model 路线。
- **🚀 实践价值**：embodiment prompt + joint pretraining 的组合，很适合借鉴到多平台机器人或空地跨载体任务里。
- **📈 扩展潜力**：后续可重点跟进它的 action space 统一方式、数据混配策略、以及跨任务负迁移是否被真正解决。

## 🎯 可落地实验点

**实验设计**：验证统一预训练是否真的提升跨任务/跨载体泛化
- 对比基线：单任务 VLA、按任务分头训练的 VLA、无 embodiment prompt 版本
- 度量指标：任务成功率、OOD 成功率、跨机器人迁移成功率、动作延迟
- 预期结果：统一模型在多任务覆盖和 OOD 泛化上更强，但可能牺牲某些单任务峰值性能
