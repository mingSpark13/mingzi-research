# 📖 花火格式笔记

## 🎯 题目

Robust Finetuning of Vision-Language-Action Robot Policies via Parameter Merging

## 📝 三句摘要

1. **问题背景**：通用机器人策略在新任务少样本微调时容易过拟合，不仅丢掉原有泛化能力，还会在新任务分布外变体上表现崩掉。
2. **核心方法**：论文提出把微调后的模型参数与预训练基座参数做插值融合，用 parameter merging 保留通用能力，同时吸收新技能。
3. **关键结果**：方法在仿真和真实实验中都优于单独的 pretrained / finetuned 模型，对新任务 OOD 变体更稳，并支持持续增量学新技能。

## 💎 价值评估

- **🔬 研究价值**：把“微调后遗忘”和“新任务过拟合”这两个 VLA 落地痛点用一个极简策略一起打掉，思路非常干净。
- **🚀 实践价值**：主人以后若要在已有通用策略上快速注入新技能，这种 merge 方案几乎是低成本必试 baseline，尤其适合 demo 少、又不想毁掉老能力的场景。
- **📈 扩展潜力**：可继续和 replay、adapter、LoRA、持续学习训练流结合，做真正可滚动更新的机器人技能库。

## 🎯 可落地实验点

**实验设计**：在一个已有 VLA 基座上做少样本新任务微调，比较纯微调与 parameter merging 的保底能力。
- 对比基线：pretrained only、finetune only、parameter merging、replay finetuning
- 度量指标：新任务成功率、旧任务保留率、OOD 泛化成功率、连续多任务后的平均遗忘度
- 预期结果：parameter merging 在保住旧能力的同时，取得更稳的新任务泛化表现
