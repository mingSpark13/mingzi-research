# 📖 花火格式笔记

## 🎯 题目

LDA-1B: Scaling Latent Dynamics Action Model via Universal Embodied Data Ingestion

## 📝 三句摘要

1. **问题背景**：现有机器人 foundation model 多靠行为克隆吃高质量示范，浪费了异构 embodied data 里的动力学知识，也难把统一世界模型规模化。
2. **核心方法**：LDA-1B 提出联合学习 dynamics、policy 与 visual forecasting 的 1B 模型，配套 EI-30k 统一格式数据集、DINO latent 预测空间和处理异步视觉/动作流的多模态 diffusion transformer。
3. **关键结果**：在仿真和真实任务中，相比 prior methods 与 π0.5 最多提升 21%/48%/23%，还能把通常有害的低质量轨迹转化成有效增益，明显强化数据利用率。

## 💎 价值评估

- **🔬 研究价值**：这篇把“统一动力学 + policy + forecasting”的 foundation 路线做到了 1B 级，是 D02 和 D01 之间很关键的桥梁论文。
- **🚀 实践价值**：对主人正在关注的 VLA / world model 融合、低质量数据再利用、统一 action interface 都很有启发。
- **📈 扩展潜力**：后续值得深挖它怎样处理异步视觉-动作流，以及 EI-30k 式数据标准化能否迁到无人机和跨载体体系。

## 🎯 可落地实验点

**实验设计**：在现有 VLA 训练链上加入 latent dynamics 辅助头和低质量数据再利用策略，验证是否能改善长程接触任务与数据效率。
- 对比基线：纯 BC/VLA、无 dynamics 头版本、丢弃低质量轨迹版本
- 度量指标：成功率、长程任务完成率、低质量数据利用增益、OOD 泛化表现
- 预期结果：联合 dynamics 学习应提升长程稳定性，并让原本噪声较大的轨迹变成有益监督
