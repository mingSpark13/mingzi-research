# 📖 花火格式笔记

## 🎯 题目

ProbeAct: Probe-Guided Training-Free Failure Recovery in Vision-Language-Action Models

## 📝 三句摘要

1. **问题背景**：预训练 VLA 对光照、相机视角和微小初始状态变化非常敏感，真实部署时常因抓取/放置失败而缺少轻量级安全兜底。
2. **核心方法**：ProbeAct 不改 VLA 权重，而是从中间 hidden state 训练轻量 probe 预测任务相关物体 3D 位置，再结合基于夹爪内部信号和运动学的失败检测状态机，以及层次化 CBF 过滤器做最小动作修正。
3. **关键结果**：作为即插即用、training-free 的运行时恢复回路，ProbeAct 在 LIBERO-plus 上把 OpenVLA-OFT 成功率从 69.6% 提升到 74.1%，还能兼容 base 与 fine-tuned VLA 策略。

## 💎 价值评估

- **🔬 研究价值**：它不是再训一个更大的 VLA，而是从已有 hidden feature 挖出可控状态，再用安全过滤和状态机做恢复，思路很利落。
- **🚀 实践价值**：对真实机器人尤其友好，因为它训练成本低、可以外挂到已有策略上，适合当部署层的通用 safety net。
- **📈 扩展潜力**：后续可和接触信号、世界模型或任务级恢复规划结合，做更强的 failure memory 与多阶段恢复器。

## 🎯 可落地实验点

**实验设计**：在主人现有 VLA 或 imitation policy 上添加“hidden-state probe + 失败检测 + CBF filter”运行时补丁，测试小扰动和视角偏移下的成功率增益。
- 对比基线：原始 VLA、仅 probe 检测无控制修正、仅规则式失败恢复、完整 ProbeAct
- 度量指标：成功率、碰撞次数、最小修正幅度、推理延迟
- 预期结果：完整框架应以较小延迟换来显著恢复收益，尤其在 grasp/place 两阶段任务更明显
