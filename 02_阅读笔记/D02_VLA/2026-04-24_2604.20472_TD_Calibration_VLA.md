# 📖 花火格式笔记

## 🎯 题目

Temporal Difference Calibration in Sequential Tasks: Application to Vision-Language-Action Models

## 📝 三句摘要

1. **问题背景**：现有 VLA 在顺序任务里常会给出看似自信但并不可靠的成功概率，导致机器人难以据此做风险感知与安全决策。
2. **核心方法**：论文把序列校准写成 episodic 任务下的时序置信度估计问题，并证明其风险最优解可与价值函数对齐，于是用 temporal-difference 学习来做 VLA 的时序校准。
3. **关键结果**：在仿真与真实机器人数据上，TD calibration 相比现有校准方法更稳，还说明单步 action probability 在正确 TD 校准后也能变成可用的不确定性信号。

## 💎 价值评估

- **🔬 研究价值**：这篇把“校准”从静态分类问题推进到具身顺序决策里，还把 VLA 置信度和 RL value estimation 接上了，桥接味很强。
- **🚀 实践价值**：对主人关心的 VLA 落地很有用，因为真实部署时最缺的不是再多一点平均成功率，而是知道“这一步到底靠不靠谱”。
- **📈 扩展潜力**：后续可以接 refusal / retry / human-in-the-loop / active perception 机制，把校准值直接变成策略切换或安全护栏的触发信号。

## 🎯 可落地实验点

**实验设计**：给现有 VLA rollout 增加 success-confidence 头，并分别测试未校准、温度缩放、TD calibration 三种方案
- 对比基线：原始 VLA action probability、temperature scaling、TD calibration
- 度量指标：ECE/Brier score、任务成功率、失败前预警提前量、需要人工接管比例
- 预期结果：TD calibration 在长时序任务里能更早识别高风险状态，并降低“高置信失败”的次数
