# 📖 花火格式笔记

## 🎯 题目

Long-Horizon Manipulation via Trace-Conditioned VLA Planning

## 📝 三句摘要

1. **问题背景**：短视野 VLA 在长程操作里很容易因多步依赖和误差累积而失效，缺少轻量又闭环的任务进度表示。
2. **核心方法**：LoHo-Manip 用 task-management VLM 在每一步重预测“已完成/剩余子任务 + 2D visual trace”，再让执行器 VLA 条件化跟随该 trace，把长程决策拆成连续局部控制。
3. **关键结果**：方法不依赖脆弱的历史缓冲或手工恢复逻辑，就能在仿真与真实 Franka 上提升长程成功率、鲁棒性和 OOD 泛化。

## 💎 价值评估

- **🔬 研究价值**：它抓住了当前 VLA 最大痛点之一——长程任务的闭环重规划——而且不是简单堆更大模型，而是用显式 progress memory + trace guidance 做结构化补强。
- **🚀 实践价值**：这套“高层剩余计划 + 低层局部跟踪”的分层思路，很适合迁到无人机任务链、具身多步操作、甚至语言导航中的中间目标刷新。
- **📈 扩展潜力**：后续可与世界模型、动作 chunking、或主人自己的中间表示接口结合，把 trace 从 2D 点轨迹扩展到语义 waypoint / affordance map / 空间约束走廊。

## 🎯 可落地实验点

**实验设计**：给现有 VLA 增加“剩余计划 + trace”中间监督，验证其对长程组合任务鲁棒性的增益。
- 对比基线：原始短视野 VLA、仅文本子任务分解、仅视觉 trace 条件、LoHo-Manip 完整版
- 度量指标：长程成功率、单步恢复率、错误累积率、OOD 指令泛化成功率
- 预期结果：显式 remaining-plan 与 trace 联合条件能明显降低中途卡死和错误传播
