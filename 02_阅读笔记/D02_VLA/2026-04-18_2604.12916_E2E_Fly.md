# 📖 花火格式笔记

## 🎯 题目

E2E-Fly: An Integrated Training-to-Deployment System for End-to-End Quadrotor Autonomy

## 📝 三句摘要

1. **问题背景**：四旋翼端到端策略从仿真到真实迁移困难，缺乏统一的全栈训练-验证-部署pipeline，系统性 zero-shot transfer 难以复现。
2. **核心方法**：提出 E2E-Fly 框架，集成高性能仿真器（可微物理+RL支持）、两阶段验证（sim-to-sim + HIL）、低层控制接口 + sim-to-real alignment（系统辨识、域随机化、延迟补偿、噪声建模）。
3. **关键结果**：首个系统性地统一可微物理学习与四旋翼训练/验证/真实部署的工作；在6个端到端控制任务上验证，zero-shot sim-to-real 成功。

## 💎 价值评估

- **🔬 研究价值**：D01世界模型+D03空地迁移的交叉点，提供了"仿真→真实"迁移的系统性工程框架
- **🚀 实践价值**：主人四旋翼研究直接可用的训练-部署pipeline
- **📈 扩展潜力**：与主人已有 PX4/无人机控制经验结合，降低端到端策略的落地门槛

## 🎯 可落地实验点

**实验设计**：复现 E2E-Fly 的 sim-to-real pipeline 用于主人四旋翼平台
- 对比基线：传统 PID / 纯 RL 无对齐
- 度量指标：轨迹跟踪误差、扰动抵抗能力、zero-shot 成功率
- 预期结果：E2E-Fly pipeline 显著提升四旋翼端到端策略的真实迁移效果
