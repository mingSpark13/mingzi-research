# 📖 花火格式笔记

## 🎯 题目

Evolvable Embodied Agent for Robotic Manipulation via Long Short-Term Reflection and Optimization

## 📝 三句摘要

1. **问题背景**：通用机械臂智能体常停留在单轮感知-规划，缺少对历史经验的压缩反思，导致长程操作任务里适应性和恢复能力偏弱。  
2. **核心方法**：论文提出 EEAgent，用 VLM 解释环境、用长短期反思优化机制（LSTRO）动态改写提示，把短期经验与长期教训同时喂回操作规划。  
3. **关键结果**：该方法在多类机械臂任务上提升任务成功率，说明“可反思、可积累经验”的 embodied agent 比一次性 prompt 规划更适合持续操作场景。  

## 💎 价值评估

- **🔬 研究价值**：它把“记忆+反思+提示优化”系统化接入机器人操作闭环，适合作为主人研究具身 agent 自进化机制的参考样板。  
- **🚀 实践价值**：对机械臂长程任务、失败恢复、经验复用很有启发，后续也可迁到 UAV 任务规划里的历史策略修正。  
- **📈 扩展潜力**：如果把 LSTRO 从文本反思扩展到动作轨迹或状态抽象层，可以进一步连接 VLA/世界模型和持续学习。  

## 🎯 可落地实验点

**实验设计**：给现有操作 agent 加一个“失败后反思重规划”模块，验证是否能提升长程任务完成率
- 对比基线：单轮 VLM 规划、无长期记忆的短期反思 agent
- 度量指标：任务成功率、失败恢复次数、重复错误率、平均规划步数
- 预期结果：引入长短期反思后，长程任务成功率提升，重复犯错显著下降
