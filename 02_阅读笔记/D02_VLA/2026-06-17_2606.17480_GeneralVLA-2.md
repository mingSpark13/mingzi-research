# 📖 花火格式笔记

## 🎯 题目

GeneralVLA-2: Geometry-Aware Reconstruction and Governed Memory for Robot Planning

## 📝 三句摘要

1. **问题背景**：通用 VLA 系统需要 object-centric 3D 表征和可复用的操作经验来规划可靠轨迹，但 GeneralVLA v1 仍存在 3D 几何证据不足、规划经验难以管理两大瓶颈。
2. **核心方法**：提出 GeneralVLA-2，引入 (1) Geometry-Aware Reconstruction 模块从 RGB-D 重建 object-centric 3D 场景；(2) Governed Memory 机制管理可复用的操作经验（成功/失败轨迹库），让 VLA 在规划时检索相关历史经验。
3. **关键结果**：相比 GeneralVLA v1 在多个 long-horizon 任务上成功率提升明显，几何感知使对未见物体/视角的鲁棒性增强，governed memory 显著提升失败恢复能力。

## 💎 价值评估

- **🔬 研究价值**：⭐⭐⭐⭐
  - 「Object-centric 3D + Memory Bank」架构对论文A的"先验+轨迹复盘"模块是直接参考
  - Governed Memory 机制是 D05 数据飞轮方向 VLA 化的新范式
- **🚀 实践价值**：⭐⭐⭐⭐
  - 几何感知重建是空中场景关键（空中视野有强深度变化，3D 表征比 2D 重要得多）
  - Governed Memory 可直接迁移到「飞行操作经验库」做 D03 空地迁移
- **📈 扩展潜力**：⭐⭐⭐⭐
  - 论文A空中场景适配：3D 重建 → 空中深度感知；Memory Bank → 飞行操作经验
  - 与 D01 已有世界模型知识结合：可把 WM 的 rollout 当作虚拟经验注入 memory

## 🎯 可落地实验点

**实验设计**：借鉴 Governed Memory 机制构建「空中操作经验库」
- 对比基线：单条链推理的 VLA（无 memory）
- 度量指标：(1) 失败恢复成功率 (2) 跨任务经验迁移效率 (3) Memory 容量与检索速度
- 预期结果：在 5+ 种失败模式上，记忆机制比单条推理提升 ≥20%
- 关键改造：把"空中操作轨迹+飞行意图"作为 memory 单元，实现 D03 空地迁移的隐式迁移
