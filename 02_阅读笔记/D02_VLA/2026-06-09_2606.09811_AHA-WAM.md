# 📖 花火格式笔记

## 🎯 题目

AHA-WAM: Asynchronous Horizon-Adaptive World-Action Modeling with Observation-Guided Context Routing

## 📝 三句摘要

1. **问题背景**：现有 world-action model 往往把视频预测与动作执行绑在同一时间粒度，导致世界分支浪费算力去拟合短时冗余变化，长程操控反而建模不足。
2. **核心方法**：论文提出双 DiT 架构的 AHA-WAM，让低频视频分支充当长程 world planner、高频动作分支闭环执行 action chunk，并用 observation-guided context routing 在层间按需检索长期上下文。
3. **关键结果**：方法针对长时程操控里的时域错配给出结构性修补，核心价值不在小补丁提分，而在把“世界想象”和“动作执行”正式解耦成异步节奏，适合作为 VLA/世界模型混合路线的新骨架。

## 💎 价值评估

- **🔬 研究价值**：把 world model 从“同频辅助分支”推进到“低频规划器”，很贴合主人关注的世界模型+操控主线。
- **🚀 实践价值**：如果后续实验真能证明低频想象+高频执行稳定收益，这套结构可直接迁移到机械臂长程多阶段任务。
- **📈 扩展潜力**：很适合继续接 memory/imagination/safety filter，往可验证中间层与长程任务规划方向扩展。

## 🎯 可落地实验点

**实验设计**：在现有 manipulation policy 上复现“低频世界分支 + 高频动作分支”的异步调度
- 对比基线：同步 world-action model、纯 VLA、带历史帧缓存的 VLA
- 度量指标：长程任务成功率、每步延迟、遮挡/状态变化下恢复成功率
- 预期结果：异步 world planner 在不显著拉高控制时延的前提下，提高多阶段任务稳定性
