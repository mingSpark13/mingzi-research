# 📖 花火格式笔记

## 🎯 题目

**DUST (Dual-Stream Diffusion)** — 世界模型增强 VLA 的双流扩散

## 📝 三句摘要

1. **问题背景**：用世界模型增强 VLA（联合预测 next-state observations + action sequences）有前景，但视觉状态与动作**模态差异大**——统一联合扩散难做（模态冲突）。
2. **核心方法**：作者提出 **DUST**——**多模态扩散 Transformer**，**显式维持独立的模态 stream**（vision stream + action stream），同时通过**跨模态知识共享**实现双向信息流。用**独立噪声扰动+解耦 flow matching loss**让模型学 joint distribution，并支持**异步采样**（action/vision tokens 不同速率）。
3. **关键结果**：在 RoboCasa、GR-1 等仿真基准上，比 SOTA VLA + world-modeling baselines 提升 **6%**；inference-time scaling 再提供 **2-5%** 增益。

## 💎 价值评估

- **🔬 研究价值**：把"模态解耦 + 跨模态共享"的双流设计用在 VLA + 世界模型联合训练上，是该方向的方法学创新。
- **🚀 实践价值**：对 AirSpark 未来"VLA+世界模型联合"路线有直接借鉴。
- **📈 扩展潜力**：双流+异步采样可与 AHEAD future-token wrapper 互补。

## 🎯 可落地实验点

**实验设计**：在 AirSpark 仿真器中复现 DUST 的双流 diffusion + 异步采样，对接 UAV 导航任务。
- **场景**：AirSpark 园区任务
- **基线**：单流联合 diffusion / 纯 VLA 无世界模型
- **度量**：任务成功率、world model 预测质量、异步采样速度增益
- **预期**：双流 + 异步比单流联合在 6% 任务成功率上接近 DUST 报告
