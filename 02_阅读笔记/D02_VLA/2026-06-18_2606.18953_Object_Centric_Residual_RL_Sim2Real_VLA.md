# 📖 花火格式笔记

## 🎯 题目

Object-Centric Residual RL for Zero-Shot Sim-to-Real VLA Enhancement

## 📝 三句摘要

1. **问题背景**：通用 VLA 模型（Pi0 / RT-2 / OpenVLA 等）虽然在泛化任务上很强，但在 sim-to-real 部署时仍有显著的性能跌落；纯 RL 训练在分布外又难以泛化，二者各有瓶颈。
2. **核心方法**：提出 Object-Centric Residual RL——把 VLA 的输出当作 base policy，在仿真里学一个**物体中心的残差策略**；residual 只在 VLA 不确定或失败时启动，主体仍由 VLA 决策，从而兼顾 VLA 的泛化能力与 RL 的精确执行。
3. **关键结果**：在仿真训练后零样本部署到真机，VLA + residual 的组合在多个操作任务上明显优于单独 VLA 和单独 RL；residual 的"物体中心"分解让策略对场景变化的鲁棒性显著提升。

## 💎 价值评估

- **🔬 研究价值**：把"residual RL + VLA base policy + 物体中心分解"三件事组合到一起，是对当前 VLA 部署栈的实用补强；零样本 sim-to-real 在 RL 领域是难啃的方向，能成功验证非常有意义。
- **🚀 实践价值**：对主人 D07（Isaac RL 机械臂）+ D02（VLA）方向直接相关——可以用 Isaac Lab 在仿真里直接训练 residual，再用 VLA 做 base policy；这种"主策略 + 残差"的范式天然契合"安全层"的定位。
- **📈 扩展潜力**：residual RL 思路与 World Pilot 的 WAM safety steering 在功能上重合（都是"VLA 之外加一层安全/精确度补丁"）；两者未来可能在主人论文 A 的"安全层"模块里融合——VLA + WAM safety + object-centric residual 的三层架构是值得探索的。

## 🎯 可落地实验点

**实验设计**：在 Isaac Lab 中构建"VLA (OpenVLA) + Object-Centric Residual RL"对比实验，验证 zero-shot sim-to-real 的提升。
- 对比基线：纯 VLA / 纯 PPO / VLA + 经典 residual（不做物体中心分解）/ VLA + object-centric residual
- 度量指标：仿真到真机的成功率保持率（%）、分布外物体成功率、residual 触发频率（统计 VLA 自身表现）
- 预期结果：object-centric residual 在零样本 sim-to-real 上比无分解 residual 高 15-25%；残差触发频率随 VLA 训练程度下降（说明确实在补 VLA 的短板）
