# 📖 花火格式笔记

## 🎯 题目

RLDX-1 Technical Report

## 📝 三句摘要

1. **问题背景**：现有 VLA 虽然继承了视觉语言模型的泛化能力，但在真实复杂操作里仍缺运动感知、记忆决策、物理感知与实时落地能力。
2. **核心方法**：RLDX-1 提出 Multi-Stream Action Transformer，把异构模态拆成独立 stream 再做 cross-modal joint self-attention，同时叠加稀有场景数据合成、针对人形/灵巧操作的训练流程和实时推理优化。
3. **关键结果**：在仿真和真实任务上持续超过 π0.5 与 GR00T N1.6，尤其在 ALLEX humanoid tasks 上达到 86.8% 成功率，明显高于约 40% 的对照模型。

## 💎 价值评估

- **🔬 研究价值**：它不是只换个 action head，而是把“多模态能力拼装 + 系统级工程优化”打包成可对标前沿闭源 VLA 的完整路线。
- **🚀 实践价值**：对主人后续做高自由度操作、具身多模态输入或人形/双臂方向很有参考意义，尤其是 modality-specific stream 的组织方式。
- **📈 扩展潜力**：值得继续追踪其训练数据构成、实时推理技巧与是否开放代码权重，这会直接影响能否复现实用级 VLA。

## 🎯 可落地实验点

**实验设计**：复现 multi-stream 多模态 action transformer 在高自由度操作任务中的收益
- 对比基线：单流 VLA、标准 VLM+action head、π0.5 风格双系统策略
- 度量指标：任务成功率、控制频率、长时记忆依赖任务表现、接触密集任务稳定性
- 预期结果：分流建模 + 跨模态 joint attention 在高 DoF 接触任务上明显优于单一表征融合
