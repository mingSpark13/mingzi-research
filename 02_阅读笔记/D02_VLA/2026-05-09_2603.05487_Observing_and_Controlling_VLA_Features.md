# 📖 花火格式笔记

## 🎯 题目

Observing and Controlling Features in Vision-Language-Action Models

## 📝 三句摘要

1. **问题背景**：VLA 比 LLM 多了多模态输入输出和 diffusion/transformer 混合头，导致 LLM 里的机理可解释性方法不能直接照搬到机器人控制。
2. **核心方法**：论文提出 feature-observability 与 feature-controllability 两个概念，先用线性分类器观测表示空间中的特征，再用基于最优控制的轻量线性干预把内部表示推向目标区域，从而在线调控行为。
3. **关键结果**：在 π0.5 和 OpenVLA 仿真测试里，不用微调就能稳定改变机器人输出行为，同时保持闭环能力，说明 VLA 内部已经存在可被实时利用的可控结构。

## 💎 价值评估

- **🔬 研究价值**：这篇把“VLA 可解释性”从分析层推进到干预层，给后续做安全约束、偏好对齐、失败修正提供了非常硬的接口。
- **🚀 实践价值**：对主人做机器人系统时特别有用，因为它说明不用重新训练大模型，也可能通过中间特征干预实现在线风格/策略修正。
- **📈 扩展潜力**：后续很适合结合 UAV 或跨载体控制，把表示层 steerability 做成任务偏好注入、安全护栏或人机协作反馈接口。

## 🎯 可落地实验点

**实验设计**：在现有 OpenVLA 类控制器上增加线性 probe + 表示干预模块，测试是否能在线修正偏航、抓取姿态或保守程度
- 对比基线：原始 VLA、仅 probe 不干预、全量微调方法
- 度量指标：任务成功率、干预延迟、偏好对齐误差、闭环稳定性
- 预期结果：轻量干预能以更低代价实现行为修正，并保留原策略的泛化能力
