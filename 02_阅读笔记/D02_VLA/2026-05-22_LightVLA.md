# 📖 花火格式笔记

## 🎯 题目

The Better You Learn, The Smarter You Prune: Towards Efficient Vision-language-action Models via Differentiable Token Pruning

## 📝 三句摘要

1. **问题背景**：VLA（视觉-语言-动作）模型推理计算成本高，大量视觉 token 中存在大量冗余，但现有方法难以在保持性能的同时实现高效推理。
2. **核心方法**：提出 LightVLA，一种**可微分 token 剪枝框架**，通过动态查询评估视觉 token 重要性，采用 Gumbel softmax 实现可微分的 token 选择，核心版本不引入额外可训练参数。
3. **关键结果**：在真实机器人任务上，LightVLA 实现 FLOPs 降低 **59.1%**、延迟降低 **38.2%**，同时任务成功率提升 **2.9%**。

## 💎 价值评估

- **🔬 研究价值**：首次将**自适应视觉 token 剪枝**引入 VLA 任务，实现性能与效率的双重提升；Gumbel softmax 方案为可微分 token 选择提供了新思路。
- **🚀 实践价值**：延迟降低 38.2% 对实时机器人系统意义重大，使 VLA 在消费级 GPU 上的实时部署成为可能。
- **📈 扩展潜力**：可与 OpenVLA、π0 等 VLA 基座结合；token 重要性评估机制可迁移至其他多模态架构。

## 🎯 可落地实验点

**实验设计**：在现有 VLA 模型（OpenVLA 或 π0-7B）上接入 LightVLA token 剪枝模块，验证在不同机械臂任务上的效率-性能 trade-off。

- **对比基线**：原始 VLA（无剪枝）、训练后静态剪枝（magician number）
- **度量指标**：任务成功率（CALVIN ABCD）、FLOPs、推理延迟（ms/step）
- **预期结果**：在保持或提升任务成功率的同时，延迟显著降低
