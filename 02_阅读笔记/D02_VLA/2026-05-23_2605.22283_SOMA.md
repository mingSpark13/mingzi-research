# 📖 花火格式笔记

## 🎯 题目

Spatial Memory for Out-of-Vision Manipulation in Vision-Language-Action

## 📝 三句摘要

1. **问题背景**：现有 VLA 默认关键目标始终在视野内，一旦目标落到相机视锥外，就会出现反应式、低效的搜索与抓取失败。
2. **核心方法**：论文提出 SOMA，通过可转动头部相机主动扫描、多视角构建持久空间记忆，并用动态记忆修正与上下文检索把 instruction relevant 空间线索送回 manipulation policy。
3. **关键结果**：在 5 个真实 out-of-vision manipulation 任务以及 RoboCasa GR1、SimplerEnv 上都显著提升成功率与目标定位效率，并被 ICML 2026 接收，说明“可见范围外记忆”已经是 VLA 补短板的关键接口。

## 💎 价值评估

- **🔬 研究价值**：这篇把“视野外操作”从数据问题变成结构问题，补上了 VLA 在部分可观测场景下最常见也最致命的一块短板。
- **🚀 实践价值**：很适合给主人后续做主动转头、主动视角调整、甚至 UAV/机械臂联合观测时，提供一个可复用的 memory + retrieval 框架。
- **📈 扩展潜力**：后续可以把 SOMA 的空间记忆接到世界模型、语义地图、甚至空中导航任务里，变成“看不见也能继续推理”的统一基础模块。

## 🎯 可落地实验点

**实验设计**：给现有 VLA 控制器加一个轻量 spatial memory buffer，并加入 head-scan 或主动转视角动作，验证遮挡与视野外目标任务的恢复能力
- 对比基线：普通单视角 VLA、只拼接历史帧的 temporal VLA、SOMA 式显式空间记忆
- 度量指标：任务成功率、目标首次定位时间、无效视角搜索步数、one-shot grasp 成功率
- 预期结果：显式空间记忆在目标初始不可见与多步操作场景下显著优于纯历史帧聚合
