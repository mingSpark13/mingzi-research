# 📖 WorldVLA: Towards Autoregressive Action World Model

## 🎯 题目

WorldVLA: Towards Autoregressive Action World Model

## 📝 三句摘要

1. **问题背景**：现有 VLA 模型存在动作生成与视觉理解分离的问题，world model 和 action model 各自独立工作，缺乏统一的框架
2. **核心方法**：提出 WorldVLA，一个自回归的 Action World Model，统一了动作和图像的理解与生成，并提出 action attention masking 策略解决自回归误差累积问题
3. **关键结果**：在 LIBERO 基准上抓取成功率提升 4%，FVD（Fréchet Video Distance）降低 10%，action chunk 生成任务中 attention masking 策略带来 4%-23% 的改进

## 💎 价值评估

- **🔬 研究价值**：提出了统一 world model 和 action model 的新范式，为 VLA 架构演进提供了新思路；action attention masking 是解决自回归动作生成误差累积的有效方法
- **🚀 实践价值**：统一框架降低了系统复杂度，attention masking 策略可直接应用于其他自回归 VLA 模型
- **📈 扩展潜力**：可与最新的 diffusion-based 世界模型结合，探索 hybrid 架构；attention masking 策略可推广到其他长序列动作生成任务

## 🎯 可落地实验点

**实验设计**：在现有 VLA 模型上验证 attention masking 策略效果
- 对比基线：标准自回归 VLA（如 OpenVLA）、vanilla world model
- 度量指标：任务成功率、FVD、动作序列一致性（ATE/APE）
- 预期结果：attention masking 应能显著提升长序列任务的成功率，缩短的动作 chunk 表现更稳定
- 可在 ALTER 机器人上用 10-20 条演示数据做 quick experiment
