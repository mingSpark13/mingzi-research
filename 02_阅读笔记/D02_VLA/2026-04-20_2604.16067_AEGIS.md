# 📖 花火格式笔记

## 🎯 题目

AEGIS: Anchor-Enforced Gradient Isolation for Knowledge-Preserving Vision-Language-Action Fine-Tuning

## 📝 三句摘要

1. **问题背景**：VLM/VLA 在用 flow-matching 动作头做机器人控制微调时，连续动作梯度会快速破坏原本的视觉问答语义能力，出现严重灾难遗忘。
2. **核心方法**：AEGIS 用静态高斯 reference anchor + Wasserstein-2 恢复梯度 + 分层 Gram-Schmidt 正交投影，把任务梯度从“破坏语义流形”的方向上掰开，同时保留大部分有效监督。
3. **关键结果**：方法几乎不依赖 replay buffer 或协同训练数据，平均仅丢弃不到 1% 梯度能量，却能显著抑制 VQA 能力塌缩，适合把预训练 VLM 更稳地推到机器人控制场景。

## 💎 价值评估

- **🔬 研究价值**：这篇很贴 D02 当前“π0/flow matching 上控制微调会不会把语义能力训坏”的核心矛盾，给了一个比 stop-gradient / LoRA 更机制化的解法。
- **🚀 实践价值**：如果明子后面做 VLA 架构切换或动作头替换，AEGIS 可直接变成训练防遗忘护栏，尤其适合小数据、高连续控制梯度场景。
- **📈 扩展潜力**：可继续外推到 packet-first / verifier-first 体系里，测试是否能把高层语义规划与低层连续动作学习更稳定地绑在一起。

## 🎯 可落地实验点

**实验设计**：在现有 VLM/VLA 微调链路里加入 AEGIS 风格的层间正交梯度投影，验证控制收益与语义保持能否同时成立。
- 对比基线：stop-gradient、LoRA、无保护直接微调
- 度量指标：任务成功率、VQA 保留率、动作 loss、训练稳定性
- 预期结果：AEGIS 风格保护项在不明显拖慢收敛的情况下，显著减轻语义遗忘并提升迁移可复用性
