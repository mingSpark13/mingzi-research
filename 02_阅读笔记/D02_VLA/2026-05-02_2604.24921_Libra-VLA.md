# 📖 花火格式笔记

## 🎯 题目

Libra-VLA: Achieving Learning Equilibrium via Asynchronous Coarse-to-Fine Dual-System

## 📝 三句摘要

1. **问题背景**：传统 VLA 常把语义规划、动作生成和细粒度控制“一锅炖”进同一个高频模型里，导致长程任务、精细操作和实时部署三者之间难以兼顾，训练与推理都容易失衡。
2. **核心方法**：Libra-VLA 提出异步 coarse-to-fine 双系统架构，把低频语义规划器和高频动作精修器解耦，前者负责给出宏观方向，后者负责细节动作修正，从而实现更稳定的层次化决策与执行。
3. **关键结果**：在 LIBERO 基准上，Libra-VLA 平均成功率达到 **97.2%**，在 precision-critical 的 Object 任务上达到 **99.4%**，在 long-horizon 任务上达到 **92.8%**，整体显著超过多项强基线。

## 💎 价值评估

- **🔬 研究价值**：这篇工作的最大价值是明确提出 VLA 不必再把所有尺度的决策混在同一个系统里做，而是通过“双系统分工”让语义规划与几何动作优化各司其职，这对具身大模型架构设计是个很强的信号。
- **🚀 实践价值**：对真实机器人部署非常友好，因为低频 planner 可以稳定给方向，高频 refiner 专注修局部误差，这比单一路径更容易兼顾长程任务和精细控制，也更符合真实控制系统的工程结构。
- **📈 扩展潜力**：这套思路对主人的无人机人体跟随也很有启发，例如低频模块可以负责目标意图与大方向跟随，高频模块负责 uv / distance 的细粒度稳定修正，未来甚至可以把 bucket-aware uncertainty 也并入高频修正器里。

## 🎯 可落地实验点

**实验设计**：把 Libra-VLA 的双系统思想迁移到无人机人体跟随
- **低频系统**：根据历史轨迹和场景上下文给出大方向跟随目标，比如保持视角、距离区间、转向趋势
- **高频系统**：根据当前 RGB/silhouette + pitch + velocity 实时修正 `target_uv` 与 `target_distance`
- **对比基线**：单模型直接输出控制目标 vs 双系统分层输出
- **度量指标**：distance MAE、uv MAE、跟丢率、长时稳定性、极端遮挡/截断场景恢复能力
- **预期结果**：双系统架构在长时跟随、边缘截断、视角急变场景下更稳定

## 📌 待探索问题

- 对无人机人体跟随来说，是否可以设计一个低频“目标状态规划器”和一个高频“视觉距离精修器”，分别负责方向与细节，而不是让单模型硬吃所有频段？
