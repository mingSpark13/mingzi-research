# 📖 花火格式笔记

## 🎯 题目

Rule-Based High-Level Coaching for UAV with Online Goal-Conditioned RL

## 📝 三句摘要

1. **问题背景**：纯端到端 RL 控制 UAV 在多目标配送、移动目标跟随等复杂任务中样本效率低，且难以保证硬约束（如禁飞区、碰撞边界）；纯规则方法又缺乏适应性，在场景变化时容易失败。
2. **核心方法**：提出层次化 UAV 决策框架，高层用规则化任务规范（rule-based coaching）将复杂任务分解为可执行的子目标序列，低层用在线 goal-conditioned RL 控制器根据当前状态自适应执行，避免了纯 RL 的盲目探索和纯规则的结构僵硬。
3. **关键结果**：在多目标配送、动态目标跟随、障碍物规避等场景中，该框架相比纯 RL 基线减少碰撞终止 60%+，样本效率提升 3-5 倍，且规则层无需训练即可迁移到新任务。

## 💎 价值评估

- **🔬 研究价值**：直接对标主人的 UAV Phase2 人体跟随项目，高层规则+低层 RL 的层次化设计思路可借鉴；goal-conditioned RL 作为低层执行器保证了策略的泛化性，规则层又提供了任务级结构，是很好的设计范式。
- **🚀 实践价值**：规则层无需训练且可解释，适合作为安全护栏；低层 RL 在线适应，可处理人体跟随中的快速姿态变化；两者结合比纯学习或纯规则都更稳健。
- **📈 扩展潜力**：规则 coaching 层可扩展为 LLM 生成的自然语言任务规范；低层 RL 可替换为更高效的离线预训练+在线适应范式（如 π0-RL）；与感知模块结合可形成完整的感知-决策-控制链。

## 🎯 可落地实验点

**实验设计**：在主人现有 UAV 人体跟随项目中引入"规则 coaching + 在线 RL"的层次化架构，高层规则根据人体运动意图（如方向变化、速度变化）生成子目标，低层 RL 根据无人机当前状态和子目标生成控制动作，对比端到端 RL、纯规则和层次化方法的表现。
- 对比基线：端到端 RL 跟随、纯规则跟随（PID 或行为树）、纯 RL 在线学习、层次化规则+RL
- 度量指标：跟随误差（ATE）、碰撞率、样本效率、任务完成率、对快速转向的适应性
- 预期结果：层次化方法在碰撞率降低 50%+ 的同时维持相近的跟随精度，样本效率比纯 RL 提升 3-5 倍
