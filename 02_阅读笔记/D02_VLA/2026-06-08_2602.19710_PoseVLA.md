# 📖 花火格式笔记

## 🎯 题目

Universal Pose Pretraining for Generalizable Vision-Language-Action Policies

## 📝 三句摘要

1. **问题背景**：现有 VLA 常把高层语义感知和 embodiment-specific 动作监督纠缠在一起，导致 3D 状态差异不敏感、训练效率低。
2. **核心方法**：Pose-VLA 先在统一相机坐标系里用离散 pose token 学通用 3D 空间先验，再做机器人动作空间对齐，把 3D 数据和示教轨迹接到同一套表示里。
3. **关键结果**：在 RoboTwin 2.0 上达到 79.5% 平均成功率、在 LIBERO 上达 96.0%，且真实机器人每任务仅 100 条示教也能对多种物体稳健泛化。

## 💎 价值评估

- **🔬 研究价值**：这篇把 VLA 的“语义强、几何弱”问题正面拆开处理，用 pose token 做中间统一接口，对主人关心的感知—动作解耦很有参考价值。
- **🚀 实践价值**：如果后续要做 UAV/机械臂跨平台策略复用，先学共享 3D 空间表示、再做 embodiment 对齐，比直接端到端硬训更稳。
- **📈 扩展潜力**：可进一步接到世界模型或 PMI 接口上，把 pose token 当作可验证中间层，探索空地平台统一控制表征。

## 🎯 可落地实验点

**实验设计**：在主人现有机器人/无人机数据上验证“pose token 预训练 + 下游动作对齐”是否优于纯端到端 VLA
- 对比基线：OpenVLA / π0.5 / Octo
- 度量指标：少样本成功率、跨物体泛化、跨平台迁移稳定性
- 预期结果：有 pose 先验的策略在 100 条以内示教时收敛更快，且对几何状态变化更稳健
