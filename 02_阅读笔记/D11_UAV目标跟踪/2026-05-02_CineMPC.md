# 📖 花火格式笔记

## 🎯 题目

CineMPC: A Fully Autonomous Drone Cinematography System Incorporating Zoom, Focus, and Camera Motion Control

## 📝 三句摘要

1. **问题背景**：传统无人机跟拍只关注"跟上目标"，忽略了摄影艺术要求（构图、焦距、景深等），无法满足专业电影摄影需求
2. **核心方法**：提出 CineMPC 框架，将相机内参（zoom、focus、depth-of-field）纳入非线性 MPC 闭环控制，用 RGB-D 估计目标相对姿态，实时优化满足用户指定拍摄目标的轨迹
3. **关键结果**：在真实无人机上验证，能够稳定执行多种电影镜头（orbit、dolly zoom、focus pull 等），构图误差 < 5%，焦点误差 < 10cm

## 💎 价值评估

- **🔬 研究价值**：首次将相机内参控制与无人机运动控制统一到 MPC 框架，从"目标跟踪"升级到"摄影任务"，为 UAV cinematography 提供了系统化方法论
- **🚀 实践价值**：真实无人机部署验证，支持多种专业镜头语言（orbit、dolly zoom、focus pull），可直接用于影视拍摄、体育赛事、活动记录等场景
- **📈 扩展潜力**：可与 VLA/VLM 结合，实现语言指令驱动的智能摄影（"给我一个低角度特写镜头"），或与 shot-conditioned policy 结合做端到端学习

## 🎯 可落地实验点

**实验设计**：shot-conditioned UAV following with CineMPC-style control
- **对比基线**：
  - 纯视觉跟踪（只保持目标在画面中心）
  - 固定焦距跟拍（无 zoom/focus 控制）
  - CineMPC（本文方法）
- **度量指标**：
  - Framing error（构图误差，目标是否在期望画面位置）
  - Distance error（跟拍距离误差）
  - Focus error（焦点误差）
  - Trajectory smoothness（轨迹平滑度）
  - Shot type success rate（镜头类型完成率）
- **预期结果**：CineMPC 在构图精度、焦点控制、镜头多样性上显著优于 baseline，证明"摄影控制"比"单纯跟踪"更符合实际需求
