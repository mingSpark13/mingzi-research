# DreamTIP: Learning Task-Invariant Properties via Dreamer

> **方向**: D01_世界模型 | **入库时间**: 2026-04-07 | **轮次**: R502
> **arXiv**: 2604.02911 | **作者**: (四足机器人方向) | **日期**: 2026-04-03

## 一句话摘要
将 LLM 引导的「任务不变属性」（接触稳定性、地形间隙）融入 Dreamer 世界模型作为辅助预测目标，使四足机器人策略学到对底层动态变化不敏感的表示，实现鲁棒 sim-to-real 迁移。

## 核心贡献
1. **DreamTIP 框架**：在 Dreamer 架构内集成 Task-Invariant Properties (TIP) 学习
2. **LLM 引导发现 invariant properties**：用 LLM 从任务描述中自动识别跨任务稳定的物理属性（接触稳定性、地形间隙等）
3. **辅助预测目标**：将这些属性作为世界模型的额外预测目标，驱动学到不变表示
4. **高效适应策略**：混合回放缓冲区 + 正则化约束，快速校准真实世界动力学同时防止表示崩溃

## 技术细节
- **世界模型**: Dreamer (基于 RSSM 的 latent world model)
- **TIP 发现**: LLM 指导，从任务描述自动提取
- **辅助目标**: 接触稳定性 + 地形间隙作为世界模型额外 losses
- **适应**: 混合回放缓冲区 + 表示正则化

## 实验结果
- 8个仿真迁移任务平均提升 **+28.1%**
- 真实世界 Climb 任务：**10% → 100%** 成功率
- 地形: Stair, Climb, Tilt, Crawl

## 与本方向的关系
- **直接支撑 D01**：Dreamer 世界模型 + 跨载体迁移的工程化路线
- **可借鉴到空中机械臂**：payload-invariant / wind-invariant properties 可作为类似辅助目标

## 可借鉴的技术点
1. LLM 引导自动发现任务不变属性 → 可迁移到主人的无人机/空中机械臂项目
2. 辅助预测目标 + 混合回放缓冲区的适应策略 → 直接可作为 sim-to-real pipeline

## 局限/不足
- 目前验证于四足机器人，空中平台（无人机+机械臂）的动态特性差异大，需验证
- LLM 引导的 property 发现质量依赖 LLM 能力
