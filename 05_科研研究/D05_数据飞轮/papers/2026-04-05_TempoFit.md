# TempoFit: VLA 免训练时间记忆改造
> 方向：06_其他方向（归入05_数据采集/数据合成） | 入库时间：2026-04-05 | R465

## 论文信息
- **标题**: TempoFit: Plug-and-Play Layer-Wise Temporal KV Memory for Long-Horizon Vision-Language-Action Manipulation
- **arXiv**: 2603.07647
- **发表**: 2026-03-08
- **作者**: Jun Sun, Boyu Yang, Jiahao Zhang 等

## 一句话摘要
对冻结的预训练 VLA，通过复用中间层 prefix attention K/V 作为状态级记忆，实现训练无关的时间上下文注入，在 LIBERO-LONG 长期任务上提升 +4.0% 成功率。

## 核心贡献
1. **训练无关**：不改变 VLA 权重，只用 KV 缓存实现历史注入
2. **Layer-wise FIFO 记忆**：在选定的中间层存储 prefix K/V，跨步复用
3. **Frame-Gap Temporal Bias (FGTB)**：固定 recency bias，保持决策当前主导
4. **Norm-preserving Rescaling**：注入检索上下文时避免冻结权重下的分布漂移

## 与主人研究的相关度
- **高**：主人龙虾项目的 VLA 需要在长期任务（"找到一辆单车"导航 + 机械臂操作）中保持历史一致性
- TempoFit 的免训练特性意味着主人可以直接在现有 VLA（如 DIAL/FocusVLA）上叠加，无需重新微调
- 对 LIBERO-LONG 提升 +4.0% 表明时间记忆对长程操作任务的关键价值

## 可落地实验点
1. 在主人现有 VLA（DIAL/FocusVLA）上叠加 TempoFit 改造，测试龙虾长期任务
2. 结合 EgoSim（已有库）的跨载体迁移能力，验证 VLA 时间记忆的跨平台泛化
3. 结合 TempoFit 的 KV 重用思想，优化龙虾项目的多轮对话/记忆接口设计
