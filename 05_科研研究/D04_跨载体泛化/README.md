# D04: 跨载体泛化 (Cross-Embodiment Generalization)

> 核心问题：如何让一个策略/模型泛化到不同机器人形态、不同物体、不同场景

## 研究背景

现有具身智能策略通常为特定机器人训练，更换硬件平台（如地面机械臂→空中机械臂）需要重新采集数据和训练。这严重制约了具身智能的通用化。本方向聚焦：统一表示、跨平台迁移、少样本/零样本适配。

## 核心研究问题

1. 动作空间统一表示（FAAS/CEI）是跨载体的关键吗？还是功能对齐（force/contact）更重要？
2. 视觉表征的泛化 vs 动作策略的泛化，哪个是真正的瓶颈？
3. 跨载体数据的质量（paired vs unpaired）如何影响迁移效果？
4. 世界模型路线（DreamZero WAM）能否从根本上绕过动作空间对齐问题？
