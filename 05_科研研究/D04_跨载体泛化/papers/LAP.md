# LAP: Language-Action Pre-Training Enables Zero-shot Cross-Embodiment Transfer

> **arXiv**: 2602.10556 | **方向**: D04_跨载体泛化 | **深挖时间**: 2026-04-08 R536 | **状态**: ✅ 已入库

## 一句话与本方向关系
LAP提出将低层动作转化为自然语言监督VLM backbone，实现首个强零样本跨载体VLA（LAP-3B），是动作空间统一表示的全新范式。

## 核心贡献
1. **语言-动作预训练（LAP）**：将低层机器人动作表示为自然语言（如"close gripper"、"move left"），用它监督VLM backbone训练，统一了action learning和VQA任务
2. **LAP-3B**：首个在未见载体上展现强零样本迁移能力的VLA
3. **关键洞察**：语言作为动作表示天然具备跨载体泛化能力——语义层面对齐，而非几何层面对齐

## 技术路线
```
动作文本化 → VLM预训练(统一视觉-语言-动作) → 轻量扩散action expert → 实时控制
核心洞察：语言是跨载体的"元语言"
```

## 可借鉴的技术点
1. **动作文本化作为中间表示**：将具体载体动作转为语义语言描述，绕开几何异构性；主人Paper A的高层VLM意图层可考虑类似语义对齐
2. **VLM + 轻量action expert架构**：大模型学语义理解，小模型学低层控制；适合主人的层次化VLA架构设计
3. **多任务VQA统一预训练**：用VQA任务增强语言-动作对齐的泛化性

## 局限/不足（我们可改进的点）
1. **语言动作的粒度问题**：自然语言动作描述粒度粗，难以描述精细快速运动；与FAAS/CEI的细粒度几何表示结合可能是改进方向
2. **未见载体仍需适配**：零样本不等于完全泛化，新载体形态差异过大时仍有退化
3. **依赖大规模预训练数据**：语言-动作配对数据的收集成本高

## 核心创新点机会
**C: 语言+几何双模态动作表示**：LAP的语言动作层负责语义泛化，CEI的几何轨迹层负责精度保证——双层接口可能是比单纯语言或几何更优的路线
