- **核心贡献**: 在开放式环境中从导航任务涌现出具身世界模型，证明世界模型可以从任务导向的交互中自发产生
- **相关度**: ⭐⭐⭐ 与龙虾/无人机导航的世界模型构建高度相关
- **PDF**: 待下载

### 2. An Embodied Aerial Agent for Natural Human-Drone Interaction
- **arXiv**: 2601.13801
- **核心贡献**: 具身空中智能体，实现自然的人-无人机语言交互， onboard AI 模型实时推理
- **相关度**: ⭐⭐⭐ 与龙虾 VLN 任务和 UE 数据采集场景设计直接对齐
- **PDF**: 待下载

### 3. Aerial World Model for Long-horizon Visual Forecasting (ANWM)
- **arXiv**: 2512.21887
- **核心贡献**: ANWM 在长距离视觉预测上显著优于现有世界模型，并提升无人机导航
- **相关度**: ⭐⭐⭐⭐ ANWM 直接面向无人机长距离预测，与龙虾/UE 项目高度相关
- **PDF**: 待下载

### 4. EmbodiedFly: Embodied LLM Agent with Autonomous Reconfigurable Drone
- **来源**: ACM Digital Library
- **核心贡献**: 将基础模型 pipeline 与可重构无人机结合，实现具身 LLM Agent
- **相关度**: ⭐⭐⭐ 平台级工作，与 UE AirSim 数据采集管线有一定关联

### 5. Carla-Air: Air-Ground Unified Infrastructure
- **arXiv**: 2603.28032
- **核心贡献**: 统一空中-地面仿真的基础架构，支持空中导航 + 视觉语言任务
- **相关度**: ⭐⭐⭐ 与 UE 数据采集场景设计和龙虾仿真平台选择相关
- **PDF**: 待下载

## D04 跨载体泛化 · 新发现
