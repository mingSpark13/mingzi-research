## 关键发现

1. **D05 world model backbone 进入 "memory + spatial" 双加固阶段**：Mem-World 加 memory token 解决时序漂移，R2RDreamer 加 3D-aware factor 解决空间漂移——两条线刚好覆盖 world model rollout 的两个最常见 failure mode。下一步把两者接进同一 pipeline 是非常自然的方向，也是 D05 §3.4 Recovery-Centric Data Flywheel 想要的"replay 时既稳又泛"的 backbone。
2. **D05 augmentation 体系补齐"空间"维度**：之前 D05 已有 CRAFT (video diffusion 增强时序) + ShapeGen (object shape 增强) + EgoGuide (采集端质量)，本轮 R2RDreamer 补上 3D-aware 空间增强——四者合起来就是一套完整的 demonstration augmentation stack。
3. **D03 本轮 0 新增的启示**：arxiv 端近期空地迁移 / sim-to-real 方向的"新方法"大量已被调度器记录（2606.18955 / 2606.18953 / 2606.17846 / 2606.17520 等），本轮扫描的 10 条候选里有 5 条已在库，说明 D03 主线方向最近的产出被高效吸收。下轮若继续 D03，建议重点关注"deployment-side ground-to-air protocol" / "空中特定扰动迁移验证"这类 reviewer-facing 协议类工作，而非纯方法论文。
