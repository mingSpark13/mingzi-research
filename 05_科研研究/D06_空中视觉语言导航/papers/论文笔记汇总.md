# D06 空中视觉语言导航 论文深挖笔记汇总

> R530 (2026-04-08) 首次扫描建库

## Aerial VLN 统一框架 (2512.08639) ⭐⭐⭐
- 统一框架处理Aerial Vision-and-Language Navigation
- 自然语言指令 + 城市环境 onboard visual observation
- 应用于：低空检查、搜救、自动空中投递
- 现有方法痛点：panoramic images 依赖，3D空间推理弱
- **可落地**：龙虾语言导航框架可直接参考其框架设计

## STMR-VLN (Semantic-Topo-Metric Relations) ⭐⭐⭐
- 语义-拓扑-度量三重关系建模
- 处理户外 aerial 场景的复杂空间关系
- **可落地**：与D01世界模型结合，建3D语义拓扑地图

## Topo-AeroVLN (OpenReview) ⭐⭐⭐
- 认知拓扑映射，brain-inspired 框架
- top-down 视角（第一人称顶视）构建多级拓扑地图
- 将 aerial observations 抽象为 road-bounded regions + 内部语义物体
- **可落地**：龙虾导航的拓扑建图可参考其分层表示思路

## AerialVLN (ICCV 2023) ⭐⭐
- 城市级UAV VLN数据集，city-level navigation
- 填补空中VLN研究空白
- 揭示天空vs地面导航的关键差异：6-DoF/高速/俯仰视角/动力学约束
- **可落地**：龙虾 VLN 评测可参考其数据集构建方式

## Awesome-Aerial-VLN (GitHub) 📚
- 空中VLN论文/资源汇总库（zhangchen98维护）
- 持续更新，是找最新工作的好来源
- **建议**：定期扫描该库补充 D06 论文地图

## GoalSwarm (2603.12908) ⭐⭐
- 多无人机开放词汇目标导航，去中心化协同搜索
- 轻量2D语义占据图 + Bayesian Value Map + UCB frontier
- **可落地**：龙虾单机版可先吸收“目标热图记忆 + 不确定性探索”中层模块，多机版再接竞价分工

## QuadAgent (2604.02786) ⭐⭐⭐ (R567新)
- training-free 异步多 agent 四旋翼系统，Foreground/Background agent 分工推理
- Impression Graph 用稀疏关键帧维护轻量场景记忆，配合 vision-based obstacle avoidance
- **可落地**：适合作为龙虾后续低时延 runtime 架构参考，不替代 APEX + AirNav 论文主线
