# 40_工作开发 🛠️

此目录管理个人工作开发相关的知识库、技术笔记和项目文档。

## 📋 目录结构

```
40_工作开发/
├── 00_笔记/             # 通用技术笔记
├── 03_工具与平台/       # 工具、平台与工作流记录
├── 10_项目/             # 具体开发项目
├── 20_代码片段/         # 可复用代码
├── 空中操作数据集设计/  # 专题设计资料
└── README.md
```

## 📝 技术笔记格式

```markdown
---
主题: 主题名称
分类: [[编程语言/Python]] 或 [[工具链/Git]]
标签: #技巧 #教程
创建时间: YYYY-MM-DD
最后更新: YYYY-MM-DD
相关资源: [链接]
---

# 主题名称

## 概述
一句话概括。

## 核心内容
详细说明。

## 示例代码
```python
# code here
```

## 常见问题
- Q: 问题？
  A: 答案。

## 相关链接
- [[相关笔记1]]
- [[相关笔记2]]
```

## 🔗 知识库链接策略

- 技术笔记 → 项目: `[[../10_项目/项目名]]`
- 项目 → 论文: `[[../30_论文研究/论文笔记]]`
- 代码片段 → 笔记: 嵌入或链接
- 学习路径 → 目标: `[[../30_论文研究/]]` 或外部资源

## 💡 使用建议

- 按领域分层（操作系统、编程语言等）而非按项目，便于知识复用
- 项目文档保持简洁，详细设计放 `10_项目/项目名/` 下
- 代码片段尽量通用化，添加使用说明
- 定期回顾 `30_学习路径/` 更新技能树
