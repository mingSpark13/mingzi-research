# 40_工作开发 🛠️

此目录管理个人工作开发相关的知识库、技术笔记和项目文档。

## 📋 目录结构

```
40_工作开发/
├── 00_笔记/                  # 技术学习笔记
│   ├── LLM理论/              #   大模型显存、后训练、ViT、Transformer 等
│   ├── 具身智能/             #   VLA SFT、RoboClaw 等
│   └── 工具与引擎/           #   UE5、3DGS、AI面试素材
├── 01_仿真平台/              # 仿真环境与数据集平台（AirSim、GenieSim、CarlaAir、nuScenes）
├── 02_视觉感知工具/          # 视觉感知开源工具（SAM、YOLO、GroundingDINO、LeRobot 等）
├── 03_世界模型与视频生成/    # 世界模型与视频生成（InSpatio-World、Wan2.1、DreamZero 等）
├── 04_Agent与LLM工具/        # Agent 框架与 LLM 工具（DeerFlow、FluxVLA、GitNexus 等）
├── 10_项目/                  # 具体开发项目
│   ├── AirSpark/             #   城市低空多模态数据采集平台
│   ├── 空中操作数据集/       #   数据集设计、VLN/VLA 报告、OpenVLA 路线图
│   └── UAV机械臂RL/          #   IsaacLab RL 训练方案
├── 20_代码片段/              # 可复用代码片段
├── 20_研究调研/              # 研究方向调研（World Model×RL、LiteResearcher、LLMinFlash）
└── 30_仿真场景生成/          # 无人机仿真场景生成工作流（程序化扩展、3DGS→UE5）
```

## 🔗 知识库链接策略

- 技术笔记 → 项目: `[[../10_项目/项目名]]`
- 项目 → 论文: `[[../30_论文研究/论文笔记]]`
- 代码片段 → 笔记: 嵌入或链接
- 学习路径 → 目标: `[[../30_论文研究/]]` 或外部资源

## 💡 使用建议

- 按领域分层（操作系统、编程语言等）而非按项目，便于知识复用
- 项目文档保持简洁，详细设计放 `10_项目/项目名/` 下
- 代码片段尽量通用化，添加使用说明
- 定期回顾 `30_学习路径/` 更新技能树
