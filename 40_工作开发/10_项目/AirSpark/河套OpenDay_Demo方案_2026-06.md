# 河套 Open Day — AirSpark Demo 方案

> 截止日期：2026-06-26 | 展示场合：河套合作区 Open Day

---

## 一、填表内容

**DEMO 名称**

```
AirSpark：新一代无人机具身智能基座
```

**简要介绍**

```
AirSpark 是一个面向无人机具身智能的仿真与数据工厂，支持无人机在高保真
虚拟场景中完成自主飞行、目标接近、环境交互和空中机械臂操作等具身任务，
并将"导航—操作"全过程转化为可训练、可评测、可复现的多模态数据。平台
面向低空巡检、目标搜索、空中操作、遥操作采集和 Sim2Real 研究，为无人机
VLA/VLN、操作策略训练和具身智能算法验证提供数据基座。
```

**展现形式**

```
视频演示 + VR 虚实结合遥操作探索场景。视频部分展示无人机自然语言任务执
行、导航—操作流程和多模态数据生成能力；现场部分提供 VR 遥操作体验，观
众可沉浸式控制无人机探索高保真低空场景。
```

---

## 二、Demo 目标

本次 Demo 需向观众清晰传递三件事：

1. **AirSpark 是什么**：一个把虚拟无人机飞行任务自动转化为具身智能训练数据的平台，不是普通仿真软件
2. **它能做什么**：导航→操作全链路任务执行，多模态数据实时生成，VR 遥操作采集
3. **研究价值在哪**：为 VLA/VLN 训练和算法评测提供数据基座，支持 Sim2Real 研究

---

## 三、展示主线

```
高保真虚拟场景
       ↓
无人机自主任务执行（自然语言驱动）
       ↓
导航 → 接近 → 环境交互 / 空中操作
       ↓
多模态传感器实时采集（RGB / Depth / Seg / Pose / IMU）
       ↓
标准化 episode 数据自动生成
       ↓
VLA / VLN 模型训练 & 算法评测
```

## 四、现场展示内容

### 视频部分（主屏，约 10 分钟）

| 段落     | 时间       | 展示内容                        | 核心价值        |
| ------ | -------- | --------------------------- | ----------- |
| 平台介绍   | 0–2 min  | 一页架构图 + 60s 宣传片             | 建立认知：具身智能基座 |
| 高保真场景  | 2–4 min  | 校园/城市低空漫游，天气/光照切换           | 仿真环境真实可用    |
| 自主任务执行 | 4–8 min  | 自然语言输入 → 任务分解 → 无人机导航→接近→操作 | 具身任务全链路     |
| 数据工厂输出 | 8–10 min | 多模态数据流、episode 生成、训练格式转换    | 不是游戏，是数据平台  |

### VR 遥操作体验（现场互动，约 5 分钟）

观众佩戴 VR 头显，第一人称沉浸式控制无人机在高保真场景中低空探索，后台实时采集遥操作轨迹数据，展示"人工遥操作 → 数据采集 → 训练样本"的具身数据采集路径。

## 九、风险提示

- **空中操作**：近距操作/机械臂是亮点但开发量较大，Week 2 优先保任务 1/2，任务 3 视进度调整
- **VR 头显**：型号待确认（Meta Quest / Valve Index / HTC Vive），确认后评估驱动方案
- **LLM 稳定性**：固定指令 + 规则模板兜底，现场不依赖实时在线推理
- **过度承诺**：Sim2Real 全链路、6 类完整接触操作均为后续目标，现场统一说"已预留接口，后续扩展"
