# DeerFlow 2.0 - ByteDance开源超级Agent Harness

## 三句话摘要
1. **定位**：ByteDance 开源的**超级 Agent Harness**（运行时框架），基于 LangGraph + LangChain，构建多步骤复杂任务的完整执行环境。
2. **核心能力**：Sub-Agent 并行编排 + Skills 渐进式加载 + 沙箱文件系统 + 长期记忆，支持研究/报告/幻灯片/网页/图文视频生成。
3. **与 OpenClaw 对比**：同为 Agent Harness，但 DeerFlow 侧重 Deep Research 场景，内置沙箱执行；OpenClaw 侧重工作流编排与多渠道集成。

---

## 核心架构

### 技术栈
- **底层**：LangGraph + LangChain
- **模型**：OpenAI 兼容 API，支持 100k+ context、多模态、Tool Use
- **部署**：Local / Docker / Kubernetes 三种沙箱模式
- **License**：MIT

### 核心模块

#### 1. Skills 系统
- 结构化 Markdown 技能模块（定义工作流+最佳实践+资源引用）
- 内置技能：research、report-generation、slide-creation、web-page、image-generation、video-generation
- 渐进式加载（只加载任务需要的 Skill，保持 context 精简）
- 自定义 Skill 可挂载到 `/mnt/skills/custom/`

#### 2. Sub-Agent 并行编排
- 主 Agent 按需生成 Sub-Agent（各自独立 context）
- Sub-Agent 并行执行，按需汇总结果
- 支持复杂多步骤任务（分钟到小时级）

#### 3. 沙箱文件系统
- 每个任务在隔离 Docker 容器中运行
- 完整文件系统：`/mnt/user-data/{uploads, workspace, outputs}`
- 沙箱执行：`Local / Docker / Docker+K8s`

#### 4. 上下文工程
- Sub-Agent 上下文隔离，避免污染
- 会话内主动 Summary（压缩已完成子任务）
- 支持 Long Context（100k+ tokens）

#### 5. 长期记忆
- 跨会话持久化：用户 profile / 偏好 / 知识积累
- 本地存储，用户可控

### IM 频道集成

| Channel | Transport | 难度 |
|---------|-----------|------|
| Telegram | Bot API (long-polling) | Easy |
| Slack | Socket Mode | Moderate |
| **Feishu/Lark** | **WebSocket** | **Moderate** |
| 飞书 | WebSocket | ✅ 已支持 |

## 与 OpenClaw 对比

| 维度 | DeerFlow 2.0 | OpenClaw |

| 定位 | Deep Research + 超级 Agent Harness | Agent 工作流编排 + 多渠道集成 |

## 落地价值

1. **Deep Research 场景**：DeerFlow 天然适合文献调研、竞品分析、多源信息聚合任务
2. **沙箱安全执行**：代码执行/文件操作隔离，适合不确定的工具调用
3. **技能可复用**：SKILL.md 格式与 OpenClaw Skills 相似，可互相借鉴
4. **未来迁移**：主人说"好用就迁移到 DERRFLOW"，可先研究 DeerFlow 的 Skill 格式和 Sub-Agent 协作模式
