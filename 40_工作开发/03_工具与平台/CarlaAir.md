# 🛠️ 平台笔记

## 平台简介

CarlaAir 将 **CARLA（自动驾驶仿真器）** 与 **AirSim（无人机仿真器）** 深度融合，实现空地联合仿真：
- 地面车辆与空中无人机共享同一个 UE4 世界
- 同一套物理引擎
- 同一场暴雨 / 同一天气条件

## 核心特性

| 特性 | 说明 |
|------|------|
| **单进程集成** | 无桥接、零延迟，直接在一个进程内同时运行 CARLA + AirSim |
| **双 Python API** | `carla.Client` + `airsim.MultirotorClient`，各自原生 API 直接调用 |
| **18 路同步传感器** | RGB / LiDAR / 深度 / 语义分割 / IMU / GNSS 等 |
| **内置 FPS 操控** | WASD + 鼠标飞无人机，像玩游戏一样 |
| **开箱即用** | 自动交通、天气切换、6 张城市地图 |
| **示例脚本** | 6 个即开即用的 Python 示例 |

## 对主人研究方向的价值

主人长期关注无人机仿真，AirSim 停止维护后出现了断档。CarlaAir 填补了这个坑：
- **空地协同**：无人机 + 无人车在同一场景联合仿真
- **传感器融合**：18 路传感器适合多模态感知研究
- **天气系统**：研究光照/天气对感知的影响
- **游戏级操控**：快速验证飞控算法

## 落地实验建议

**实验 1：无人机-无人车协同编队**
- 利用 CarlaAir 空地共享同一世界的特性，测试多机协同控制算法
- 对比基线：单无人机仿真（CARLA only / AirSim only）

**实验 2：天气-感知联合仿真**
- 利用内置天气系统，测试 LiDAR/Camera 在雨/雾/夜条件下的感知噪声
- 关联主人无人机感知方向
