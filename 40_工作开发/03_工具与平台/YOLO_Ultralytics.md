# Ultralytics YOLO (YOLO11 / YOLO12)

> 端到端目标检测/分割框架，工业级易用性

## 📋 基本信息

| 项目 | 说明 |
|------|------|
| 类型 | 端到端目标检测/分割 |
| 机构 | Ultralytics |
| 官方文档 | https://docs.ultralytics.com/tasks/detect/ |
| 最新版本 | YOLO12 (Ultralytics 主推) |

## 🎯 定位

YOLO 是**感知层组件**，强调：

- **封闭类别、快速部署**：适合类别已知的固定场景
- **端到端 NMS-free**：减少后处理延迟
- **边缘部署友好**：支持多种导出格式（ONNX、TensorRT 等）

## 📝 主人提供的技术定位

> 封闭类别、追求快和稳选 Ultralytics YOLO这条线最省心。Ultralytics 现在主推 YOLO12，强调端到端、NMS-free、边缘部署友好。

## 💡 在机器人项目中的用法

1. **固定任务检测**：已知类别的物体检测（如「零件A/零件B」）
2. **快速原型**：快速验证检测 pipeline
3. **与 SAM 组合**：`YOLO 检测 → SAM 分割 → 位姿估计`
