## 📌 关键概念

- **SceneSpec**：统一场景规格中间层，连接 LLM 语义规划、程序化生成、真实重建、UE 实例化
- **3DGS**（3D Gaussian Splatting）：用高斯椭球表达连续表面+外观，渲染效果接近照片级
- **MLSLabsRenderer**：UE5 中渲染 3DGS 的插件
- **Q9000**：实验室 LiDAR 扫描仪，提供精准几何真值点云
