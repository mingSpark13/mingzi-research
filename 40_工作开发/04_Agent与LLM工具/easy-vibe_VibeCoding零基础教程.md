# easy-vibe — Vibe Coding 零基础保姆级教程

## 🎯 是什么

> **直接上手，一起 vibe！会说话就会做应用**
> Learn Vibe Coding From 0 to 1

Datawhale 出品的 **Vibe Coding 零基础教程**，教你"会说话就会做应用"。从 AI 编程初体验到全栈 AI 应用开发，配有交互式教程、保姆级图文和沉浸式模拟编程。

**支持 10 种语言**：简体中文、繁体中文、英语、日语、西班牙语、法语、韩语、阿拉伯语、越南语、德语
