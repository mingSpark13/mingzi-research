# InSpatio-World 技术谱系总览

## 主结论
InSpatio-World 不是单一路线的自然延伸，而是四条技术链的系统汇合：
1. 开放视频生成骨干（Wan2.1）
2. 自回归实时视频生成训练法（Self-Forcing）
3. 几何恢复 / 深度支撑（Depth-Anything-3）
4. 视觉语义描述与相机控制视频重渲染（Florence-2 / ReCamMaster / GCD / Trajectory Attention / DaS）
