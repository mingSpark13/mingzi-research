## 📚 仓库说明

本仓库包含明子的论文研究笔记、知识图谱和科研方向整理，**不包含 PDF 原文**。

### 目录结构

```
30_论文研究/
├── 01_论文库/          # PDF 文件（不在 Git 中）
├── 02_阅读笔记/        # 论文阅读笔记（Markdown）
├── 05_科研研究/        # 7 个研究方向（D01-D07）
├── 06_知识Wiki/        # 知识图谱与概念索引
└── scripts/            # 辅助脚本
```

### 📄 如何获取 PDF 原文

**PDF 文件因体积过大（7.1GB，641 篇）未纳入 Git 版本控制。**

如需获取特定论文 PDF，请：

1. **查看笔记标题**：在 `02_阅读笔记/` 中找到对应笔记
2. **联系花火**：通过以下方式请求 PDF
   - 飞书：@花火
   - GitHub Issue：在本仓库提 Issue，注明论文标题
   - 邮件：bigetua@gmail.com

花火会根据笔记标题从 `01_论文库/` 中找到对应 PDF 并发送给你。
