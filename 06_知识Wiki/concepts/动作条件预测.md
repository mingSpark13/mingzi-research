# 概念：动作条件

> Action-conditioned Prediction，World Model 以机器人动作（action）为显式条件输入，预测动作施加后的世界状态变化。

## 说明

动作条件是具身 World Model 与普通视频生成的核心区别：
- **视频续接**（无 action）：给定帧 t，预测帧 t+1（无控制信号）
- **动作条件**（有 action）：给定帧 t + 动作 a，预测 a 施加后的帧 t+1（有控制信号）

没有 action conditioning 的模型只是在做"视频看起来合理"，而不是在建模"动作与世界的因果关系"。

## 核心意义

1. **因果建模**：建立 action → consequence 的因果链条
2. **长时稳定性**：有 action 条件时 rollouts 更稳定（条件不断刷新）
3. **物理一致性**：动作是物理交互的触发器，action conditioning 是物理正确性的关键
4. **策略支持**：Action-conditioned future prediction 可直接用于策略选择
