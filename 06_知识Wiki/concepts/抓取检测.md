# 概念：抓取检测 - Grasp Detection

> 机器人抓取的前置感知模块，输出可供抓取的候选位姿

## 基本定义

Grasp Detection 输出：「在哪里抓、怎么抓」的候选位姿，通常包含：
- 抓取点 (approach direction)
- 夹爪旋转角
- 抓取宽度
- 抓取质量分数

## 代表工作

| 工作 | 说明 |
|------|------|
| GraspNet-1B | 大规模抓取 benchmark，10亿+标注 |
| AnyGrasp | GraspNet 的产品化版本，零样本 |
| Contact-GraspNet | 基于点云的抓取检测 |
| 6-DOF GraspNet | 任意角度 6-DOF 抓取 |

## 在机器人操作 pipeline 中的位置

```
视觉输入
    ↓
物体检测/分割 (YOLO / Grounding DINO / SAM)
    ↓
6D Pose Estimation (FoundationPose) [可选]
    ↓
抓取候选生成 (GraspNet / AnyGrasp)
    ↓
抓取选择 + 运动规划
    ↓
机械臂执行
```

## 主人建议的感知链路

主人建议的完整感知抓取链：
> Grounding DINO + SAM2/3 + FoundationPose + GraspNet/AnyGrasp + 机械臂执行
