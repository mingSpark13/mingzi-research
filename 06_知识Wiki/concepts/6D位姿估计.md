# 概念：6D Pose Estimation

> 6自由度位姿估计，机器人操作的关键感知能力

## 基本定义

6D Pose = 3D 平移 (x, y, z) + 3D 旋转 (roll, pitch, yaw)

是机器人「知道物体在哪里、什么朝向」的核心感知能力。

## 代表工作

| 工作 | 特点 |
|------|------|
| FoundationPose | NVIDIA 开源，model-based 和 model-free 统一框架 |
| BundleSDF | NVIDIA，RGB-D 跟踪 + 神经重建 |
| MegaPose | 粗略到精细的两阶段方法 |
| GPNP | 基于对应点的解析解法 |

## 在机器人操作 pipeline 中的位置

```
视觉输入 (RGB-D)
    ↓
6D Pose Estimation (FoundationPose 等)
    ↓
抓取候选生成 (GraspNet/AnyGrasp)
    ↓
运动规划 (RRT/MPC)
    ↓
机械臂执行
```

## 主人定位

FoundationPose 是最值得关注的开源 6D pose 方案。
