# 概念：物理一致性

> World Model 生成的视频/状态预测是否符合物理定律（重力、接触、摩擦、能量守恒等）。

## 说明

物理一致性是衡量 world model 是否"真正建模了世界运行规律"的核心指标，与视频质量的区别在于：
- 视频质量好 ≠ 物理一致性高（画面漂亮但可能违反物理）
- 物理一致性高意味着 action → consequence 的因果预测正确

## 核心问题

1. **刚体物理**：物体不能悬空、碰撞要有响应、重力要正确
2. **接触与摩擦**：抓取/放置时的接触力是否符合现实
3. **时间累积**：长时 rollout 中物理违规是否漂移
4. **流体/柔体**：液体、柔性体、绳索等复杂物理的一致性
