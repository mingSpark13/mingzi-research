# OpenFly (2502.18041)

**核心价值**: OpenFly 把多引擎渲染、自动轨迹与自动指令生成拼成统一数据工厂，是 D06 很强的空中 VLN 基础设施样板。

**与我们的关系**:
- 它更像 D06 的 **数据工厂 + benchmark + 轻量历史建模基座**，不是直接替代 APEX / AirNav 的方法主骨架，但很适合作为龙虾后续实验基础设施参考。
