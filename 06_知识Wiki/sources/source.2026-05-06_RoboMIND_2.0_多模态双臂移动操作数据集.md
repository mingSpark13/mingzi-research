# RoboMIND 2.0

**核心价值**: 现有机器人数据集常局限于单机械臂或单载体，难支撑长程双臂具身泛化训练。作者构建覆盖6种载体、31万双臂轨迹、2万移动操作和1.2万触觉样本的 RoboMIND 2.0，并提出 MIND-2 层级 VLM+VLA 控制框架。在多任务 VLA 与真实执行评测中，数据规模、仿真混合与触觉接入都显著提升泛化表现。
