# Failure Mechanisms and Risk Estimation for Legged Robot Locomotion on Granular Slopes (2603.06928)

**核心价值**: 腿足机器人在沙坡等可变形地形上经常不是“下陷太深”而是“抓地延迟+后滑加剧”，但这类失效机理缺少可量化模型。 论文结合六足机器人倾斜颗粒床实验与地形相互作用建模，分离法向/切向阻力随坡度变化规律，并构建可预测 anchoring timing、步长和速度的 physics-informed 风险模型。 模型给出了 sinkage / slippage 两类 failure phase diagram，可定量估计不同坡度与地形强度下的失效风险，为腿足机器人在松散坡面的安全运行提供判据。
