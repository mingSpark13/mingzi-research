# AerialVLA (2603.14363)

**核心要点**:
- **极简端到端 VLA**: 原始视觉 + 模糊语言指令 → 连续物理控制信号，无需密集 oracle 引导或外部目标检测器。
- **双视角感知**: 前视 + 下视相机融合，减少视觉冗余同时保留导航和精确着陆所需的关键线索，并利于 sim-to-real 迁移。
- **模糊方向提示机制**: 仅用机载传感器生成模糊方向提示，完全消除对 oracle 引导的依赖，实现真正自主。

**与我们的关系**:
- **直接对标龙虾导航路线**：龙虾项目也是"语言指令 → 无人机自主找到目标"，AerialVLA 提供了极简端到端 baseline，其 dual-view + fuzzy prompting 设计对龙虾的"手持D435i + 语义搜索"链路有直接参考价值。
