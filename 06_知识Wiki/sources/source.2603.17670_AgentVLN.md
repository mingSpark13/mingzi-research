# AgentVLN: Towards Agentic Vision-and-Language Navigation (2603.17670)

**核心价值**: AgentVLN 将 VLN 重构为 POSMDP 问题，提出 VLM-as-Brain 范式，通过即插即用技能库解耦高层语义推理与底层感知规划，仅用 3B 参数即在 R2R-CE/RxR-CE 上超越 8.3B 大模型，并实现四足机器人端侧实时部署。
