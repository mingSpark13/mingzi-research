# source.2605.03269_RLDX_1_Technical_Report

**核心价值**: 现有 VLA 虽然继承了视觉语言模型的泛化能力，但在真实复杂操作里仍缺运动感知、记忆决策、物理感知与实时落地能力。 RLDX-1 提出 Multi-Stream Action Transformer，把异构模态拆成独立 stream 再做 cross-modal joint self-attention，同时叠加稀有场景数据合成、针对人形/灵巧操作的训练流程和实时推理优化。 在仿真和真实任务上持续超过 π0.5 与 GR00T N1.6，尤其在 ALLEX humanoid tasks 上达到 86.8% 成功率，明显高于约 40% 的对照模型。
