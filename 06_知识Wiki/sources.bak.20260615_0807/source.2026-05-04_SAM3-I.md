# SAM3-I: Segment Anything with Instructions (ACL 2026)

**核心价值**: 问题背景: SAM3 只能处理简短名词短语（NP）级别的概念分割，无法直接理解和执行包含属性、关系、功能、状态甚至推理过程的自然语言指令，目前依赖外部多模态 Agent 将复杂指令转为 NP 并进行迭代过滤，效果粗糙且效率低下。
