# 论文库 (PDF 存档)

按**研究方向**分类，会议/来源信息在笔记 frontmatter 中。

## 📂 目录结构

```
01_论文库/
├── 空中操作/      # aerial manipulation, UAV
├── 通用操作/      # general manipulation, dexterous
├── 强化学习/      # reinforcement learning
├── 运动控制/      # locomotion, hybrid control
└── (按需扩展)     # 计算机视觉 / 大模型 / SLAM与感知 ...
```

## 📥 添加新 PDF

1. 确定论文的核心研究方向
2. 放入对应子目录，命名: `YYYY_论文标题.pdf`
3. 在 `02_阅读笔记/` 用 `_TEMPLATE.md` 创建笔记

## 📏 分类规则

- **按研究方向分**，不按会议/来源分
- 综述论文按其主题方向归类
- 同一论文只放一个方向（选最核心的）
- 新方向超过2篇论文时创建子目录
